<Title>How to pass mixed stream name information between different users?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="All" />



- - -

ZegoExpress SDK provides the following two methods:

* Use stream extra information: Users call the [setStreamExtraInfo](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~Java~class~im-zego-zegoexpress-zego-express-engine&jumpType=route#set-stream-extra-info) interface to pass in, and other users in the same room get the stream extra information by listening to the [onRoomStreamExtraInfoUpdate](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~Java~class~im-zego-zegoexpress-callback-i-zego-event-handler#on-room-stream-extra-info-update) callback.
* Use custom signaling: Call the [sendCustomCommand](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~Java~class~im-zego-zegoexpress-zego-express-engine#send-custom-command) interface to send stream name information to a specified user, and the user gets the corresponding information by listening to the [onIMRecvCustomCommand](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~Java~class~im-zego-zegoexpress-callback-i-zego-event-handler#on-im-recv-custom-command) callback.

<Warning title="Note">


Users can also use their own signaling service to send and receive mixed stream name information.

</Warning>
