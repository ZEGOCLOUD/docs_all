<Title>When using the "uni-app" framework, "playerStateUpdate" shows successful stream pulling, but using "srcObject" cannot render the video?</Title>

<ArticleMetadata language="en" product="Video Call" platform="Web / uni-app" />



- - -

The "uni-app" framework does not expose the "srcObject" property. You need to use native JavaScript to pull and render the stream.
