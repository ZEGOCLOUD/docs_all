<Title>How to handle abnormal font display in file whiteboard?</Title>

<ArticleMetadata language="en" product="Super Board" platform="All" />


---

If the uploaded source file does not use fonts supported by Super Board, it will cause abnormal font display in the file whiteboard. Please modify the fonts in the source file.

For the fonts supported by Super Board, please refer to [PPT Supported Font List](/super-board-ios/product-desc/use-restrictions/pptfont).
