<Title>For the MixInputList parameter of mixed stream, if ViewType is set to 2, how should StreamId be set?</Title>

<ArticleMetadata language="en" product="Cloud recording" platform="All" />


---

StreamId is an optional parameter. When ViewType is set to 2, StreamId does not need to be included in the request.
