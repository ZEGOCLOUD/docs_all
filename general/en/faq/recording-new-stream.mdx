<Title>After starting recording, if a new stream is added to the room, will this new stream be recorded?</Title>

<ArticleMetadata language="en" product="Cloud recording" platform="All" />


---

1. RecordStreamList not set: If the current task recording stream count has not reached the limit (36 streams for single stream recording, 25 streams for mixed stream recording), the new stream will be recorded.

2. RecordStreamList is set: If the new stream belongs to RecordStreamList, the new stream will be recorded. If the new stream does not belong to RecordStreamList, the new stream will not be recorded.
