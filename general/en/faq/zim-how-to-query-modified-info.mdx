<Title>After users modify their name, avatar, and personal information extension fields through ZIM SDK interface, how to query the modified information?</Title>

<ArticleMetadata language="en" product="In-app chat" platform="All" />



- - -

Users can query the complete information of specified users through the [queryUsersInfo](https://docs.zegocloud.com/article/api?doc=zim_API~java_android~class~ZIM#query-users-info) interface, including user name and user extension fields.

**User avatar URL and user extension field information can only be obtained through this interface.**
