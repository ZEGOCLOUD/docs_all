<Title>What is the reason for file transcoding loading failure with error code 60018?</Title>

<ArticleMetadata language="en" product="File sharing" platform="iOS / Android / Web" />



- - -

This error code indicates that the file cannot be found when loading the file. The possible reason is: the AppID used during upload is inconsistent with the AppID used when opening the document.
