<Title>After cloud recording ends, I don't see the recorded files in cloud storage. How to handle this?</Title>

<ArticleMetadata language="en" product="Cloud recording" platform="All" />



- - -

In the following cases, no recording files will be generated after cloud recording ends:

- When performing cloud recording, the StorageParams storage parameter configuration passed in is incorrect. For specific configuration, please refer to [How to fill in each cloud storage related parameter in StorageParams?](/faq/cloud-storage-params-config)

- No access key with "write-only permission" is provided.

- When performing cloud recording, the RoomID passed in does not exist.

- When performing cloud recording, no users in the room are publishing streams.
