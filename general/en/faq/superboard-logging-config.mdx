<Title>How to set Superboard logs?</Title>

<ArticleMetadata language="en" product="Super Board" platform="iOS / Android" />


---

iOS default log path

```objc
[NSHomeDirectory() stringByAppendingPathComponent:@"Documents/ZegoLogFile"]
```

Android default log path

```java
/sdcard/Android/data/your application package name/files/zegologs
```

You can modify the log path through [setCustomizedConfig](https://docs.zegocloud.com/article/api?doc=superboard_API~java_android~class~ZegoSuperBoardManager#set-customized-config).
