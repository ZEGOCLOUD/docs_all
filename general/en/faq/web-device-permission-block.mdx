<Title>How to prohibit the access permission popup for microphone/camera on Web platform?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="Web" />



---

There are the following 3 scenarios:

#### Scenario 1: Prohibit camera access permission popup

- When calling the [createZegoStream](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-zego-stream) interface, set the video property of the camera capture stream related configuration to false.
- Comment out the [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) interface related code, or set the camera property of [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) to false.

Both of the above must be satisfied.

<br/>

#### Scenario 2: Prohibit microphone access permission popup:

- When calling the [createZegoStream](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-zego-stream) interface, set the audio property of the camera capture stream related configuration to false.

- Comment out the [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) interface related code, or set the microphone property of [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) to false.

Both of the above must be satisfied.

<br/>

#### Scenario 3: Prohibit camera and microphone access permission popup

- Do not call the [createZegoStream](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-zego-stream) interface, or when calling the [createZegoStream](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-zego-stream) interface, set the video and audio properties of the camera capture stream related configuration to false.
- Comment out the [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) interface related code, or set the camera and microphone properties of [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#check-system-requirements) to false.

Both of the above must be satisfied.
