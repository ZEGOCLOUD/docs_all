<Accordion title='1. When the Android platform is integrated through "jCenter", the "jCenter" dependency fails, and an error message is displayed: "Failed to resolve: im.zego:express-audio:x.x.x"?' defaultOpen="false">

Please confirm whether the SDK version you rely on is the latest, and check the current SDK version through [SDK Downloads](https://docs.zegocloud.com/article/3690). If it is not the latest version, the dependency will fail.

</Accordion>

<Accordion title='2. Why does the SDK fail to load when the Electron platform uses the "Electron-Vue" framework?' defaultOpen="false">

The "Vue" framework needs "Webpack" and the "require" parameter of "Webpack" is changed to "window.require".
</Accordion>

<Accordion title='3. How does the Web platform determine the hardware support and authorization status?' defaultOpen="false">

The hardware condition can be obtained through the [checkSystemRequirements](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web_web~class~ZegoExpressEngine#check-system-requirements) interface to obtain the "CapabilityDetection" judgment.
</Accordion>
