<Title>The `ZIMMessageSentCallback` in ZIM Flutter [sendMessage](https://pub.dev/documentation/zego_zim/latest/zego_zim/ZIM/sendMessage.html) interface does not expose `errorcode`. How to get it?</Title>

<ArticleMetadata language="en" product="In-app chat" platform="Flutter" />



------
You can get the `errorcode` by catching the `PlatformException` exception. For details, please refer to the following code snippet.   
```
try { 
    var ret = await ZIM.getInstance()!.sendMessage(
        message, 
        obj.toUserID, 
        ZIMConversationType.peer, 
        config, 
        messageSendNotification); 
    } 
on PlatformException catch (onError) {
     onError.code; onError.message; 
}
```
