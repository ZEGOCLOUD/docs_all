<Title>How to push pure audio stream on Web platform?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="Web" />



- - -

The SDK internally defaults to pushing audio and video. If you want to push a pure audio stream, when calling the [createStream ](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-stream) method, you need to set the "video" property of the camera object to "false".

```js
let localStream = await zg.createStream({
        camera: {
            video: false,//Do not push video
            audio: true,
       }
  });
```
