<Title>After updating the ZIM SDK version, do I need to rebuild the base?</Title>

<ArticleMetadata language="en" product="In-app chat" platform="uni-app" />



- - -
 
You must rebuild the debug base (please refer to [Implementing Basic Message Sending and Receiving](/zim-uniapp/send-and-receive-messages#24-自定义调试基座) "Custom Debug Base"), and uninstall the old client application. After reinstalling, it can run correctly.
