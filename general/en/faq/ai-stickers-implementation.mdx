<Title>How to implement pendant function in AI?</Title>

<ArticleMetadata language="en" product="AI Effects" platform="iOS / Android / macOS / Windows" />


-----

<Warning title="Note">

It is not recommended to use the pendant capability of AI Effects. Starting from version 2.0.0, the AI Effects SDK no longer supports pendants. If you are still using an older version of the SDK and confirm that you need pendants and the current capabilities meet your requirements, please refer to the following documentation.

</Warning>



## Feature Introduction

ZegoEffects SDK provides facial effect pendants. Based on facial keypoint detection and tracking algorithms, it provides facial effects. The SDK can directly fit 2D pendant materials to the face, trigger pendant effects through facial actions, and make live streaming more interesting.

## Prerequisites

Before using the pendant function of ZegoEffects SDK, please ensure that you have completed the following operations:

<Warning title="Note">


The following links take iOS as an example. If you are integrating other platforms, please switch accordingly.

</Warning>




- Integrated ZegoEffects SDK in the project and implemented basic image processing functions. For details, please refer to [iOS - Integrate SDK](/ai-effects-ios-objc/quick-starts/import-the-sdk).
- Import the "PendantResources.bundle" resource path. For details, please refer to [iOS Platform - Import Resources and Models](/ai-effects-ios-objc/quick-starts/import-resources-and-models).



## Usage Steps

Pendant stickers include: three animals (Animal), clown (Clown), deer (Deer), diving goggles (Dive), cool face mask (Facefilm), sailor moon girl (Girl), cat head (Cat), watermelon (Watermelon) and other effects. When developers call the interface, they can select the corresponding filter resources as needed.

<Tabs>
<Tab title="iOS-Objective-C Platform">
Call the [setPendant](https://docs.zegocloud.com/article/api?doc=Effects_SDK_API~objective-c_ios~class~ZegoEffects#set-pendant-path) interface to set the pendant path to the SDK. If you need to remove the pendant, you can pass `nil` to the [setPendant](https://docs.zegocloud.com/article/api?doc=Effects_SDK_API~objective-c_ios~class~ZegoEffects#set-pendant-path) interface.

<Warning title="Note">

- Pendant effects cannot be stacked. After setting a new pendant path, it will override the previous pendant effect.
- Pendant effects cannot be used together with [Makeup - Style Makeup](/ai-effects-ios-objc/guides/makeups) function. Enabling the pendant function will disable the "style makeup" effect.

</Warning>


```objc
// Set pendant
[effects setPendant:@"/xxx/xxx/pendantBaby.bundle"];
// Remove pendant
[effects setPendant:nil];
```

</Tab>
