<Title>When logging in to a room on the Web platform, the error "content":"cmd=login, err_code=1000000101, err_message=login liveroom request fail", or "content":"server error=1000000101" appears. What is the reason?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="Web" />



- - -

There are two possible reasons for this error:

- AppID has expired.
- Server address is incorrect. Please go to [ZEGOCLOUD Console](https://console.zegocloud.com) to confirm.
