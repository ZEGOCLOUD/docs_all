<Title>Does In-app Chat support overseas access?</Title>

<ArticleMetadata language="en" product="In-app chat" platform="iOS / Android / macOS / Windows / Web / Mini Program / Flutter / ReactNative / uni-app / Unity3D" />



- - -

Yes. In-app Chat provides a globally covered, highly connected, highly reliable, and secure network connection channel. ZIM SDK is based on an excellent global network scheduling strategy and can meet messaging interoperability in more than 200 countries or regions.

- China: South China, North China, East China, Hong Kong, Taiwan, etc.
- Other countries (or regions):
    - Asia: Singapore, Saudi Arabia, Thailand, Japan, Vietnam, India, South Korea, Philippines, Malaysia, Dubai, etc.
    - Europe: United Kingdom, Netherlands, Germany, Italy, Norway, France, Russia, Spain, etc.
    - North America: United States, Canada, Mexico, etc.
    - Oceania: Australia.
    - Africa: South Africa, Nigeria, Kenya, etc.
