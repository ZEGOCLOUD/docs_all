<Title>Why does the "background callback - recording file generation callback" callback twice? And the video content of both callbacks is the same?</Title>

<ArticleMetadata language="en" product="Cloud recording / Local server recording" platform="All" />



- - -

The file formats supported by CDN recording are: MP4, MP3, M3U8, FLV.

If you select two recording formats, the callback will be triggered twice. Generally, customers only select one recording format, in which case only one callback will be generated.
