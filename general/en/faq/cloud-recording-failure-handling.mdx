<Title>What happens when the cloud recording server experiences network disconnection or abnormal crash?</Title>

<ArticleMetadata language="en" product="Cloud recording" platform="All" />


---

The cloud recording server will switch to a new server within two minutes and automatically resume recording. After recovery, a new file will be recorded.
