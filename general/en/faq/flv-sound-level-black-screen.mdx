<Title>When mixed stream is enabled with sound level feature, using "flv.js" to pull and play streams results in black screen on some browsers?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="iOS / Android / macOS / Windows / Web" />



- - -

### Problem Cause

When mixed stream is enabled with sound level "soundLevel", the sound level uses "SEI", which is not compatible with some browsers.

### Solution

Please set the parameter "withSoundLevel" to "false" to disable the mixed stream sound level feature.
