<Title>How to get Roomkit SDK logs?</Title>

<ArticleMetadata language="en" product="Roomkit" platform="iOS / Android" />


---

By default, Roomkit SDK generates eight types of TXT log files starting with zegoapplog, zegoavlog, zegoconnection, zegodocslog, zegogatewaylog, zegowhiteboardviewlog, zegoroomkitdclog, zegoverboselog, with a maximum of 3 files generated. The file names end with sequence numbers "1", "2", "3". The default size limit for a single log file is 5MB (5 * 1024 * 1024 Bytes). After 3 log files are full, the log file with the earliest modification time will be deleted.

### iOS/iPadOS

1. Connect the iOS device to Mac, open Xcode, and select "Window > Devices and Simulators" from the top menu bar.
<Frame width="512" height="auto" caption=""><Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/FAQ/xcode_device_window_menu.png" /></Frame></Frame>

2. Select the specified device on the left, then find the App that needs to get logs in "INSTALLED APPS", click the gear icon below, select "Download Container..." and save.
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/FAQ/xcode-device-window.png" /></Frame>

3. Open "Finder" to find the saved ".xcappdata" file, right-click and select "Show Package Contents", select the "/AppData/Documents/ZegoLogFile" directory to find the log files.

### Android

Go to the "/sdcard/Android/data/[Application package name]/files/ZegoLog/" directory to find the log files.
