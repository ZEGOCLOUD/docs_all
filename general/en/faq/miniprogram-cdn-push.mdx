<Title>How does a mini program push directly to CDN?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call" platform="Mini Program" />



- - -

Call the [startPublishingStream ](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_wxxcx~class~ZegoExpressEngine&jumpType=route#start-publishing-stream) method and set the parameter "sourceType" in "publishOption" to "CDN" to get the CDN publish stream address, then use that address to push the stream.

```JavaScript
const { url } = await zg.startPublishingStream(data.pushStreamID,{sourceType:"CDN"})
```
