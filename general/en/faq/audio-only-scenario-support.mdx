<Title>Can I implement an audio-only call scenario using the Video Call SDK?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call" platform="All" />


----
Yes, you can disable the camera to implement an audio-only call scenario by setting the `enable` parameter of the `enableCamera` interface to false.
