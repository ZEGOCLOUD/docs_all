<Title>Are there file size limits?</Title>

<ArticleMetadata language="en" product="File sharing" platform="iOS / Android / Web" />



- - -

Yes, different types of files have different size limits:

- PPT (100 MB)
- Word (100 MB)
- PDF (100 MB)
- Excel (10 MB)
- Image (100 MB)
- TXT (2 MB)
- H5 file ZIP package (50 MB)
