<Title>How to get the SDK version number?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / AI Effects / In-app chat / Super Board / Live streaming" platform="All" />



- - -

Each product SDK provided by ZEGOCLOUD provides a corresponding API interface to get the current version number. Taking the Android platform as an example:

| Product | SDK Name | Interface to Get Version Number |
|-|-|-|
| Video Call | ZEGOCLOUD Express-Video SDK | [getVersion](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~java_android~class~ZegoExpressEngine#get-version) |
| Audio Call | ZEGOCLOUD Express-Audio SDK | [getVersion](https://docs.zegocloud.com/article/api?doc=Express_Audio_SDK_API~java_android~class~ZegoExpressEngine#get-version) |
| Live Streaming | ZEGOCLOUD Express SDK | [getVersion](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~java_android~class~ZegoExpressEngine#get-version) |
| In-app Chat | ZIM SDK | [getVersion](https://docs.zegocloud.com/article/api?doc=zim_API~java_android~class~ZIM#get-version) |
| Superboard | ZEGOCLOUD Superboard SDK | [getSDKVersion](https://docs.zegocloud.com/article/api?doc=superboard_API~java_android~class~ZegoSuperBoardManager#get-sdk-version) |
| AI-Effects | ZEGOCLOUD Effects SDK | [getVersion](https://docs.zegocloud.com/article/api?doc=effects-sdk_API~java_android~class~ZegoEffects#get-version) |
