<Title>When using CDN recording, if the streaming end experiences stream interruption, multiple recording files will be generated. How can these recording files be merged?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="All" />



- - -

You can merge them in the following two ways (choose one):

- **(Recommended)** Use the ZEGOCLOUD server-side interface [Merge Media Files](/real-time-video-server/api-reference/cdn/merge-media).
- Use the [FFmpeg](https://ffmpeg.org/download.html) tool.
