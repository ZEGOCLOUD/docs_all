<Title>Which platforms does ZEGO Cloud Communication SDK support, and what are the system version requirements for each platform?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="All" />

ZEGO Cloud Communication includes the following products and SDKs:
- Video Call: [Express-Video SDK](/real-time-video-android-java/client-sdk/download-sdk)
- Real-time Voice: [Express-Audio SDK](/real-time-voice-android/client-sdk/download-sdk)
