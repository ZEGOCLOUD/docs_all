<Title>What should I do when creating a stream on the Web platform and getting the error "devices detect error: NotReadableError Could not start video source"?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="Web" />



- - - 

Please handle it in the following ways:

1. The browser does not have permission to access media devices. Please check if the page is allowed to access media device information.
2. The media device is occupied, such as the camera being occupied. Please check if any other program is using the camera device.
