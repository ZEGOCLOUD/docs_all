<Title>How to get the log information of the ZegoEffects SDK?</Title>

<ArticleMetadata language="en" product="AI Effects" platform="iOS / Android / macOS / Windows / ReactNative" />


---

You can get the log information of the ZegoEffects SDK by the following methods on each platform.

- Android

    The TXT files starting with “ZefLog” in the “/storage/Android/data/[application package name]/files” path are the log information.

- iOS/iPadOS

    1. Connect the iOS device to the Mac, open Xcode, and select “Window > Devices and Simulators” in the top menu bar.
    <Frame width="512" height="auto" caption=""><Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/FAQ/xcode_device_window_menu.png" /></Frame></Frame>

    2. Select the specified device on the left, then find the App you need to get the log in “INSTALLED APPS”, click the gear icon below, select “Download Container...” and save.
    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/FAQ/xcode_device_window.png" /></Frame>

    3. Open “Finder” to find the saved “.xcappdata” file, right-click to select “Show Package Contents”, select the “AppData > Library > Caches > ZefLogs` directory, and the TXT files starting with “ZefLog” are the log information.


- macOS

    - The TXT files starting with “ZefLog” in the “~/Library/Containers/[application package name Bundle ID]/Data/Library/Caches/ZefLogs” path are the log information.
    - The TXT files starting with “ZefLog” in the “~/Library/Caches/ZefLogs” path are the log information.

- Windows

    The log folder is named "[process name]ZEGO.SDK". To locate the log folder, open "This PC" and enter "%APPDATA%" in the address bar, then press Enter. The TXT files starting with "ZefLog" in this folder are the log files.
