<Title>When logging in to a room on the Web platform, the error "cmd=login, err_code=1011, err_message=token format error" is displayed?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="Web" />



- - -


The error is caused by incorrect format of the "token" parameter passed when logging in to the room. The "token" is a string type. You need to pass the json format after base64 encryption to the [loginRoom](https://www.zegocloud.com/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#login-room) interface. For specific implementation, please refer to the "Login Room" section in [Quick Start - Implementation Process](/real-time-video-web/quick-start/implementing-video-call#登录房间).
