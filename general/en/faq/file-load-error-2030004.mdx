<Title>What is the reason for file transcoding loading failure with error code 2030004?</Title>

<ArticleMetadata language="en" product="File sharing" platform="iOS / Android / Web" />



- - -

Unable to get the width/height of the current View.

When loading a file, the width and height of the DocsView will be obtained for related settings. If it is not obtained, an error will be reported.
