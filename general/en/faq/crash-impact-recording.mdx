<Title>What impact will user program crashes have on cloud recording?</Title>

<ArticleMetadata language="en" product="Cloud recording" platform="All" />


---

Cloud recording will continue to run normally, and the interface layer can also be called normally.
When there is no stream pushing in the recording room for a continuous time exceeding the set continuation recording time, the recording task will automatically end.
