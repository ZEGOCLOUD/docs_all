<Title>How to handle font misalignment or size inconsistency in dynamic transcoding?</Title>

<ArticleMetadata language="en" product="File sharing" platform="iOS / Android / Web" />



- - -

It may be caused by using font libraries not supported by the transcoding server. You can remove or modify unsupported fonts in the source PPT.

It is recommended that developers refer to [PPT Supported Font List](/super-board-ios/product-desc/use-restrictions/pptfont) when creating PPT to avoid using other unsupported fonts, otherwise transcoding failure or format anomalies after transcoding may occur.
