<Title>What file formats does file transcoding support?</Title>

<ArticleMetadata language="en" product="File sharing" platform="iOS / Android / Web" />



- - -

File transcoding currently supports the following formats:

- Static presentation files: pptx, ppt
- Dynamic presentation files: pptx, ppt
- Text files: doc, docx
- Spreadsheet files: xls, xlsx
- Image files: jpg, jpeg, png, bmp
- Other formats: pdf, txt
