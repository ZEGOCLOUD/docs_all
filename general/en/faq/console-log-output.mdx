<Title>How to print SDK logs to the console using Express SDK?</Title>

<ArticleMetadata language="en" product="Video Call / Audio Call / Live streaming" platform="iOS / Android / macOS / Windows / Linux / Flutter / Electron / ReactNative / uni-app / Unity3D / HarmonyOS" />



- - -

On each platform and framework, you can set it through the Express SDK interface. Taking the Android platform as an example:

Developers can enable the debug assistant by setting the enable parameter of the [enableDebugAssistant](https://docs.zegocloud.com/article/api?doc=Express_Video_SDK_API~java_android~class~ZegoExpressEngine#enable-debug-assistant) interface to TRUE. After successful setup, the SDK will print logs to the console and display error prompts in a UI popup when problems occur in the calls of other SDK functions.

<Warning title="Note">


The above method applies to all platforms and frameworks except Web and mini programs.

</Warning>
