# How to create projects

- - -


## Introduction
A project is a carrier for you to use the products and services of ZEGOCLOUD. Through the project, you will quickly integrate related products and services and enjoy the advanced ultra-low latency audio and video technology of ZEGOCLOUD. This document describes how to create projects in the ZEGOCLOUD Admin Console.

## Quick way
### UIKits project
UIKits refer to pre-built UIKits and UI components launched by ZEGOCLOUD based on years of audio and video call practice for users' business scenarios. You can quickly develop apps using UIKits.

1. Log in to the ZEGOCLOUD Admin Console. On the homepage, click the **+** icon above **Create your project**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT1.png" />
</Frame>


2. Select the corresponding use case and click **Next**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT2.png" />
</Frame>

<Note title="Note">

Some use cases support only the creation of ordinary projects instead of UIKits projects. ZEGOCLOUD will support more scenarios in the future.
</Note>


3. Enter a project name and select the UIKits method for project creation.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT3.png" />
</Frame>

4. Wait until the creation completes.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT4.png" />
</Frame>

5. Different from an ordinary project, a UIKits project can be quickly integrated using only a few lines of code. Now, let's select a framework to be supported by the project. (A web framework is used as an example.)
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT5.png" />
</Frame>

6. On the **UI Configuration** page, you can complete the basic configurations on the right of the page based on your needs. The UI preview on the left will change along with the configurations. After you complete the configurations, save the configurations and go to the next step.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT6.png" />
</Frame>

7. A demo generated based on the configurations is provided, and you can start to experience it by sharing the link or joining the room.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT7.png" />
</Frame>


8. If no issues exist during the experience, click **Get the config file & Integrate** to obtain the corresponding configuration file for integration. Multiple integration methods are supported.
    a. Click **Download HTML file** to download the corresponding HTML file to be integrated into your project. ZEGOCLOUD will provide all information you need.
    b. Click **One-click deployment** for one-click deployment of your app through an online code management platform.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT8.png" />
</Frame>

9. If you want to experience other platforms or use cases, select the platform or click **Select the use case you want to add**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_UIKIT9.png" />
</Frame>

10. Now, the UIKits project creation is totally completed.

### Ordinary project

Different from UIKits projects, there are no integration or experience steps after the creation of ordinary projects. The first two steps are the same as those of creating a UIKits project. In step 3, select the **SDKs method**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_Ordinary.png" />
</Frame>
Wait until the creation completes.

## General way

In addition to quickly creating a project on the homepage of the ZEGOCLOUD Admin Console, you can also create projects on the **Projects Management** page.
1. Log in to the ZEGOCLOUD Admin Console. In the navigation pane on the homepage, choose **Projects > Projects Management**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_General1.png" />
</Frame>

2. Click **+** Create to start project creation. You can refer to quick project creation for specific steps.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/CreateProject_General2.png" />
</Frame>


<Note title="Note">

1. Each user can create up to 20 projects. To create more projects, contact ZEGOCLOUD by sending an email to **global_support@zegocloud**.
2. After project creation, the Voice & Video Call and Super Board services will be activated by default. Activation does not incur fees. The fees will be generated only after you use these services.
</Note>
