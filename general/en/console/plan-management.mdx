# How to query the usage of packages

- - -


## Introduction
To query the usage of a package after you purchase it, you can refer to this document. You can query both the packages you purchased and complimentary packages in the ZEGOCLOUD Admin Console.

## Procedure
### Query paid packages
> Paid packages: packages that you have purchased. Unlike complimentary packages, the purchase of paid packages is restricted. If you encounter purchase failure, contact ZEGOCLOUD by sending an email to **global_support@zegocloud.com**.

1. Log in to the ZEGOCLOUD Admin Console. In the navigation pane on the homepage, choose **Package > Package Query**. On the page that appears, the **Purchased packages** tab is displayed by default, which shows the following information:
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/Packages/PaidPackages1.png" />
</Frame>


- Package type. You can filter packages by type. Currently, the types supported are Voice Call, Video Call, Live Streaming, Interactive Live Streaming, Livestream Creation, Super Board, In-app Chat, AI Effects, and Support Plan.
- Package name.
- package usage, including the total volume, the consumed volume, and the percentage of the consumed volume in the total volume.
- Purchase date. You can filter packages by purchase date.
- Effective date and expiration date. You can sort packages by the two dates and the purchase date.
- Package status, which can be Not Used, In Use, Expired, or Used Up.


2. For packages in the Used Up and Expired states, you can click **Purchase** to quickly purchase the new package you need.

### Query complimentary packages
> Complimentary packages: There are two types of complimentary packages. One is a 30-day package you receive as a new user upon registration, with 10,000 minutes for Voice & Video Call. If the package is in the Used Up or Expired state, you can renew the package in the console. The other is a monthly complimentary package you receive with 10,000 minutes for Voice & Video Call for free after your account becomes a standard account. This package can be used to offset the usage of related products. For specific rules for the 10,000-minute usage, see [Deduction rules for the Complementary Package with 10,000 free minutes](./faqs/pricing/calculate-min.mdx).


1. Log in to the ZEGOCLOUD Admin Console. In the navigation pane on the homepage, choose **Package > Package Query**. On the page that is displayed, click the **Complementary packages** tab.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/Packages/FreePackages1.png" />
</Frame>

- You can view the usage of free minutes for Voice & Video Call of the current month, which includes 10,000 minutes for new users (trial accounts), of the Super Board package for certain users, and of the monthly package.
- You can view information such as the type (package for trial account and monthly package), name, start date, and usage of complementary packages
- The monthly complimentary package will take effect on the 1st of each month and expire at the end of that month.

2. You can click View usage details to view the usage details of monthly complementary packages for the current month and previous months.
