# Overview

- - -


ZEGOCLOUD Admin Console is a portal for you to use ZEGOCLOUD products and services such as project management, user management, service management, and billing. This document briefly introduces the core functions of the ZEGOCLOUD Admin Console.

## Homepage

On the homepage of ZEGOCLOUD Admin Console, you can view important information to be focused on and quickly access the corresponding function page.
- Create your project: Click the **+** icon above Create your project on the homepage to quickly enter the project creation process, and use UIKits or other methods to quickly experience the product functions.
- Bills overview: View information about packages, balance, and technical support plans related to your account bills.
- Resources: Quickly access various resource portals and access the product help in one-click mode.
- Function entries: Click an item in the navigation pane on the left to quickly access the corresponding function page.

## Core functions

ZEGOCLOUD Admin Console provides the following functions.

### Project management
Create and manage projects (including those of the UIKits type and common type) and obtain AppID, configurations (such as ServerSecret and Sever URL), and Temporary Token for these projects. For details, see [How to create projects\|_blank](!console_new-Console_createproject).

### Service management
Enable and manage related products and services and configure information about these services based on the project requirements. You need to focus on the Live Streaming and In-app Chat services. You can configure the services based on the actual requirements. For details, see [How to use the Live Streaming Service\|_blank](!console_new-SM_LiveStreaming). and [How to activate the In-app Chat service\|_blank](!console_new-SM_InappChat).

### Analytics dashboard
Analytics Dashboard (AD) is an audio and video quality operations platform of ZEGOCLOUD that helps you comprehensively monitor audio and video services, restore the call quality and interaction scenarios, and easily and effectively locate and troubleshoot issues to improve the audio and video quality and user experience. For details, see [Analytics Dashboard - User Behavior Monitoring \|_blank](!console_new-User_behavior_monitoring).

### Account management
When you register an account with ZEGOCLOUD Admin Console, the account is the main account. You can create sub-accounts with specific permissions for project members to assist the main account in the scenarios such as project integration and project operations. For details, see [How to manage members and roles\|_blank](!console_new-UsersandPermissions).

### Billing center
After using ZEGOCLOUD products and services, you can use the Billing Center function to recharge the account online, query corporate transfer information, query income and expenditure details, and check bills. For details, see [Billing Center\|_blank](!console_new-BillingCenter_Bills).

### Package management
Various service packages are provided for you to enjoy the benefits of ZEGOCLOUD. You can purchase the packages and query the package usage. For details, see [Packages\|_blank](!console_new-Package_PackageSubscribe).

### Usage statistics
After service activation on ZEGOCLOUD Admin Console, you can view detailed usage data for each service. For details, see **Usage Statistics** (coming soon).
