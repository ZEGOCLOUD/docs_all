# How to activate the In-app Chat Service

- - -


## Introduction to the In-app Chat service

In-app Chat is a real-time communication and interaction service launched by ZEGOCLOUD, which features all-platform interaction, high concurrency, ultra-low latency, and message reachable. It applies to scenarios such as large-scale live streaming, voice chatroom, and customer service systems.

ZEGOCLOUD provides a solution that combines In-app Chat Kit and Call Kit and sample source code for scenarios such as live audio room and live streaming. Developers can build perfect business scenarios in a very short time using the solution and sample code. For details, see [Use in conjunction with Call Kit\|_blank](https://www.zegocloud.com/docs/uikit/in-app-chat-kit-android/advanced-features/use-in-conjunction-with-call-kit).

The ZEGOCLOUD Admin Console provides multiple editions of In-app Chat. For details about the differences and prices, see [In-app Chat pricing information\|_blank](https://www.zegocloud.com/pricing/product-list#in-app-chat).

## Activate the In-app Chat service

The following section describes how to activate the In-app Chat service for the living streaming project.
1. Log in to the ZEGOCLOUD Admin Console. Go to the project details page directly from the homepage or by choosing **Projects > Projects Management**, where you need to click the corresponding project in the **My Projects** pane.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/ServiceManagement/SM_InAppChat1.png" />
</Frame>


2. On the **Service Management** tab, click **In-app Chat**, check the scenarios, read the notes, and click **Activate**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/ServiceManagement/SM_InAppChat2.png" />
</Frame>

3. Read the user agreement, and click **Ready to activate**.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/ServiceManagement/SM_InAppChat3.png" />
</Frame>

4. Wait until the In-app Chat service is successfully activated. On the In-app Chat tab, you can view related information and manage the service version.
<Frame width="auto" height="auto">
  <img src="https://doc-media.zego.im/sdk-doc/Pics/A_AdminConsole/ServiceManagement/SM_InAppChat4.png" />
</Frame>
