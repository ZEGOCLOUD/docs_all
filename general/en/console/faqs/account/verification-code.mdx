# Fail to receive a verification code via email

- - -


When you perform any of the following operations on ZEGOCLOUD Admin Console, you will receive a verification code sent by ZEGOCLOUD to verify the email address you entered:
1. Account registration
2. Password retrieval
3. Password resetting
4. Other operations that trigger security verification

Generally, a verification code consists of six digits. You can perform subsequent operations after the verification code is entered and verified.

If you fail to receive a verification code, you can perform the following steps:

1. Check your spam folder or ad email group of your mailbox for emails with the subject "[ZEGOCLOUD] Email Verification Code". If no such email can be found, go to the next step.
2. Check whether you have added the email address **system@no-reply.zegocloud.com** to the whitelist. If yes, request to send the verification code again. If the fault persists, go to the next step.
3. Contact ZEGOCLOUD for help by sending an email to **global_support@zegocloud.com**.
