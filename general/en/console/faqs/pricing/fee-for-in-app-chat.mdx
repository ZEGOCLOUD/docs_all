# How to calculate the fee for In-app Chat

- - -


## Fee differences among In-app Chat editions
Basic, Pro, and Enterprise editions of In-app Chat are provided. The functions of different editions are almost the same. You can select an edition based on your business scale. Each edition contains an initial quota of MAUs. If the number of your MAUs exceeds the initial quota, you need to pay for the excessive MAUs.

|Edition	|Basic |	Pro|	Enterprise|
|-|-|-|-|
|Pricing	|Free|	$449|	Contact ZEGOCLOUD|
|Monthly active users	| Up to 100|	Up to 10,000	|Customized|
|Maximum number of group members	|200 members/group	|200 members/group	|Customized|
|Number of days for historical message storage|	7 days	|180 days	|Customized|

## MAU calculation
When a single user logs in to the In-app Chat service, it will be counted as one MAU. During the month, if the user changes devices and logs in repeatedly, it will be recorded as the same MAU. The number of MAUs is counted from the first day of each month. If the service period is **shorter than a month (for example, you enable or disable the service on the 15th day of the same month), it will be counted as an entire month**.

## Excessive MAUs
The MAU quota varies with the edition.

1. In the Basic edition, the MAU quota is 100. When the number of MAUs exceeds 100, the server declines the subsequent login requests. To continue with your use, upgrade to the In-app Chat edition.
2. In the Pro edition, the MAU quota is 10,000. When the number of MAUs exceeds 10,000, additional fees will be deducted according to the following rules:
    - $449 is charged for every 10,000 MAUs.
    - For a count that is less than 10,000, calculate fees based on 10,000 MAUs.
    
For example, if you use the Pro edition and the number of your MAUs is 32,000, the number of MAUs exceeding the quota is 22,000, which is greater than 20,000 and less than 30,000. The number of excessive MAUs is counted as 30,000. The fee to be paid for the month is calculated based on the following formula:
In-app Chat edition fee + Excessive MAU fee = $449 + 2 x $449 = $1347
