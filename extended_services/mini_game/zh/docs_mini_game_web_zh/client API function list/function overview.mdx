# 功能总览

- - -

ZEGO SDK 拥有全面覆盖全球重点区域的网络节点，通过客户端 API 为多端提供丰富的游戏内容及个性化能力。

API 返回的错误码请参考：[SDK 错误码](../Error Codes.mdx)。


## 前置功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [init](/zh/api?doc=ZegoMiniGameEngine_API~javascript_web~class~ZegoMiniGameEngine#init) | 初始化实例。
[getAllGameList](/zh/api?doc=ZegoMiniGameEngine_API~javascript_web~class~ZegoMiniGameEngine#get-all-game-list) | 获取游戏列表。
[getGameList](/zh/api?doc=ZegoMiniGameEngine_API~javascript_web~class~ZegoMiniGameEngine#get-game-list) | 根据游戏模式获取游戏列表。
[getGameInfo](/zh/api?doc=ZegoMiniGameEngine_API~javascript_web~class~ZegoMiniGameEngine#get-game-info) | 获取游戏详细信息。
[setGameContainer](/zh/api?doc=ZegoMiniGameEngine_API~javascript_web~class~ZegoMiniGameEngine#set-game-container) | 设置加载游戏的容器 |
