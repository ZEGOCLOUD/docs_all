# 集成 SDK

- - -

## 准备环境

在开始集成 ZegoMiniGameEngine SDK 前，请确保开发环境满足以下要求：

- 准备一台可以连接到互联网的 Windows 或 macOS 计算机。
- 使用最新版本的 Google Chrome 浏览器。

## 集成 SDK

### 集成 ZEGO MiniGameEngine SDK

<Steps>
<Step title="通过 npm 安装 SDK。">
    在您的项目文件夹，打开终端，执行 `npm i zego-mini-game-engine` 命令安装依赖。

    <Note title="说明">
      - npm 下载包支持 typescript 语言（推荐）。

      - 如果在 macOS 或 Linux 系统中执行 npm 命令失败，提示 “permission denied”，请在 npm 命令前加上 `sudo` 重新执行即可。
    
    </Note>

</Step>
<Step title="引入 SDK。">
    ```js
    import { ZegoMiniGameEngine } from 'zego-mini-game-engine'
    ```
</Step>
<Step title="在您需要挂载游戏的页面中添加容器，并设置 `id`。">
    ```html
    <template>
        <div>
            // 弹幕游戏的容器
            <div id="cloudGame"></div>
        </div>
    </template>
    ```
</Step>
</Steps>


### 集成 ZEGO Express SDK

在实现主播在浏览器开播弹幕游戏之前，还需要额外集成 ZEGO Express SDK 搭配使用。请参考 [实时音视频 - 集成 SDK](https://doc-zh.zego.im/article/199)。如果您的项目已集成相关 SDK，可忽略此步骤。
