## 开始弹幕游戏

#### PC 端开播场景

主播仅需安装弹幕游戏的 PC 客户端即可开始游戏。

#### 移动端开播场景

您需要先获取弹幕游戏的游戏启动码，再调用 [startCloudGame](https://doc-zh.zego.im/article/api?doc=ZegoMiniGameEngine_API~objective-c_ios~class~ZegoCloudGameEngine#start-cloud-game-game-id-room-id-game-config-callback) 接口启动弹幕游戏，接口调用示例清参考 [主播在移动端开播弹幕游戏 - 主播开始游戏](../quick%20start/Damaku%20Games%20Quick%20Start/Live%20streaming%20games%20on%20mobile.mdx#客户端主播开始游戏)。
