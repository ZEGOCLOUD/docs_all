# 功能总览

- - -

ZEGO SDK 拥有全面覆盖全球重点区域的网络节点，通过客户端 API 为多端提供丰富的游戏内容及个性化能力。

API 返回的错误码请参考：[SDK 错误码](../ErrorCodes.mdx)
