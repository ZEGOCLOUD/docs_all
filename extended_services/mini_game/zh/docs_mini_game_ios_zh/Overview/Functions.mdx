## 弹幕游戏

### 通用流程

<table>
  
<tbody>
<tr>
<th>功能</th>
<th>描述</th>
</tr>
<tr>
<td>获取游戏启动码</td>
<td>需要在开始游戏之前，获取唯一的游戏启动码，从而将游戏与主播、房间等信息进行关联，</td>
</tr>
<tr>
<td>评论、点赞、送礼</td>
<td>将观众在直播间中的评论、点赞、送礼行为告知服务端，从而产生游戏效果。</td>
</tr>
</tbody>
</table>

### 云渲染开播附加流程

<table>
  
<tbody>
<tr>
<th>功能</th>
<th>描述</th>
</tr>
<tr>
<td>开启云游戏</td>
<td>将云端的弹幕游戏资源进行加载，从而主播侧可以控制云端的游戏。</td>
</tr>
<tr>
<td>结束云游戏</td>
<td>关闭当前主播的云游戏，释放资源。</td>
</tr>
<tr>
<td>开始转推游戏画面</td>
<td>将云端游戏的画面转推到 ZEGO RTC 的房间中，并能够被 RTC 房间中的其他观众拉流观看。</td>
</tr>
<tr>
<td>结束转推游戏画面</td>
<td>结束将云端游戏的画面转推到 ZEGO RTC 的房间中。</td>
</tr>
</tbody>
</table>
