# API 概览

- - -

ZEGO 小游戏平台产品服务端 API 支持全球就近接入、统一的鉴权方式、统一的参数风格和公共错误码，给开发者带来简单快捷的使用体验。

## 弹幕游戏相关

ZEGO 小游戏平台产品服务端提供以下与弹幕游戏相关接口：

| 名称 | 描述 | QPS 限制 |
| -- | -- | -- |
| [DescribeGameLaunchCode](./bullet%20comments%20games%20related/Get%20Activation%20Code.mdx) | 获取游戏启动码。 | 20 次/秒 |
| [SendDanmakuLikes](./bullet%20comments%20games%20related/Like.mdx) | 点赞。 | 100 次/秒 |
| [SendDanmakuComments](./bullet%20comments%20games%20related/Comments.mdx) | 发送评论。 | 50 次/秒 |
| [SendDanmakuGifts](./bullet%20comments%20games%20related/Send%20Gifts.mdx) | 送礼。 | 20 次/秒 |
| [DescribeDanmakuMsgResults](./bullet%20comments%20games%20related/Check%20Gift%20Sending%20Results.mdx) | 查询礼物发送结果。 | 20 次/秒 |
