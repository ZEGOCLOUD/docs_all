# 点赞

- - -
## 描述

调用此接口触发弹幕游戏内的点赞事件，会在弹幕游戏客户端产生相应的效果。

<Note title="说明">

- 游戏平台会尽量保证点赞的有效性。
- 若用户未加入游戏，则用户的点赞会被丢弃。
- 请以房间的维度，合并用户的点赞消息，再请求至此接口，建议每 50-200ms 整合一次用户的点赞信息。

</Note>

## 接口原型

- 请求方法：POST
- 请求地址：`https://mini-game-api.zego.im/?Action=SendDanmakuLikes`
- 传输协议：HTTPS
- 调用频率限制：100 次/秒（以房间为维度）
