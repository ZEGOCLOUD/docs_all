# 上报弹幕游戏信息

- - -
## 描述

通过本回调，您可在弹幕游戏开始、弹幕游戏结算时收到具体的事件通知。

## 回调说明

- 请求方法：POST。
- 请求地址：请联系 ZEGO 技术支持配置回调地址。
- 传输协议：HTTPS。
