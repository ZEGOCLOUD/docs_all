
# 返回码
---

调用小游戏平台服务端 API 后，服务端都会返回携带 Code 字段的响应结果，通过该字段可以判断请求是否成功。

| 错误码 | 描述说明 | 处理建议 |
| ---- | ---- | ---- | 
| 0 | 请求成功。 | - | - |
| 7200004 | 参数错误。可能造成的原因包含：<ul><li>缺失必填参数。</li><li>参数名称写错。</li></ul> | 请检查参数。 |
| 7200401 | Token 校验失败。 | 请重新生成 Token。 |
| 7201003 | Token 解码失败。 | 请重新生成 Token。 |
| 7201005 | Token 过期。 | 请重新生成 Token。 |
| 7202000 | 游戏服务接口报错。 | 请重试。若重试后仍提示此错误码，请联系 ZEGO 技术支持。 |
| 7202001 |  游戏未找到。可能造成的原因包含：<ul><li>MiniGameId 输入错误。</li><li>未开通此游戏的权限。</li></ul> | <ul><li>请检查 MiniGameId 是否正确。</li><li>请联系 ZEGO 商务人员开通此游戏的权限。</li></ul> |
| 7202002 |  MiniGameId 格式错误。 | 请确认格式是否为 String。 |
| 7202003 |  AppID 无效。可能造成的原因包含：<ul><li>AppID 未开通小游戏服务。</li><li>AppID 的格式有误。</li></ul> | <ul><li>请联系 ZEGO 商务人员开通小游戏服务权限。</li><li>请确认格式是否为 Uint32。</li></ul> |
| 7206004 |  获取游戏启动码时，房间已被其他主播占用。 | 请确认传入的 AnchorId 与房主是否一致。 |
| 7206005 |  获取游戏码太频繁。| 请稍后再试。 |
| 7203001 |  身份验证失败。 | 请检查 Signature 是否正确。 |
| 7208000 |  游戏业务错误 。 | 请按照错误信息提示进行处理。 |
| 7210000 |  系统错误。 | 请重试。若重试后仍提示此错误码，请联系 ZEGO 技术支持。 |
| 7210001 |  服务异常。 | 请联系 ZEGO 技术支持。 |
