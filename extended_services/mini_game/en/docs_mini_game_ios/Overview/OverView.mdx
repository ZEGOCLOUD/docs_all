# Overview

## What's the ZEGOCLOUD Mini Game Platform?
ZEGOCLOUD Mini Game Platform provides game resources and play-together services for developers through SDK and server-side API. 

It stands out with its small size, rich game variety, and quick integration. 

With this product, you can quickly implement single-player or multiplayer casual, competitive, or other types of mini games in various entertainment scenarios such as live streaming, voice chat rooms, and 1-on-1 chats. 

This enhances the fun and interactivity of the scenes, increases user engagement, and promotes revenue growth.

## Advantages

### Abundant game resources

The mini game platform developed by ZEGOCLOUD provides abundant live comment-based games and is easy to integrate.

## Complete and efficient solution

A complete "bullet screen game + live streaming" solution that covers the entire process and is completely developed by ZEGOCLOUD, helping you to quickly integrate and monetize, with rich customer cases.

## High-quality audio and video PaaS capabilities

ZEGOCLOUD has deeply engaged in the RTC and live streaming fields for many years, strengthening rich foundations, and can provide developers with 500+ global nodes, achieving end-to-end latency as low as 200ms and up to 80% packet loss resistance for audio and video capabilities.

## Multiple live streaming solutions

ZEGOCLOUD provides two live streaming solutions, supporting broadcasters to go live on both PC and mobile platforms (or web browsers), with more flexible scenarios.

### Advantages of PC solution

- Standard live streaming process for hosts. The complete ZEGOCLOUD solution simplifies the process for hosts, from starting games, obtaining the game activation code, adjusting the game streaming parameters, officially starting the game and live streaming, to finally ending the game, thus achieving the optimal live streaming process for hosts.
- We can provide the sample code related to PC live streaming tool.

### Mobile and web browser live streaming (cloud rendering) solution

#### Low integration cost

- Easy to enable and manage services. All service activation and game deployment and operation are handled by ZEGOCLOUD, eliminating many technical integration and cooperation processes.
- Simple resource management. You can focus on the lifecycle of the game, without worrying about the resource lifecycle of cloud rendering.

#### Low business cost

You only need to enable ZEGOCLOUD services and can spend at a lower cost, compared to directly communicate with suppliers of games and cloud rendering services.

#### Better live streaming performance

- Deep compatibility between games and cloud rendering. With optimization and adjustment, games and cloud rendering can better adapt to mobile live streaming, eliminating various errors caused by game incompatibility during live streaming.
- High quality and low latency. The cloud gaming for mobile live streaming has better image quality and lower operation latency. Combined with ZEGOCLOUD's high-quality strategy for RTC, better image quality can be achieved.
- Standard live streaming process for hosts. The complete ZEGOCLOUD solution simplifies the process for hosts, from starting games, obtaining the game activation code, adjusting the game streaming parameters, officially starting the game and live streaming, to finally ending the game, thus achieving the optimal live streaming process for hosts.
