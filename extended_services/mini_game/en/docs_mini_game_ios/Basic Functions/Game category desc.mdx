## Start a live comment-based game

## Host start the game on PC

The host only need to install the PC client of the live comment-based game to start the game.

## Host start the game on mobile devices

The host need to obtain the game activation code for the live comment-based game first, and then call the [startCloudGame](https://doc-en.zego.im/article/api?doc=ZegoMiniGameEngine_API~objective-c_ios~class~ZegoCloudGameEngine#start-cloud-game-game-id-room-id-game-config-callback) interface to start the game. Please refer to the API example in [Implement live comment-based games on mobile devices as a host - Start the game](../quick%20start/Damaku%20Games%20Quick%20Start/Live%20streaming%20games%20on%20mobile.mdx#getGameList).
