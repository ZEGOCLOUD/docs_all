# SDK Integration

---

## Prerequisites

Before integrating the on-premise server recording SDK, ensure your development environment meets the following requirements:

- Any Linux distribution with GLIBC 2.16 or later, supporting x86_64, aarch64, armhf, and armel architectures
- libasound (ALSA).
- libv4l2 (v4l utils).
- JDK 1.7 or later.
- Apache Maven 3.6 or later.

<Note title="Note">

- The SDK depends on libasound (ALSA) and libv4l2 (v4l utils).
- CentOS (RHEL/Fedora): Install by running `yum install alsa-lib-devel libv4l-devel`.
- Ubuntu (Debian/Deepin): Install by running `apt install libasound2-dev libv4l-dev`.
- For other platforms and systems, please install them manually.
- For cross-compilation, refer to [How to cross-compile the Linux alsa-lib dependency?](https://www.zegocloud.com/faq/alsa_lib_cross_compile?product=ExpressAudio&platform=linux) and [How to cross-compile the Linux v4l-utils dependency?](https://www.zegocloud.com/faq/v4l_utils_cross_compile?product=ExpressVideo&platform=linux). The target machine must also have libasound and libv4l2 installed.
</Note>

## Integrate the SDK

Download the latest version of the SDK through the [Download](../DownloadSDK.mdx) page.

<Note title="Note">

After integrating the SDK, the runtime depends on native dynamic libraries. It is recommended to follow the instructions in [Run the Demo Source Code](../DownloadDemo.mdx).
</Note>

1. Open IntelliJ IDEA and open your Java project.

2. Add the SDK reference to your project.

    a. In IDEA, go to "File > Project Structure..." from the menu bar.

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Linux/ServerRecording/Java/5.jpeg" /></Frame>

    b. In the "Project Structure" dialog, select "Libraries", click "+", and select "Java".

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Linux/ServerRecording/Java/6.jpeg" /></Frame>

    c. Select the SDK jar file and click "Open" to import it.

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Linux/ServerRecording/Java/7.jpeg" /></Frame>
