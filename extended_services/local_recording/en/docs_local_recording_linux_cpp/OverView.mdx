# Overview
- - - 
ZEGOCLOUD Local Server Recording SDK is a recording plugin designed for real-time voice/video calls and live streaming. Through simple integration and usage, it helps developers quickly and flexibly deploy local recording services to implement recording of one-to-one or many-to-many real-time voice/video calls or live streaming sessions.

ZEGOCLOUD Local Server Recording SDK can record voice chats, video chats, and live streaming content, allowing more people to watch at a convenient time.

## Key Features

| Feature | Description | 
|--------|---------|
| Single-Stream Recording | Supports recording individual audio/video streams for each user in a room. | 
| Mix-Stream Recording | Supports mixing and recording all audio/video streams from all users in a room. |
| Multi-Format Recording | Supports simultaneously outputting both audio and video files in a room. |
| Audio-Only Extraction | Supports extracting and recording only the audio from audio/video streams in a room. |
| Blank Frame Fill | Supports filling in blank frames when a user's stream is interrupted in a room. |
| Video Screenshot | Supports taking screenshots of all users' audio/video streams in a room. |
| Custom Layout | Supports custom layout for room video output. |
| Recording Watermark | Supports adding image watermarks, text watermarks, and timestamp watermarks to room video. |

## Use Cases

| Industry | Use Case | 
|------|--------|
| Online Education | Provides multi-party audio/video recording for 1-on-1 sessions, small classes, and large classes. |
| Social Live Streaming | Supports live clip recording, live replay, and screenshot-based content moderation. |
| Financial Services | Provides audio/video recording for video signing, remote witnessing, and other business processes for archiving. |
| Customer Service Centers | Recorded audio/video can be used for user research, agent evaluation, and other scenarios. |
| Telemedicine | Records remote consultations and online advisory sessions for later review. |

## Product Features

| Feature | Description |
|------|-----|
| High Reliability | Supports cluster deployment and dynamic scaling for high availability. |
| High Security | Provides end-to-end security for video calls, data transmission, and data storage. |
| Compatibility | Supports recording real-time audio/video from ZegoLiveRoom SDK, ZegoAudioRoom SDK, and ZegoExpress SDK. |
| Adaptability | Supports CentOS 6.5+ x64 and Ubuntu 14.04+ x64 operating systems. |
| Stable and Easy to Use | Simple and easy to operate, quick to get started, with flexible deployment of recording services for mobile and web recording. |
| Flexible Combination | By flexibly combining various features, it can be seamlessly applied to multiple scenarios for more comprehensive services. |
