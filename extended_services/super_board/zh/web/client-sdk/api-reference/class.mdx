## ZegoLoadOption

预览文件加载配置项

**详情**

设置加载动态文件回调方法触发时机。通常情况下从文件服务器获得加载文件的信息后马上触发回调。

<Note title="">
- **业务场景**：获取加载动态 PPT 文件准备就绪状态
</Note>

## ZegoSuperBoardManager

**详情**

接口类，SDK的初始化，房间内全局性的功能设置, 创建文件或者纯白板等。

<Note title="">
- **业务场景**：主要用于初始化SDK，设置白板的一些工具、缩放同步、画笔效果，在开启SuperBoardView模式下,获取SuperBoardView, 创建纯白板、文件等。
- **平台差异**：无。
</Note>

3. 当用户强制刷新页面，旧缓存可能会被清除掉，或者禁用浏览器缓存的时候，该缓存也不会生效。
