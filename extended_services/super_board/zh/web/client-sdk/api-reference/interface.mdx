## ReloadOptions

重载配置项

**详情**

该配置，可以对重新加载 subview 时进行一些配置，例如是否触发强制刷新

<Warning title="">
- **支持版本**：2.7.1及以后
</Warning>

### 属性
<ParamField
  name="forceReload"
  prototype="forceReload: boolean "
  parent_name="ReloadOptions"
  parent_type="class">
设置是否强制刷新

<Note title="">
- **业务场景**：在在部分 iOS 移动设备上渲染文件异常时，执行该 API 进行重新加载文件的操作。
- **是否必填**：是。

</Note>
</ParamField>
<ParamField
  name="reloadType"
  prototype="reloadType: [reloadType](./enum#reloadtype) "
  parent_name="ReloadOptions"
  parent_type="class">
该配置，可以针对释放 canvas 的范围进行配置

<Note title="">
- **是否必填**：是。
</Note>
</ParamField>

示例：如要将鼠标作用点设置在图标的左上角，则 Y 轴 设置 0。
- **取值范围**：0～32。
- **平台差异**：该属性平台差异请参考 https://developer.mozilla.org/zh-CN/docs/Web/CSS/cursor#specifications。

</ParamField>

</ParamField>
