## RenderImgType

转码图片格式类型

**详情**

上传静态文件默认转码格式为 SVG，可配置转为 PNG 或 SVG

<Note title="">
- **业务场景**：设置 SVG  类型，适合对文件有高清晰度的要求，缺点是转码后的图片文件会较大。设置 PNG 类型，转码后的图片文件体积较小，优化在网络状态较差的情景加载速度，缺点是清晰度相对 SVG 较差。
</Note>

<Warning title="">
- **注意事项**：文件转码出来的体积大小与文件本身包含元素有关。建议包含图片元素较多的文件可使用 PNG 格式，如若是文本文件可使用 SVG。
</Warning>

设置转码后的文件为 PNG 格式
</ParamField>
<ParamField

  parent_name="RenderImgType"
  parent_type="enum">
设置转码后的文件为 SVG 格式
</ParamField>
