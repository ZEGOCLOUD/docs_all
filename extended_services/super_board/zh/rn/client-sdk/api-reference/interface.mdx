## ZegoCursorAttribute

详情描述：自定义光标可个性化属性。

### 属性
<ParamField
  name="iconPath"
  prototype="iconPath: string | File "
  parent_name="ZegoCursorAttribute"
  parent_type="class">
光标路径，可输入网络图片或者本地上传图标文件。
</ParamField>
<ParamField
  name="offsetX"
  prototype="offsetX: number "
  parent_name="ZegoCursorAttribute"
  parent_type="class">
鼠标作用点 X 轴偏移

<Note title="">
- **是否必填**：是
    
示例：如要将鼠标作用点设置在图标的左上角，则 X 轴 设置 0。
- **取值范围**：0～32。
- **平台差异**：该属性平台差异请参考 https://developer.mozilla.org/zh-CN/docs/Web/CSS/cursor#specifications。
</Note>
</ParamField>
<ParamField
  name="offsetY"
  prototype="offsetY: number "
  parent_name="ZegoCursorAttribute"
  parent_type="class">
鼠标作用点 Y 轴偏移

<Note title="">
- **是否必填**：是
    
示例：如要将鼠标作用点设置在图标的左上角，则 Y 轴 设置 0。
- **取值范围**：0～32。
- **平台差异**：该属性平台差异请参考 https://developer.mozilla.org/zh-CN/docs/Web/CSS/cursor#specifications。
</Note>
</ParamField>
<ParamField
  name="titleStyle"
  prototype="titleStyle: ZegoSuperBoardCursorTitleStyle "
  parent_name="ZegoCursorAttribute"
  parent_type="class">
光标自定义文字信息

<Note title="">
- **是否必填**：否

- **默认值**：参考 ZegoCursorTitleStyle 类

</Note>

<Warning title="">
- **支持版本**：v2.13.0
</Warning>
</ParamField>

## loadOptions

加载文件配置

**详情**

该配置，可以对加载文件时进行一些配置，例如 触发动态 PPT 的加载回调时机

<Warning title="">
- **注意事项**：注意每个配置生效条件，可能某些配置项只针对某类型文件生效。
</Warning>

### 属性
<ParamField
  name="PPTReady"
  prototype="PPTReady: boolean "
  parent_name="loadOptions"
  parent_type="class">
设置加载动态文件回调方法触发时机。通常情况下从文件服务器获得加载文件的信息后马上触发回调。

<Note title="">
- **业务场景**：获取加载动态 PPT 文件准备就绪状态。
- **是否必填**：否。
- **默认值**：false，默认从文件服务器获得文件信息后立即触发。
</Note>

<Warning title="">
- **注意事项**：该参数仅对动态 PPT 文件生效。

</Warning>
</ParamField>

## uploadOptions

**详情**

针对该上传的文件，进行转码配置

### 属性
<ParamField
  name="renderImgType"
  prototype="renderImgType: [RenderImgType](./enum#renderimgtype) "
  parent_name="uploadOptions"
  parent_type="class">
针对静态文件类型生效，可配置不同参数进行转码为对应的图片格式类型
</ParamField>

## ~

重载配置项

**详情**

该配置，可以对重新加载 subview 时进行一些配置，例如是否触发强制刷新

<Warning title="">
- **支持版本**：2.7.1及以后
</Warning>

### 属性
<ParamField
  name="forceReload"
  prototype="forceReload: boolean "
  parent_name="~"
  parent_type="class">
设置是否强制刷新

<Note title="">
- **业务场景**：在在部分 iOS 移动设备上渲染文件异常时，执行该 API 进行重新加载文件的操作。
- **是否必填**：是。

</Note>
</ParamField>
<ParamField
  name="reloadType"
  prototype="reloadType: reloadType "
  parent_name="~"
  parent_type="class">
该配置，可以针对释放 canvas 的范围进行配置

<Note title="">
- **是否必填**：是。
</Note>
</ParamField>
