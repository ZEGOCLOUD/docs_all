# API overview

- - -
The Super Board server provides the following Server APIs.

### Server APIs related to permissions on a whiteboard

|API|Description|
|-|-|
|[Set room permissions](./set-white-board-room-auth.mdx)|Sets default room permissions on a whiteboard. |
|[Set user permissions](./set-white-board-user-auth.mdx)|Sets user permissions on a whiteboard. |
|[Query room permissions](./get-white-board-room-auth.mdx)|Queries room permissions on a whiteboard. |
|[Query user permissions](./get-white-board-user-auth.mdx)|Queries user permissions on a whiteboard. |
