# 功能介绍

---

借助多维分析，您可从不同维度对用户体验、服务质量、业务规模等各类指标进行分析。针对每个指标，还能查看对应的明细数据，从具体用户、流、房间层面切入，辅助推拉流问题分析。


## 功能概述

多维分析包括3类子页面：

| 子页面-15% | 功能 |
| -- | -- |
| 指标看板页 | 可选取一段时间内用户体验、服务质量、业务规模等不同类别的指标，以时间趋势图展示，用于发现问题。<br/>针对已选取的指标，还可通过对比、过滤等方式进行分析查看，以进一步剖析问题原因。 |
| 多维分析页 | 从数据分布、国家地区、网络类型、平台系统、SDK版本、拉流源、运营商等展示指标在不同维度的分布，用于帮助您开展指标下钻，以追溯原因，锁定问题根源。 |
| 数据明细页 | 用于进一步查看该指标在所选时间和维度下的具体用户、流或房间信息，可快速获取分析样例，辅助问题解决。 |

## 访问步骤

1. 登录 [ZEGO 控制台](https://console.zego.im/)，从左侧导航栏进入星图；
2. 在导航栏点击“多维分析”，即可访问“指标看板页”。

### 功能介绍
#### 多维概览
以饼图或气泡图的形式，展示了指标在已选时间内各维度下的样本量（采样时长或者采样点个数）和指标值。通过提供该指标在不同维度的整体概览情况，帮助快速判断不同因素对指标的影响程度，为排查和解决问题提供方向。点击右上角可往前/往后切换时间。
<Frame width="512" height="auto" >
  <img src="https://doc-media.zego.im/extended_services/analytics_dashboard/zh/all/guides/dimensional-analysis/dashbaord-9.png" />
</Frame>

#### 数据分布
将指标值划分在不同的分布区间，展示每个区间的样本量，帮助用户了解指标的数据集中情况，判断问题严重程度。仅用户体验类指标有此维度。   
下方展示了不同分布区间的数据明细，支持排序和下载。部分指标支持查看数据详情，可点击“查看详情”跳转，查看指定区域的用户/房间/流明细。   
归属用户体验和服务质量分类的指标，支持查看数据详情，可点击“查看详情”跳转，查看指定区间的用户/房间/流明细。

- 右边图表展示了该指标样本量Top10的国家分布，用于了解该appid影响的主要国家/地区以及指标现状。   
- 下方则展示了不同国家地区的数据明细。部分指标支持查看数据详情，可点击“查看详情”跳转，查看指定区域的用户/房间/流明细。
