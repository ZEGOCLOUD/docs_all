# Get Business Usage

- - -

## Description
This interface can be used to query the real-time audio-video data statistics within a specified time period, including room concurrency peak, user concurrency, push and pull stream concurrency, and push and pull stream cumulative total.

<Note title="Note">The query time period is consistent with the <strong>Data Storage and Query</strong> effective period in the package you purchased. For details, please refer to [Pricing - Package Function](/analytics-dashboard/introduction/pricing#package-function).</Note>

## Request method and endpoint

- Request method: GET
- Request URL: `https://analytics-api.zego.im/?Action=GetBizUsage`
- Transfer protocol: HTTPS
- QPS limit (same AppID): 3 times/minute, 480 times/day

## Request Example

```txt
https://analytics-api.zego.im/?Action=GetBizUsage
&StartDate=20250110
&EndDate=20250112
&Metrics[]=publish_count
&Metrics[]=play_count
&<Public Request Parameters>
```

## Return Code

| Return Code | Description | Action |
|--------|------|----------|
| 0      | Success. | -        |
| 500    | Interface error. | Contact ZEGOCLOUD Technical Support. |
| 10001  | Parameter format error. | Check parameters. |
| 10002  | Parameter error or parameter invalid. | Check parameters. |
| 30002  | Interface request frequency exceeds the limit. | Please confirm the QPS limit of the corresponding interface and reduce the request frequency. |
