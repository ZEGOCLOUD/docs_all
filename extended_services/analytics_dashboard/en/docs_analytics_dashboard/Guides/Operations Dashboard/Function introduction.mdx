# Function Introduction

- - -

Operations Dashboard provides you with user quality and usage information, helping you monitor and track changes in business service quality and scale.

- The data update mechanism of Operations Dashboard is T+1, if you need to view real-time data, please refer to [Real-time Monitoring](./../Real-time%20monitoring.mdx).
- For the meaning of each data indicator, please refer to [Data Description](./Data%20description.mdx).
- The CDN analysis related functions are currently in the Beta stage, for reference only.

## Access Steps

<Note title="Note">

The main account automatically opens the "Operations Dashboard" function, and the sub-account needs to be configured by the main account before it can use this function.
</Note>

1. Log in to the [ZEGOCLOUD Console](https://console.zego.im/), click the "Star Graph" in the left navigation bar.
2. After entering Star Graph, click the "Operations Dashboard" in the left navigation bar, expand its sub-options, as follows:
    - [Quality Trend](#quality-trend)
    - [Region Quality](#region-quality) or [Region Quality (Advanced)](./Regional%20quality(Pro).mdx)
    - [Operation Scale](#operation-scale)
    - [User Portrait](#user-portrait)
3. Select any option, enter the page. For example, select "Quality Trend", select AppID and the time range you want to query, click "Search", and you can view the related data.


## Quality Trend

The "Quality Trend" page helps you track and view business performance from a macro perspective, achieving the following functions:

- In the "User Experience" module, evaluate the basic situation of user experience, view data changes at different time periods (fluency, entry room delay, etc.), and analyze the trend of audio/video experience.
- In the "Service Availability" module, view the status of ZEGOCLOUD room service and media service, evaluate service availability, and analyze whether there are service exceptions in the current network.

## Region Quality

In the "Region Quality" page, you can view the audio/video quality of different regions, view the quality change trend of a specific region, help you monitor and analyze the operational quality of each region; at the same time, support viewing the end-to-end pull stream details of a specific region, understand the pull stream distribution of the region.

<Warning title="Warning">

The region quality monitoring function is divided into basic and advanced two types, **free version users only support using basic functions, professional version and enterprise version users support using advanced functions.**

This document only introduces the basic functions, if you want to learn about the advanced functions, please refer to [Region Quality (Advanced)](./Regional%20quality(Pro).mdx).
</Warning>

## Operation Scale

The "Operation Scale" page provides you with the daily number of rooms, users, and push/pull streams, helping you monitor and track the changes in business scale in real time.

### Sample Page

<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Prism/Operation_Dashboard_intro.png" /></Frame>

### Function Introduction

**User and Room**

In the "User" and "Room" two modules, you can view the daily active users and concurrent users of the user and room, used to analyze the changes in user scale. The "User Daily Active" chart supports viewing daily data according to three different dimensions: "All Users", "Push Stream Users", and "Pull Stream Users".

<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Prism/yonghurihuoshujuleixing_EN.png" /></Frame>

**Push/Pull Stream**

In the "Push/Pull Stream" module, you can view the daily cumulative number and concurrent number of push/pull streams, view the push stream resolution, video encoding format, and pull stream source distribution details, so as to understand the basic scale and distribution of daily push/pull streams.

1. In the "Push Stream Number" chart, you can select "Push Stream Target" in the "Filter" drop-down box to view the push stream number corresponding to the user pushing to different target services; select "Region", you can view the push stream number in different regions.

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Prism/tuiliumubiao2_EN.png" /></Frame>

2. In the "Pull Stream Number" chart, you can select "Pull Stream Source" in the "Filter" drop-down box to view the pull stream number corresponding to the user pulling from different services; select "Region", you can view the pull stream number in different regions.

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Prism/laliuyuan3_EN.png" /></Frame>
