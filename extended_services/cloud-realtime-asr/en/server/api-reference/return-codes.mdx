# API Return Codes

- - -

## Common Return Codes

All ZEGO products and services share the same meaning for common return codes.

| Return Code | Description | Handling Suggestion |
| --------- | --------------------------- | ------------------------------------------------- |
| 0 | Success | - |
| 100000000 | Host in request header is incorrect. | Please confirm if the Host address in the request header is correct. |
| 100000001 | AppId field format error. | Please enter the correct AppId. |
| 100000002 | Timestamp field is empty. | Please enter the Timestamp. |
| 100000003 | Timestamp field format error. | Please enter the correct Unix timestamp (in seconds). |
| 100000004 | Signature expired. | Please regenerate the signature. |
| 100000005 | Signature error. | Please confirm if the parameters for generating the signature are correct. |
| 100000006 | Action is empty. | Please specify the Action. |
| 100000007 | Unsupported Action. | Please enter the correct Action. |
| 100000008 | SignatureNonce field is empty. | Please enter the SignatureNonce. |
| 100000009 | Signature field is empty. | Please enter the Signature. |
| 100000010 | Failed to get ServerSecret. | Please confirm if the AppId is correct. |
| 100000011 | Failed to get App configuration. | Please confirm if the AppId is correct, or contact ZEGOCLOUD Technical Support. |
| 100000012 | Failed to read HTTP body. | Please contact ZEGOCLOUD Technical Support. |
| 100000013 | SignatureVersion field is empty. | Please enter the SignatureVersion. |
| 100000014 | Unsupported SignatureVersion. | SignatureVersion only supports 2.0. |
| 100000015 | Gateway internal error. | Please contact ZEGOCLOUD Technical Support. |
| 100000017 | App has not enabled this service. | Please contact ZEGOCLOUD Technical Support. |

## Service Return Codes

Return codes have different meanings for different ZEGO services. The following are the return codes and meanings for the cloud real-time speech recognition service.

| Return Code | Description | Handling Suggestion |
| --------- | -------------------------- | ------------------------------------------------------ |
| 410000001 | System error. | Please contact ZEGOCLOUD Technical Support. |
| 410000002 | AppID is invalid. | Please check if the AppID is correct. |
| 410000003 | Parameter is invalid. | Please check the invalid input parameters according to the Message prompt. |
| 410000004 | JSON parsing failed. | Please check if the request parameters are valid JSON. |
| 410000005 | Gateway authentication failed. | Please contact ZEGOCLOUD Technical Support. |
| 410000006 | App has not enabled this service. | Please contact ZEGOCLOUD Technical Support. |
| 410000009 | Service internal call failed. | Please contact ZEGOCLOUD Technical Support. |
| 410000010 | Insufficient service resources. | Please contact ZEGOCLOUD Technical Support. |
| 410000011 | Exceeded concurrency control. | Please contact ZEGOCLOUD Technical Support. |
| 410001002 | Speech recognition task not found. | Please check if the speech recognition task has stopped or timed out. |
| 410001003 | Failed to start speech recognition task. | Please contact ZEGOCLOUD Technical Support. |
