# Exception Event Error Codes

This document lists the exception event error codes that may be encountered during the operation of real-time speech recognition, along with handling suggestions. For receiving exception event callbacks, please refer to [Receiving Callbacks](./receiving-callback.mdx).

| Event Code | Event Name | Handling Suggestion |
| --- | --- | --- |
| 1001 | General error | Error that cannot be refined, refer to exception message description |
| 1004 | Server VAD detected human speech but speech recognition did not recognize valid text | Monitor statistics or ignore |
| 1201 | RTC general error | Provide Message to ZEGOCLOUD Technical Support for troubleshooting |
| 1202 | RTC room idle exceeds 120 seconds | Check if there are real users in the RTC room |
| 1204 | Failed to login to RTC room | Provide Message to ZEGOCLOUD Technical Support for troubleshooting |
| 1207 | Failed to pull stream | Provide Message to ZEGOCLOUD Technical Support for troubleshooting |
| 2101 | Speech recognition general error | Provide Message to ZEGOCLOUD Technical Support for troubleshooting |
