# 暂停录制

---

## 1 描述

调用本接口可以暂停录制中的任务，暂停期间将不会录制房间内的音视频流、白板等。

<Warning title="Warning">

仅混流录制任务可通过本接口暂停。
</Warning>

## 2 接口原型

- 请求方法：POST
- 请求地址：https://cloudrecord-api.zego.im/?Action=PauseRecord
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

&Timestamp=1234567890
&Signature=7a2c0f11145fb760d607a07b54825013
