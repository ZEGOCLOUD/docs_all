# 更新混流布局

---

## 1 描述

在录制过程中，可以随时调用本接口更改混流布局的设置。

## 2 接口原型

- 请求方法：POST
- 请求地址：`https://cloudrecord-api.zego.im/?Action=UpdateLayout`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 4 请求示例

以下是使用 UpdateLayout 接口更新自定义布局的请求示例。
- 请求 URL  
```
https://cloudrecord-api.zego.im/?Action=UpdateLayout
&AppId=1234567890
&SignatureNonce=15215528852396
&Timestamp=1234567890
&Signature=7a2c0f11145fb760d607a07b54825013
&SignatureVersion=2.0
```
- 请求消息体  
```json
{
	"TaskId": "xxxx",
	"MixMode": 1,
	"MixInputList": [
		{
            "StreamId": "xxxx", 
            "ViewType": 1, 
            "Top": 0, 
            "Left": 0, 
            "Bottom": 720, 
            "Right": 540, 
            "Layer": 1
		},
		{
            "StreamId": "", 
            "ViewType": 1, 
            "Top": 0, 
            "Left": 540, 
            "Bottom": 360, 
            "Right": 1080, 
            "Layer": 1
		},
		{
            "StreamId": "", 
            "ViewType": 1, 
            "Top": 360, 
            "Left": 540, 
            "Bottom": 720, 
            "Right": 1080, 
            "Layer": 1
		}
	]
}
```
