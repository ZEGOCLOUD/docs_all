# 结束录制

---

## 1 描述

调用本接口结束云端录制。

当开发者成功调用 StopRecord 接口后，云端录制服务会退出录制的房间并上传录制文件至设定的第三方云存储。

## 2 接口原型

- 请求方法：POST
- 请求地址：https://cloudrecord-api.zego.im/?Action=StopRecord
- 传输协议：HTTPS
- 调用频率限制：50 次/秒
