# API overview

---

ZEGOCLOUD's Cloud Recording Server provides the following APIs, and the following are their reference documents: 

|API reference doc|Description| Default QPS |
|-|-|-|
|[Start recording](./Start%20Recording.mdx)| Starts cloud recording. | 50 times/second |
|[Stop recording](./Stop%20Recording.mdx)| Stops cloud recording. | 50 times/second |
|[Update the mixed-stream layout](./Update%20the%20Mixed%20Stream%20Layout.mdx)|Updates the mixed-stream layout settings at any time during the recording. | 10 times/second                    |
|[Update the whiteboard](./Whiteboard%20Update.mdx)| Updates the whiteboard at any time during the recording. | 10 times/second         |
|[Query recording status](./Recording%20Status%20Check.mdx)| Queries the status of a recording task after recording starts. | 10 times/second                       |
|[Query the recording task list](./Query%20Record%20Task%20List.mdx)| Queries the list of recording tasks after recording starts. | 10 times/second                       |
|[Pause recording](./Pause%20Recording.mdx)| Pauses a task of recording audio streams, video streams, and whiteboards in a room. | 10 times/second                       |
|[Resume recording](./Recovery%20Recording.mdx)| Resumes a task of recording audio streams, video streams, and whiteboards in a room. | 10 times/second                       |
|[Callback of the recording status](./Recording%20End%20Callback.mdx)| Gets the task status event. You need to provide a callback URL when enabling the Cloud Recording service. | - |
