# Release notes

- - -

## 2021-11-29

**New features**

| Feature | Description | Method |
|-------|-------|-------|
| Query recording tasks | Queries the list of ongoing and completed recording tasks that started within 3 days. |[Query the recording task list](/en/cloud-recording-server/query-record-task-list)|

**Enhancement**

| Feature | Description | Method |
|-------|-------|-------|
| Enhanced the callback of recording status | Supports users receiving an event callback immediately upon an abnormal status change during the recording. |[Callback of the recording status](/en/cloud-recording-server/recording-end-callback)|
| Error codes  | Error codes enhanced. | [Error codes](./Error%20Codes.mdx) |


## 2021-07-23

**New features**

| Feature | Description | Method |
|-------|-------|-------|
| Huawei Cloud OBS supported | Support for saving the recording files to Huawei Cloud OBS. |[Start recording](/en/cloud-recording-server/start-recording)|
| Pause recording feature added|Support for pausing and resuming the stream mixing recording tasks. After pausing the recording, the audio, video streams, whiteboards in the room will not be recorded. |<ul><li>[Pause recording](/en/cloud-recording-server/pause-recording)</li><li>[Resume recording](/en/cloud-recording-server/recovery-recording)</li></ul>|

## 2021-07-02

**New features**

| Feature | Description | Method |
|-------|-------|-------|
|Room ID added to the returned callback |Add a `room_id` field for the callback of stop recording, which is convenient for users to directly associate the recording result to the recorded room. |[Callback of the recording status](/en/cloud-recording-server/recording-end-callback)|

## 2021-06-21
