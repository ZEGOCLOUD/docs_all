# Common Error Codes

- - -

## Overview

When the SDK encounters network, resource loading, file operation, or related errors during runtime, the SDK cannot automatically recover and requires App intervention or user notification. If error information is returned after calling an interface, error information appears in event callbacks, or error logs appear in logs, it indicates that the interface call was abnormal.

Error codes are typically thrown from callbacks such as `onError` and `onIsLoadedError`. Please handle them according to the following error codes.

## Success

| Error Code Enum Value | Description |
|-|-|
| 0 | Execution successful |

|-|-|

| -200001 | Description: Digital human ID is empty or invalid.<br />Possible cause: The passed digital human ID is empty, has incorrect format, or does not exist.<br />Suggested action: Please check if the passed digital human ID is correct, ensure the digital human ID is not empty and the format is correct. |

| -200003 | Description: User ID is empty or invalid.<br />Possible cause: The passed user ID is empty or has incorrect format.<br />Suggested action: Please check if the passed user ID is correct, ensure the user ID is not empty and the format is correct. |

## -200010 ~ -200019 Network Related Errors

Network related errors refer to errors that occur when requesting network resources or communicating with the server, generally manifesting as network connection failures, request timeouts, etc.

| Error Code Enum Value | Description |
|-|-|
| -200010 | Description: General network error.<br />Possible cause: Abnormal network connection, unstable network, etc.<br />Suggested action: Please check if the network connection is normal. It is recommended to switch network environment and retry. |
| -200011 | Description: Request timeout.<br />Possible cause: High network latency, slow server response, etc.<br />Suggested action: Please check if the network connection is normal. It is recommended to switch network environment and retry. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200012 | Description: Network connection lost.<br />Possible cause: Network interruption, WiFi disconnection, etc.<br />Suggested action: Please check if the network connection is normal, ensure the network connection is stable and retry. |
| -200013 | Description: Unable to connect to host.<br />Possible cause: Incorrect server address, network unreachable, firewall blocking, etc.<br />Suggested action: Please check if the server address is correct, check if the network connection is normal, check firewall settings. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200014 | Description: Network request failed.<br />Possible cause: Network abnormality, server error, etc.<br />Suggested action: Please check if the network connection is normal. It is recommended to switch network environment and retry. If the problem persists, please contact ZEGOCLOUD Technical Support. |

## -200020 ~ -200029 Resource Loading Errors

Resource loading errors refer to errors that occur when loading digital human resources, downloading material packages, extracting resources, etc.

| Error Code Enum Value | Description |
|-|-|
| -200020 | Description: General resource loading error.<br />Possible cause: Corrupted resource file, incorrect resource path, etc.<br />Suggested action: Please check if the resource file is complete, check if the resource path is correct. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200021 | Description: Invalid rendering information data.<br />Possible cause: Incorrect rendering information data format, corrupted data, etc.<br />Suggested action: Please check if the rendering information data is correct, ensure the data format meets requirements. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200022 | Description: Download failed.<br />Possible cause: Network abnormality, server error, insufficient storage space, etc.<br />Suggested action: Please check if the network connection is normal, check if device storage space is sufficient. It is recommended to switch network environment and retry. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200023 | Description: File copy failed.<br />Possible cause: Source file does not exist, no permission for target path, insufficient storage space, etc.<br />Suggested action: Please check if the source file exists, check if the target path has write permission, check if device storage space is sufficient. |
| -200024 | Description: Resource extraction failed.<br />Possible cause: Corrupted archive, insufficient storage space, no permission for extraction path, etc.<br />Suggested action: Please check if the archive is complete, check if device storage space is sufficient, check if the extraction path has write permission. If the problem persists, please re-download the resource. |
| -200025 | Description: Material package download failed.<br />Possible cause: Network abnormality, server error, insufficient storage space, etc.<br />Suggested action: Please check if the network connection is normal, check if device storage space is sufficient. It is recommended to switch network environment and retry. If the problem persists, please contact ZEGOCLOUD Technical Support. |

## -200030 ~ -200039 File Operation Errors

File operation errors refer to errors that occur during file reading/writing, cache management, etc.

| Error Code Enum Value | Description |
|-|-|
| -200030 | Description: General file operation error.<br />Possible cause: Incorrect file path, insufficient file permissions, insufficient storage space, etc.<br />Suggested action: Please check if the file path is correct, check if file permissions are sufficient, check if device storage space is sufficient. |
| -200031 | Description: Failed to get cache directory.<br />Possible cause: Application has no permission to access cache directory, cache directory does not exist, etc.<br />Suggested action: Please check if the application has permission to access the cache directory, check if the cache directory exists. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200032 | Description: Failed to delete cache directory.<br />Possible cause: Cache directory is occupied, insufficient file permissions, etc.<br />Suggested action: Please check if the cache directory is occupied by other programs, check if file permissions are sufficient. If the problem persists, please contact ZEGOCLOUD Technical Support. |

## -200040 ~ -200049 Internal Operation Errors

Internal operation errors refer to errors that occur during SDK internal processing.

| Error Code Enum Value | Description |
|-|-|
| -200040 | Description: General internal error.<br />Possible cause: SDK internal processing abnormality.<br />Suggested action: If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200041 | Description: Operation cancelled.<br />Possible cause: User actively cancelled the operation, operation was interrupted, etc.<br />Suggested action: This error usually indicates that the operation was normally cancelled and requires no special handling. If you need to continue the operation, please initiate the request again. |
| -200042 | Description: Duplicate start error.<br />Possible cause: Repeatedly started while an instance is already running.<br />Suggested action: Please ensure to stop or destroy the previous instance before starting to avoid duplicate starts. |

## -200050 ~ -200059 Backend Service Errors

Backend service errors refer to errors that occur when communicating with ZEGO backend services.

| Error Code Enum Value | Description |
|-|-|
| -200050 | Description: General backend error.<br />Possible cause: Backend service abnormality, server error, etc.<br />Suggested action: Please retry later. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200051 | Description: Backend request failed.<br />Possible cause: Network abnormality, server error, incorrect request parameters, etc.<br />Suggested action: Please check if the network connection is normal, check if request parameters are correct. It is recommended to switch network environment and retry. If the problem persists, please contact ZEGOCLOUD Technical Support. |
| -200052 | Description: Invalid backend response.<br />Possible cause: Server returned data with incorrect format, corrupted data, etc.<br />Suggested action: Please retry later. If the problem persists, please contact ZEGOCLOUD Technical Support. |
