# Function Overview

- - -

ZEGO Digital Human Mobile SDK provides high-quality digital human rendering and interaction capabilities. With this SDK, developers can easily integrate real-time interactive digital human avatars into mobile applications.

For API error codes, please refer to: [DigitalMobileErrorCode](./enum#digitalmobileerrorcode).

The following is a list of core API functions.

## Engine Management and Global Configuration

| Method Name | Description |
| --- | --- |
| [create](./class#create-zegodigitalhuman-class) | Create a digital human instance (entry method). |
| [setLogConfig](./class#setlogconfig-zegodigitalhuman-class) | Globally set log save path and rotation configuration. |

## Digital Human Lifecycle Management

| Method Name | Description |
| --- | --- |
| [start](./interface#start-izegodigitalmobile-interface) | Start digital human task based on configuration. |
| [stop](./interface#stop-izegodigitalmobile-interface) | Stop digital human task and release resources. |
| [attach](./interface#attach-izegodigitalmobile-interface) | Bind digital human rendering content to a view control. |
| [getDigitalView](./interface#getdigitalview-izegodigitalmobile-interface) | Get the currently displayed view. |

## Real-time Data Interaction (RTC Integration)

| Method Name | Description |
| --- | --- |
| [onRemoteVideoFrameRawData](./interface#onremotevideoframerawdata-izegodigitalmobile-interface) | Feed raw frame data from remote video stream. |
| [onPlayerSyncRecvSEI](./interface#onplayersyncrecvsei-izegodigitalmobile-interface) | Feed SEI metadata synchronously received by the player. |

## Status Monitoring Callbacks

| Method Name | Description |
| --- | --- |
| [onDigitalMobileStartSuccess](./interface#ondigitalmobilestartsuccess-izegodigitalmobilezegodigitalmobilelistener-interface) | Digital human start success notification. |
| [onError](./interface#onerror-izegodigitalmobilezegodigitalmobilelistener-interface) | Error notification during runtime. |
| [onSurfaceFirstFrameDraw](./interface#onsurfacefirstframedraw-izegodigitalmobilezegodigitalmobilelistener-interface) | First frame rendering notification. |

## Avatar Resource Preload Management

| Method Name | Description |
| --- | --- |
| [preload](./class#preload-zegodigitalhumanresource-class) | Asynchronously download and cache avatar resources for the specified digital human. |
| [isLoaded](./class#isloaded-zegodigitalhumanresource-class) | Check if resources for the specified digital human are ready. |
| [unload](./class#unload-zegodigitalhumanresource-class) | Delete locally cached digital human resources. |

## Resource Management Callbacks

| Method Name | Description |
| --- | --- |
| [onSuccess](./class#onsuccess-zegodigitalhumanresourcepreloadcallback-class) | Preload successful. |
| [onProgress](./class#onprogress-zegodigitalhumanresourcepreloadcallback-class) | Preload progress update. |
| [onError](./class#onerror-zegodigitalhumanresourcepreloadcallback-class) | Preload failed. |
| [onResult](./class#onresult-zegodigitalhumanresourceisloadedcallback-class) | Resource loading status query result. |
