# Download SDK

- - -

ZEGO Express Video SDK is provided by ZEGOCLOUD. You can download the Express-Video SDK for Android clients on this page. The current available version is 3.22.0. For release notes, please refer to [Release Notes](!DownloadSDK/Release_Notes). For compliance matters, please refer to [ZEGO Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

<Warning title="Note">


- Starting from version 3.17.0, Express SDK has upgraded its NDK dependency to NDK r25c. After upgrading, developers need to perform compatibility testing to prevent incompatibility issues.
- If you encounter compatibility issues during testing or need to use an older version of NDK, please contact ZEGOCLOUD Technical Support.
</Warning>
