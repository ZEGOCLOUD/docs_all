# Integrate SDK

---

## Prepare Environment

Before starting to integrate the Digital Human SDK, please ensure your development environment meets the following requirements:

* Android Studio 2021.1.1 or above.
- Android SDK 25, Android SDK Build-Tools 25.0.2, Android SDK Platform-Tools 25.x.x or above.
- Android 6.0 (API 23) or above device or emulator with audio/video capabilities and internet connection.

## Integrate SDK

The digital human functionality is provided by ZEGO Express SDK for basic audio/video capabilities, and the Digital Human SDK for rendering and interaction capabilities. Follow the steps below to integrate.

### 1 Import SDK

#### Integrate ZEGO Express SDK

Since the current digital human functionality requires the ZEGO Express SDK optimized for AI Agent, the following two integration methods are supported:

<Tabs>
<Tab title="Method 1: Maven Integration (Recommended)">

Add the following dependency in the application module `app/build.gradle`:

```groovy
dependencies {
    // ZEGO Express SDK optimized for Digital Human
    implementation 'im.zego:express-private:3.22.0.46726'
}
```

</Tab>
<Tab title="Method 2: Manual Integration">

1. Please get the ZEGO Express SDK optimized for AI Agent from the [download page](./introduction/download.mdx).
2. Refer to the [Integrate Express SDK](https://www.zegocloud.com/docs/real-time-voice-android/quick-start/integrating-sdk) documentation to integrate the SDK into your project.
2. Copy the SDK's `ZegoExpressEngine.jar` file to the application module's `libs` directory.
3. Copy the `.so` files under `armeabi-v7a`, `arm64-v8a` directories to the corresponding `jniLibs` directory (usually `app/src/main/jniLibs`).

</Tab>
</Tabs>

#### Integrate Digital Human SDK

The Digital Human SDK has been published in the Maven repository and can be integrated as follows.

**1. Add Maven Repository**

Select the configuration method based on the Android Gradle Plugin version:

<Tabs>
<Tab title="AGP 7.1.0 and above">

Open `settings.gradle` in the project root directory and add under `dependencyResolutionManagement` > `repositories`:

```groovy
dependencyResolutionManagement {
  repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
  repositories {
      google()
      mavenCentral()
      maven { url 'https://maven.zego.im' }   // ZEGO SDK repository
  }
}
```

</Tab>
<Tab title="AGP below 7.1.0">

Open `build.gradle` in the project root directory and add under `allprojects` > `repositories`:

```groovy
allprojects {
    repositories {
        google()
        mavenCentral()
        maven { url 'https://maven.zego.im' }   // ZEGO SDK repository
    }
}
```

</Tab>
</Tabs>

**2. Add Dependency**

Add the Digital Human SDK dependency in the application module `app/build.gradle`:

```groovy
dependencies {
    // Digital Human SDK
    implementation "im.zego:digitalmobile:1.5.0.119"
}
```

### 2 Initialize SDK

After integrating the SDK, follow the steps below to configure permissions and initialize the ZEGO Express SDK.

<Steps>
<Step title="Add Permission Declarations">
Go to the "app/src/main" directory, open the "AndroidManifest.xml" file, and add permissions.

```xml AndroidManifest.xml
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
<uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.MODIFY_AUDIO_SETTINGS" />
```
</Step>
<Step title="Request Recording Permission at Runtime">

```java
private final ActivityResultLauncher<String> requestPermissionLauncher = registerForActivityResult(
    new ActivityResultContracts.RequestPermission(), new ActivityResultCallback<Boolean>() {
        @Override
        public void onActivityResult(Boolean isGranted) {
            if (isGranted) {
                // Permission granted
            }
        }
    });
// Initiate request
requestPermissionLauncher.launch(Manifest.permission.RECORD_AUDIO);
```
</Step>
<Step title="Create and Initialize ZegoExpressEngine">

```java {3}
ZegoEngineProfile zegoEngineProfile = new ZegoEngineProfile();
zegoEngineProfile.appID = ; // AppId obtained from ZEGOCLOUD Console
zegoEngineProfile.scenario = ZegoScenario.HIGH_QUALITY_CHATROOM;
zegoEngineProfile.application = getApplication();
ZegoExpressEngine.createEngine(zegoEngineProfile, null);
```
</Step>
</Steps>

## Prevent Code Obfuscation

Add keep rules in `proguard-rules.pro` to avoid obfuscating ZEGO related class names:

```txt
-keep class **.zego.**{*;}
```
