# Common Error Codes

- - -

## Introduction

When the SDK encounters errors related to network, resource loading, file operations, etc., it cannot recover automatically and requires App intervention or user notification. If the API call result contains error information, event callbacks contain error information, or error logs appear, it indicates the API call has failed.

Error codes are typically thrown from callbacks such as `onError` and `onIsLoadedError`. Please handle them accordingly based on the following error codes.

## Success

| Error Code Enum Value | Description |
|-|-|
| 0 | Success |

## -200000 ~ -200009 Parameter Validation Errors

Parameter validation errors occur when the parameters passed by the developer when calling the API do not meet the requirements, usually manifested as empty parameters, incorrect parameter formats, etc.

| Error Code Enum Value | Description |
|-|-|
| -200000 | Description: General parameter error.<br />Possible cause: The passed parameter is empty or has an invalid value.<br />Recommended action: Please check if the passed parameter is correct, ensuring the parameter is not empty and the format is correct. |
| -200001 | Description: Digital Human ID is empty or invalid.<br />Possible cause: The passed Digital Human ID is empty, has an incorrect format, or does not exist.<br />Recommended action: Please check if the passed Digital Human ID is correct, ensuring the Digital Human ID is not empty and the format is correct. |
| -200002 | Description: Token is empty or invalid.<br />Possible cause: The passed Token is empty, has an incorrect format, or has expired.<br />Recommended action: Please check if the passed Token is correct, ensuring the Token is not empty and has not expired. If the Token has expired, please obtain a new Token. |
| -200003 | Description: User ID is empty or invalid.<br />Possible cause: The passed User ID is empty or has an incorrect format.<br />Recommended action: Please check if the passed User ID is correct, ensuring the User ID is not empty and the format is correct. |
| -200004 | Description: URL is empty or invalid.<br />Possible cause: The passed URL is empty, has an incorrect format, or is inaccessible.<br />Recommended action: Please check if the passed URL is correct, ensuring the URL is not empty and the format is correct. |

## -200010 ~ -200019 Network-Related Errors

Network-related errors occur when requesting network resources or communicating with the server, usually manifested as network connection failures, request timeouts, etc.

| Error Code Enum Value | Description |
|-|-|
| -200010 | Description: General network error.<br />Possible cause: Abnormal network connection, unstable network, etc.<br />Recommended action: Please check if the network connection is normal. It is recommended to switch to a different network environment and retry. |
| -200011 | Description: Request timeout.<br />Possible cause: High network latency, slow server response, etc.<br />Recommended action: Please check if the network connection is normal. It is recommended to switch to a different network environment and retry. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200012 | Description: Network connection lost.<br />Possible cause: Network interruption, WiFi disconnection, etc.<br />Recommended action: Please check if the network connection is normal, ensure the network connection is stable, and retry. |
| -200013 | Description: Unable to connect to host.<br />Possible cause: Incorrect server address, network unreachable, firewall blocking, etc.<br />Recommended action: Please check if the server address is correct, check if the network connection is normal, and check the firewall settings. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200014 | Description: Network request failed.<br />Possible cause: Network abnormality, server error, etc.<br />Recommended action: Please check if the network connection is normal. It is recommended to switch to a different network environment and retry. If the issue persists, please contact ZEGOCLOUD Technical Support. |

## -200020 ~ -200029 Resource Loading Errors

Resource loading errors occur when loading Digital Human resources, downloading material packages, extracting resources, etc.

| Error Code Enum Value | Description |
|-|-|
| -200020 | Description: General resource loading error.<br />Possible cause: Corrupted resource files, incorrect resource paths, etc.<br />Recommended action: Please check if the resource files are complete and if the resource paths are correct. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200021 | Description: Invalid rendering information data.<br />Possible cause: Incorrect rendering information data format, data corruption, etc.<br />Recommended action: Please check if the rendering information data is correct, ensuring the data format meets the requirements. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200022 | Description: Download failed.<br />Possible cause: Network abnormality, server error, insufficient storage space, etc.<br />Recommended action: Please check if the network connection is normal, check if the device storage space is sufficient. It is recommended to switch to a different network environment and retry. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200023 | Description: File copy failed.<br />Possible cause: Source file does not exist, no write permission for the target path, insufficient storage space, etc.<br />Recommended action: Please check if the source file exists, if the target path has write permission, and if the device storage space is sufficient. |
| -200024 | Description: Resource extraction failed.<br />Possible cause: Corrupted archive, insufficient storage space, no write permission for the extraction path, etc.<br />Recommended action: Please check if the archive is complete, if the device storage space is sufficient, and if the extraction path has write permission. If the issue persists, please re-download the resource. |
| -200025 | Description: Material package download failed.<br />Possible cause: Network abnormality, server error, insufficient storage space, etc.<br />Recommended action: Please check if the network connection is normal, check if the device storage space is sufficient. It is recommended to switch to a different network environment and retry. If the issue persists, please contact ZEGOCLOUD Technical Support. |

## -200030 ~ -200039 File Operation Errors

File operation errors occur during file read/write, cache management, etc.

| Error Code Enum Value | Description |
|-|-|
| -200030 | Description: General file operation error.<br />Possible cause: Incorrect file path, insufficient file permissions, insufficient storage space, etc.<br />Recommended action: Please check if the file path is correct, if the file permissions are sufficient, and if the device storage space is sufficient. |
| -200031 | Description: Failed to get cache directory.<br />Possible cause: The app has no permission to access the cache directory, the cache directory does not exist, etc.<br />Recommended action: Please check if the app has permission to access the cache directory and if the cache directory exists. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200032 | Description: Failed to delete cache directory.<br />Possible cause: Cache directory is in use, insufficient file permissions, etc.<br />Recommended action: Please check if the cache directory is being used by another program and if the file permissions are sufficient. If the issue persists, please contact ZEGOCLOUD Technical Support. |

## -200040 ~ -200049 Internal Operation Errors

Internal operation errors occur during the SDK's internal processing.

| Error Code Enum Value | Description |
|-|-|
| -200040 | Description: General internal error.<br />Possible cause: SDK internal processing abnormality.<br />Recommended action: If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200041 | Description: Operation cancelled.<br />Possible cause: User actively cancelled the operation, operation was interrupted, etc.<br />Recommended action: This error usually indicates the operation was cancelled normally. No special handling is needed. To continue the operation, please initiate a new request. |
| -200042 | Description: Duplicate start error.<br />Possible cause: Starting again while an instance is already running.<br />Recommended action: Please ensure you stop or destroy the previous instance before starting, to avoid duplicate starts. |

## -200050 ~ -200059 Backend Service Errors

Backend service errors occur when communicating with the ZEGO backend service.

| Error Code Enum Value | Description |
|-|-|
| -200050 | Description: General backend error.<br />Possible cause: Backend service abnormality, server error, etc.<br />Recommended action: Please retry later. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200051 | Description: Backend request failed.<br />Possible cause: Network abnormality, server error, incorrect request parameters, etc.<br />Recommended action: Please check if the network connection is normal, check if the request parameters are correct. It is recommended to switch to a different network environment and retry. If the issue persists, please contact ZEGOCLOUD Technical Support. |
| -200052 | Description: Invalid backend response.<br />Possible cause: Incorrect data format returned by the server, data corruption, etc.<br />Recommended action: Please retry later. If the issue persists, please contact ZEGOCLOUD Technical Support. |
