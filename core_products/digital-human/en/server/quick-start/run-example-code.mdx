# Run Example Code

We provide example code for two scenarios: Digital Human real-time broadcasting and Digital Human AI interactive conversation.

Example code repository: [GitHub](https://github.com/ZEGOCLOUD/digital-human-quick-start-example)

Please download the example code first before running it:
```bash
git clone https://github.com/ZEGOCLOUD/digital-human-quick-start-example.git
```

## Digital Human Real-Time Broadcasting

<Video src="https://doc-media.zego.im/core_products/digital-human/zh/server/quick-start/real-time-broadcasting.mp4" />

The [Digital Human Real-Time Broadcasting](https://github.com/ZEGOCLOUD/digital-human-quick-start-example/tree/main/digital-human-real-time-broadcasting-scenario) example code includes server-side and client-side example code. Follow the instructions below to run the example code:

<Tabs>
<Tab title="Server">

<Note title="Note">
Please run the server before running the client.
</Note>

<Steps titleSize="p">
<Step title="Navigate to the server directory of the Digital Human real-time broadcasting example">
```bash
cd digital-human-quick-start-example/digital-human-real-time-broadcasting-scenario/server
```

</Step>
<Step title="Install dependencies">
```bash
npm install
```
</Step>
<Step title="Configure environment variables">
```bash
cp .env.example .env
```

Edit the `.env` file and fill in your **AppID** and **ServerSecret**.

```env
APP_ID=your_AppID
SERVER_SECRET=your_ServerSecret
TOKEN_EXPIRE_SECONDS=3600
```
</Step>
<Step title="Start the server">
```bash
npm run dev
```
</Step>
<Step title="Open the broadcast content composition page">
Open a browser and visit `http://localhost:3000`. You will see the broadcast content composition page. Select a Digital Human avatar and timbre, configure the broadcast content, and click the Start Broadcast Task button. You can then view the Digital Human broadcasting through the client.
</Step>
</Steps>

</Tab>
<Tab title="Android">

<Steps titleSize="p">
<Step title="Open Android Studio and run the project">
Open Android Studio and load the Android project from the example code.

Edit the `app/src/main/java/com/example/digitalhumanquickstartdemo/config/Config.kt` configuration file:

```kotlin
object Config {
    const val APP_ID: Long = 1234567890L  // Replace with your AppID
    const val API_BASE_URL = "http://10.0.2.2:3000"  // Use 10.0.2.2 for emulator, use your computer's LAN IP for real device
}
```

Click the run button to install and launch the app.
</Step>
<Step title="Select a broadcast task and play">
After the app launches, it will automatically fetch the broadcast list and select the first broadcast task for broadcasting. You should now see the Digital Human broadcasting screen.
</Step>
</Steps>

</Tab>
<Tab title="iOS">

<Steps titleSize="p">
<Step title="Install dependencies and open the project">
```bash
cd digital-human-quick-start-example/digital-human-real-time-broadcasting-scenario/ios-oc
pod install
```

Open `ZegoDigitalHumanQuickStart.xcworkspace` (note: the .xcworkspace file).
</Step>
<Step title="Configure AppID and backend URL">
Edit `ZegoDigitalHumanQuickStart/Config.m`:

```objc
@implementation Config

+ (NSUInteger)APP_ID {
    return 1234567890;  // Replace with your AppID
}

+ (NSString *)API_BASE_URL {
    return @"http://localhost:3000";  // Use localhost for simulator, use your computer's LAN IP for real device
}

@end
```
</Step>
<Step title="Run the project">
Select the target device in Xcode and click the run button.
</Step>
<Step title="Select a broadcast task and play">
After the app launches, it will automatically fetch the broadcast list and select the first broadcast task for broadcasting. You should now see the Digital Human broadcasting screen.
</Step>
</Steps>

## Digital Human AI Interactive Conversation

<Video src="https://doc-media.zego.im/core_products/digital-human/zh/server/quick-start/interactive-chat.mp4" />

The [Digital Human AI Interactive Conversation](https://github.com/ZEGOCLOUD/digital-human-quick-start-example/tree/main/digital-human-interactive-chat-scenario) example code includes server-side and client-side example code. Follow the instructions below to run the example code:

<Tabs>
<Tab title="Server">

<Note title="Note">
Please run the server before running the client.
</Note>

<Steps titleSize="p">
<Step title="Navigate to the server directory of the Digital Human AI interactive conversation example">
```bash
cd digital-human-quick-start-example/digital-human-interactive-chat-scenario/server
```

</Step>
<Step title="Install dependencies">
```bash
npm install
```
</Step>
<Step title="Configure environment variables">
```bash
cp .env.example .env
```

Edit the `.env` file and fill in your **AppID**, **ServerSecret**, and **DigitalHumanID**.

```env
APP_ID=your_AppID
SERVER_SECRET=your_ServerSecret
DIGITAL_HUMAN_ID=your_DigitalHumanID
```
<Note title="Note">
DIGITAL_HUMAN_ID can be obtained by calling the Digital Human server API `GetDigitalHumanList`. Or you can directly copy it from the broadcast content composition page.
</Note>

</Step>
<Step title="Start the server">
```bash
npm run dev
```
</Step>
</Steps>

</Tab>
<Tab title="Android">

<Steps titleSize="p">
<Step title="Open Android Studio and run the project">


Click the run button to install and launch the app.
</Step>
<Step title="Start interactive conversation">
After the app launches, click the "Start Call" button to create a Digital Human task. After the task is created successfully, click the "Speak" button to simulate a conversation with the Digital Human, and finally click the "End Call" button to stop the task.
</Step>
</Steps>
