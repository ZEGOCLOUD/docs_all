# 图片数字人素材规范

- - -

本文介绍制作图片数字人所需的素材规范。

## 图片素材制作规范

### 图片生成方法

- 真人形象可直接通过相机、手机拍摄
- 可通过 AI 软件生成图片素材。

### 支持的形象风格

- 真人(真人拍摄照片)
- 拟真人(生成拟真人图片)
- 3D 动漫
- 3D 卡通
- 二次元

### 图片素材要求

制作图片素材时，请参考以下要求：

| 类别               | 说明                                                                                                                                                                                                                                                                           |
|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 图片尺寸           | 推荐 16:9 竖屏（或 9:16 比例），脸部清晰，分辨率为1080P及以上（推荐2K），需要彩色照片，暂不支持黑白                                                                                                                                |
| 人物角度&图片占比  | 1. 推荐正脸正身，侧脸尽量与屏幕角度在 30° 以下<br/>2. 支持半身照、全身照（最好是上半身 1/2 ~ 2/3，人物在图片正中间且无其他遮挡）                                                                       |
| 人物风格           | 支持真人形象五官清晰、轮廓清晰或立体的二次元形象                                                                                                                                                                                    |
| 人物动作表情       | 自然站立/坐立，不要有抬手、曲臂、弯腰等大幅度动作；表情自然，不做夸张表情（如大笑、皱眉等）；嘴巴闭上不要露牙齿                                                                                   |
| 人物细节要求       | 五官清晰不能被遮挡；头部不要有帽子、夸张发饰等加高头部的物件或配饰，不要有过多碎发或发丝；脸部和脖子附近不能有遮挡物或其他装饰物（如耳饰、碎发、手等）                                                  |
| 背景               | 以纯色背景为佳；背景和人物需要有明显区分度；不支持透明度背景                                                                                                                                     |
| 合规要求           | 不能有大量露肤                                                                                                                                                                                                                         |

### 素材最佳实践
为了让数字人的形象符合预期，图片中人物建议满足以下要求：

- **有动作要求**： 推荐露出臀部及以上的上半身，双手自然下垂。
