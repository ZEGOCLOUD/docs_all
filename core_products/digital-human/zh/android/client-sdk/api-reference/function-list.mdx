# 功能总览

- - -

ZEGO 数字人移动端 SDK 提供高质量的数字人渲染和交互能力。通过本 SDK，开发者可以轻松地在移动应用中集成实时互动的数字人形象。

API 返回的错误码请参考：[DigitalMobileErrorCode](./enum#digitalmobileerrorcode)。

以下为核心 API 的功能列表。

## 引擎管理与全局配置

| 方法名 | 描述 | 
| --- | --- | 
| [create](./class#create-zegodigitalhuman-class) | 创建数字人实例（入口方法）。 |
| [setLogConfig](./class#setlogconfig-zegodigitalhuman-class) | 全局设置日志保存路径及轮转配置。 |

## 数字人生命周期管理

| 方法名 | 描述 | 
| --- | --- | 
| [start](./interface#start-izegodigitalmobile-interface) | 根据配置启动数字人任务。 |
| [stop](./interface#stop-izegodigitalmobile-interface) | 停止数字人任务并释放资源。 |
| [attach](./interface#attach-izegodigitalmobile-interface) | 将数字人渲染内容绑定到视图控件。 |
| [getDigitalView](./interface#getdigitalview-izegodigitalmobile-interface) | 获取当前正在显示的视图。 |

## 实时数据交互 (RTC 对接)

| 方法名 | 描述 | 
| --- | --- | 
| [onRemoteVideoFrameRawData](./interface#onremotevideoframerawdata-izegodigitalmobile-interface) | 喂入远端视频流原始帧数据。 |
| [onPlayerSyncRecvSEI](./interface#onplayersyncrecvsei-izegodigitalmobile-interface) | 喂入播放器同步接收到的 SEI 元数据。 |

## 状态监听回调

| 方法名 | 描述 | 
| --- | --- | 
| [onDigitalMobileStartSuccess](./interface#ondigitalmobilestartsuccess-izegodigitalmobilezegodigitalmobilelistener-interface) | 数字人启动成功通知。 |
| [onError](./interface#onerror-izegodigitalmobilezegodigitalmobilelistener-interface) | 运行过程中的错误通知。 |
| [onSurfaceFirstFrameDraw](./interface#onsurfacefirstframedraw-izegodigitalmobilezegodigitalmobilelistener-interface) | 首帧渲染上屏通知。 |

## 形象资源预加载管理

| 方法名 | 描述 | 
| --- | --- | 
| [preload](./class#preload-zegodigitalhumanresource-class) | 异步下载并缓存指定数字人的形象资源。 |
| [isLoaded](./class#isloaded-zegodigitalhumanresource-class) | 检查指定数字人的资源是否已准备就绪。 |
| [unload](./class#unload-zegodigitalhumanresource-class) | 删除本地缓存的数字人资源。 |

## 资源管理回调

| 方法名 | 描述 | 
| --- | --- | 
| [onSuccess](./class#onsuccess-zegodigitalhumanresourcepreloadcallback-class) | 预加载成功。 |
| [onProgress](./class#onprogress-zegodigitalhumanresourcepreloadcallback-class) | 预加载进度更新。 |
| [onError](./class#onerror-zegodigitalhumanresourcepreloadcallback-class) | 预加载失败。 |
| [onResult](./class#onresult-zegodigitalhumanresourceisloadedcallback-class) | 资源加载状态查询结果。 |
