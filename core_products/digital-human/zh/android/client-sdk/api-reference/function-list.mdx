# 功能总览

- - -

ZEGO 数字人 SDK 提供完整的数字人渲染功能，支持web模式和mobile模式），为 Android 应用提供高质量的数字人体验。

API 返回的错误码请参考：[DigitalMobileErrorCode](./enum#digitalmobileerrorcode)。

以下为基本的 API 简要介绍。

## 初始化相关的前置功能

 | 方法名 | 描述 | 
 | --- | --- | 
| [create](./class.mdx#create-zegodigitalhuman-class) | 创建数字人实例。 |
| [setLogConfig](./class.mdx#setlogconfig-zegodigitalhuman-class) | 设置数字人日志配置。 |

## 数字人任务管理

 | 方法名 | 描述 | 
 | --- | --- | 
| [start](./interface.mdx#start-izegodigitalmobile-interface) | 创建数字人任务。 |
| [stop](./interface.mdx#stop-izegodigitalmobile-interface) | 停止数字人。 |
| [attach](./interface.mdx#attach-izegodigitalmobile-interface) | 绑定数字人显示视图。 |
| [getDigitalView](./interface.mdx#getdigitalview-izegodigitalmobile-interface) | 获取数字人显示视图。 |

## 视频帧数据处理

 | 方法名 | 描述 | 
 | --- | --- | 
| [onRemoteVideoFrameRawData](./interface.mdx#onremotevideoframerawdata-izegodigitalmobile-interface) | 远端视频帧原始数据回调。 |
| [onPlayerSyncRecvSEI](./interface.mdx#onplayersyncrecvsei-izegodigitalmobile-interface) | 播放器 SEI 信息同步接收回调。 |

 | 方法名 | 描述 |
