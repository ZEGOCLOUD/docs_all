# 功能总览

- - -

ZEGO 数字人移动端 SDK 为 iOS 平台提供高效、稳定的数字人渲染和交互能力。开发者可以通过本 SDK 在 iOS 应用中快速集成实时互动的数字人形象。

API 返回的错误码请参考：[ZegoDigitalMobileError](./enum#zegodigitalmobileerror)。

以下为核心 API 的功能列表。

## 引擎管理与全局配置

| 方法名 | 描述 | 
| --- | --- | 
| [create](./class#create-zegodigitalhuman-class) | 创建数字人实例（入口方法）。 |
| [setLogConfig:](./class#setlogconfig-zegodigitalhuman-class) | 全局设置日志保存路径及轮转配置。 |
| [getDigitalhumanUrlbyID:userId:token:deviceId:digitalHumanId:completion:](./class#getdigitalhumanurlbyiduseridtokendeviceiddigitalhumanidcompletion-zegodigitalhuman-class) | 通过数字人 ID 获取渲染信息下载 URL。 |

## 数字人生命周期管理

| 方法名 | 描述 | 
| --- | --- | 
| [start:delegate:](./protocol#startdelegate-izegodigitalmobile-protocol) | 根据指定配置启动数字人任务。 |
| [stop](./protocol#stop-izegodigitalmobile-protocol) | 停止数字人任务并释放资源。 |
| [attach:](./protocol#attach-izegodigitalmobile-protocol) | 将数字人渲染内容绑定到 UIView 视图。 |
| [getPreviewView](./protocol#getpreviewview-izegodigitalmobile-protocol) | 获取当前正在引用的渲染视图。 |

## 实时数据交互 (RTC 对接)

| 方法名 | 描述 | 
| --- | --- | 
| [onRemoteVideoFrameRawData:dataLength:param:streamID:](./protocol#onremotevideoframerawdatadatalengthparamstreamid-izegodigitalmobile-protocol) | 喂入远端视频流原始帧数据。 |
| [onPlayerSyncRecvSEI:data:](./protocol#onplayersyncrevseidata-izegodigitalmobile-protocol) | 喂入播放器同步接收到的 SEI 元数据。 |

## 状态监听回调

| 方法名 | 描述 | 
| --- | --- | 
| [onDigitalMobileStartSuccess](./protocol#ondigitalmobilestartsuccess-izegodigitalmobilezegodigitalmobiledelegate-protocol) | 数字人启动成功通知。 |
| [onError:errorMsg:](./protocol#onerrorerrormsg-izegodigitalmobilezegodigitalmobiledelegate-protocol) | 运行过程中的错误通知。 |
| [onSurfaceFirstFrameDraw](./protocol#onsurfacefirstframedraw-izegodigitalmobilezegodigitalmobiledelegate-protocol) | 首帧渲染上屏通知。 |

## 形象资源预加载管理

| 方法名 | 描述 | 
| --- | --- | 
| [sharedInstance](./class#sharedinstance-zegodigitalhumanresource-class) | 获取资源管理类单例。 |
| [preloadWithAuth:digitalHumanId:delegate:](./class#preloadwithauthdigitalhumaniddelegate-zegodigitalhumanresource-class) | 异步下载并缓存指定数字人的形象资源。 |
| [isLoadedWithAuth:digitalHumanId:delegate:](./class#isloadedwithauthdigitalhumaniddelegate-zegodigitalhumanresource-class) | 检查指定数字人的资源是否已准备就绪。 |
| [unLoad:](./class#unload-zegodigitalhumanresource-class) | 删除本地缓存的数字人资源。 |

## 资源管理回调协议

| 协议名 | 描述 | 
| --- | --- | 
| [ZegoDigitalHumanResourceDelegate](./protocol#zegodigitalhumanresourcedelegate) | 资源预加载的进度和结果回调。 |
| [IsLoadedDelegate](./protocol#isloadeddelegate) | 资源加载状态查询的结果回调。 |
