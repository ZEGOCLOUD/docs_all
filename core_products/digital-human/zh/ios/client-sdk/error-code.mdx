# 常见错误码

- - -

## 简介

当 SDK 运行出现网络、资源加载、文件操作等相关错误时，SDK 无法自动恢复，需要 App 干预或进行用户提示。调用接口后返回结果存在错误信息、事件回调中有错误信息或日志中出现错误日志，则表示该接口调用异常。

错误码通常会从 `onError`、`onIsLoadedError` 等回调中抛出，请根据以下错误码并进行相应处理。

## 成功

| 错误码枚举值 | 说明 |
|-|-|
| 0 | 执行成功 |

## -200000 ~ -200009 参数验证错误

参数验证错误指的是开发者在调用接口时传入的参数不符合要求，一般表现为参数为空、参数格式错误等。

| 错误码枚举值 | 说明 |
|-|-|
| -200000 | 描述：通用参数错误。<br />可能原因：传入的参数为空、或参数为非法值。<br />处理建议：请检查传入的参数是否正常，确保参数不为空且格式正确。 |
| -200001 | 描述：数字人ID为空或无效。<br />可能原因：传入的数字人ID为空、格式错误或不存在。<br />处理建议：请检查传入的数字人ID是否正确，确保数字人ID不为空且格式正确。 |
| -200002 | 描述：Token为空或无效。<br />可能原因：传入的Token为空、格式错误或已过期。<br />处理建议：请检查传入的Token是否正确，确保Token不为空且未过期。如Token已过期，请重新获取Token。 |
| -200003 | 描述：用户ID为空或无效。<br />可能原因：传入的用户ID为空或格式错误。<br />处理建议：请检查传入的用户ID是否正确，确保用户ID不为空且格式正确。 |
| -200004 | 描述：URL为空或无效。<br />可能原因：传入的URL为空、格式错误或无法访问。<br />处理建议：请检查传入的URL是否正确，确保URL不为空且格式正确。 |

## -200010 ~ -200019 网络相关错误

网络相关错误指的是在请求网络资源、与服务器通信时出现的错误，一般表现为网络连接失败、请求超时等。

| 错误码枚举值 | 说明 |
|-|-|
| -200010 | 描述：通用网络错误。<br />可能原因：网络连接异常、网络不稳定等。<br />处理建议：请检查网络连接是否正常，建议切换网络环境后重试。 |
| -200011 | 描述：请求超时。<br />可能原因：网络延迟较大、服务器响应慢等。<br />处理建议：请检查网络连接是否正常，建议切换网络环境后重试。如问题持续，请联系 ZEGO 技术支持。 |
| -200012 | 描述：网络连接丢失。<br />可能原因：网络中断、WiFi断开等。<br />处理建议：请检查网络连接是否正常，确保网络连接稳定后重试。 |
| -200013 | 描述：无法连接到主机。<br />可能原因：服务器地址错误、网络不通、防火墙拦截等。<br />处理建议：请检查服务器地址是否正确，检查网络连接是否正常，检查防火墙设置。如问题持续，请联系 ZEGO 技术支持。 |
| -200014 | 描述：网络请求失败。<br />可能原因：网络异常、服务器错误等。<br />处理建议：请检查网络连接是否正常，建议切换网络环境后重试。如问题持续，请联系 ZEGO 技术支持。 |

## -200020 ~ -200029 资源加载错误

资源加载错误指的是在加载数字人资源、下载素材包、解压资源等操作时出现的错误。

| 错误码枚举值 | 说明 |
|-|-|
| -200020 | 描述：通用资源加载错误。<br />可能原因：资源文件损坏、资源路径错误等。<br />处理建议：请检查资源文件是否完整，检查资源路径是否正确。如问题持续，请联系 ZEGO 技术支持。 |
| -200021 | 描述：渲染信息数据无效。<br />可能原因：渲染信息数据格式错误、数据损坏等。<br />处理建议：请检查渲染信息数据是否正确，确保数据格式符合要求。如问题持续，请联系 ZEGO 技术支持。 |
| -200022 | 描述：下载失败。<br />可能原因：网络异常、服务器错误、存储空间不足等。<br />处理建议：请检查网络连接是否正常，检查设备存储空间是否充足，建议切换网络环境后重试。如问题持续，请联系 ZEGO 技术支持。 |
| -200023 | 描述：文件复制失败。<br />可能原因：源文件不存在、目标路径无权限、存储空间不足等。<br />处理建议：请检查源文件是否存在，检查目标路径是否有写入权限，检查设备存储空间是否充足。 |
| -200024 | 描述：资源解压失败。<br />可能原因：压缩包损坏、存储空间不足、解压路径无权限等。<br />处理建议：请检查压缩包是否完整，检查设备存储空间是否充足，检查解压路径是否有写入权限。如问题持续，请重新下载资源。 |
| -200025 | 描述：素材包下载失败。<br />可能原因：网络异常、服务器错误、存储空间不足等。<br />处理建议：请检查网络连接是否正常，检查设备存储空间是否充足，建议切换网络环境后重试。如问题持续，请联系 ZEGO 技术支持。 |

## -200030 ~ -200039 文件操作错误

文件操作错误指的是在文件读写、缓存管理等操作时出现的错误。

| 错误码枚举值 | 说明 |
|-|-|
| -200030 | 描述：通用文件操作错误。<br />可能原因：文件路径错误、文件权限不足、存储空间不足等。<br />处理建议：请检查文件路径是否正确，检查文件权限是否充足，检查设备存储空间是否充足。 |
| -200031 | 描述：获取缓存目录失败。<br />可能原因：应用无权限访问缓存目录、缓存目录不存在等。<br />处理建议：请检查应用是否有访问缓存目录的权限，检查缓存目录是否存在。如问题持续，请联系 ZEGO 技术支持。 |
| -200032 | 描述：删除缓存目录失败。<br />可能原因：缓存目录被占用、文件权限不足等。<br />处理建议：请检查缓存目录是否被其他程序占用，检查文件权限是否充足。如问题持续，请联系 ZEGO 技术支持。 |

## -200040 ~ -200049 内部操作错误

内部操作错误指的是 SDK 内部处理过程中出现的错误。

| 错误码枚举值 | 说明 |
|-|-|
| -200040 | 描述：通用内部错误。<br />可能原因：SDK 内部处理异常。<br />处理建议：如问题持续，请联系 ZEGO 技术支持。 |
| -200041 | 描述：操作已取消。<br />可能原因：用户主动取消操作、操作被中断等。<br />处理建议：此错误通常表示操作被正常取消，无需特殊处理。如需继续操作，请重新发起请求。 |
| -200042 | 描述：重复启动错误。<br />可能原因：在已有实例运行的情况下重复启动。<br />处理建议：请确保在启动前先停止或销毁之前的实例，避免重复启动。 |

## -200050 ~ -200059 后台服务错误

后台服务错误指的是与 ZEGO 后台服务通信时出现的错误。

| 错误码枚举值 | 说明 |
|-|-|
| -200050 | 描述：通用后台错误。<br />可能原因：后台服务异常、服务器错误等。<br />处理建议：请稍后重试。如问题持续，请联系 ZEGO 技术支持。 |
| -200051 | 描述：后台请求失败。<br />可能原因：网络异常、服务器错误、请求参数错误等。<br />处理建议：请检查网络连接是否正常，检查请求参数是否正确，建议切换网络环境后重试。如问题持续，请联系 ZEGO 技术支持。 |
| -200052 | 描述：后台响应无效。<br />可能原因：服务器返回数据格式错误、数据损坏等。<br />处理建议：请稍后重试。如问题持续，请联系 ZEGO 技术支持。 |
