# 视频数字人素材规范

本文介绍视频数字人定制所需的视频素材规范

## 视频素材要求

如若您尝试自行拍摄视频素材，可参考[视频数字人采集指南](/aigc-digital-human-server/digital-human-customization/process/video-digital-human-shooting-guide)。

已采集完成，准备提供制作形象的素材，请确保视频素材是否满足以下要求：
|类别-150px | 说明|
|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 视频规格       | <ul> <li>视频时长：视频总时长不小于3分钟，5-10分钟最佳</li> <li>视频体积：视频大小需小于20G</li> <li>视频规格：分辨率为 1080P 及以上（推荐4K），帧率为 25fps 及以上</li> <li>视频比例：推荐 16:9 竖屏，或 9:16 横屏</li> </ul>      |
| 画面布局       | <ul> <li>人物占比控制在整个视频画面的 1/2 - 2/3，特别是面部、头部区域不能低于整个画面5%，要能清晰可见（可参考下方示例）</li> <li>支持人物半身、全身视频，人物需保持在视频正中间且无其他遮挡。</li> </ul>                       |
| 人物表现要求   | <ul> <li>正脸正身，头部转动尽量与屏幕角度在 30° 以下，最大支持60°转角</li> <li>自然站立/坐立，不要有弯腰等大幅度动作</li> <li>表情自然，不做夸张表情（如大笑、皱眉等）</li> <li>人物眼睛保持正视，尽量避免飘忽不定</li> <li>手势肢体动作不要挡住头部和脖子区域，不要超出画面之外</li> </ul> |
| 画面背景       | <ul> <li>光滑平整的绿幕背景（后期可抠图换背景）或静态场景背景（不支持换背景）；</li> <li>画面打光均匀自然，不能过暗或过度曝光，打光可参考 视频数字人采集指南</li> </ul>                                  |
| 人物着装要求   | <ul> <li>人物着装需与背景有明显区分度</li> <li>绿幕背景下人物不穿绿色、青色系；</li> <li>任何背景下都严禁穿着反光、透光材质以及格子条纹的服装和装饰，无法定制形象；</li> <li>如果客户衣服必须为黄色、蓝色等与绿幕相似的色系，则绿幕可替换成蓝幕等其他颜色，具体可咨询 ZEGO售前</li> </ul> |
