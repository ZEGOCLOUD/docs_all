# 功能总览



- - -



ZEGO SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为多端提供清晰稳定的实时语音视频云服务。

以下为基本 API 的简要介绍。





## 初始化相关的前置功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [destroyEngine](./class.mdx#destroyengine-zegoexpressengine) | 销毁引擎实例。
[callExperimentalAPI](./class.mdx#callexperimentalapi-zegoexpressengine) | ZEGO 通过此 API 提供 RTC 业务中的部分技术预览或特别定制功能，需要获取功能的使用或详情其详情可咨询 ZEGO 技术支持
[checkSystemRequirements](./class.mdx#checksystemrequirements-zegoexpressengine) | 支持能力检测接口
[off](./class.mdx#off-zegoexpressengine) | 删除注册过的回调事件
[on](./class.mdx#on-zegoexpressengine) | 注册回调事件
[initContext](./class.mdx#initcontext-zegoexpressengine) | 初始化小程序上下文
[enableMultiRoom](./class.mdx#enablemultiroom-zegoexpressengine) | 是否开启多房间模式
[setLogConfig](./class.mdx#setlogconfig-zegoexpressengine) | 日志高级配置
[setDebugVerbose](./class.mdx#setdebugverbose-zegoexpressengine) | 是否开启Debug模式，打开时错误信息 alert 提示，默认测试环境开启
[getVersion](./class.mdx#getversion-zegoexpressengine) | 获取当前SDK版本 |
