- - -

The ZEGO Real-time Audio (Express Audio) SDK is provided by ZEGO. You can obtain the Express-Audio SDK for macOS clients on this page. For release notes, please refer to [Release Notes](/real-time-voice-macos/client-sdk/release-notes). For compliance matters, please refer to [ZEGO Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

- Express macOS SDK supports OS X 10.10+.
- Express Mac Catalyst SDK supports macOS 11.0+.
