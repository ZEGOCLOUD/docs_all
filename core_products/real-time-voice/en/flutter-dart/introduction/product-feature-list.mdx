# Product Features

- - -

## Communication Capabilities

### Basic Features

| Basic Feature - 20% | Feature Description - 50% | Business Scenario - 30% |
|-----|-----|-----|
| [Audio Call](/real-time-voice-flutter/quick-start/implementing-voice-call) | Users join the same room and make audio calls. | <ul><li>1v1 audio call</li><li>Multi-person audio conference</li></ul> |
| [Audio Live Streaming](/real-time-voice-flutter/quick-start/implementing-voice-call) | In the same room, there are hosts and audience. Hosts can start an audio live stream and the audience in the room can listen. | <ul><li>Emotional FM</li><li>Voice chat room</li></ul> |
| [User Permission Control](/real-time-voice-flutter/communication/using-token-authentication) | Use tokens to control user permissions, such as allowing specific users to enter/leave the room, speak/mute, or otherwise manage participants. | Video conference |
| [Pre-Call Detection](/real-time-voice-flutter/communication/pre-call-detection) | Test devices such as cameras, microphones, displays before audio/video calling or live streaming to ensure smooth operation. | Regular call function check |
| [Call Quality Monitoring](/real-time-voice-flutter/communication/monitor-stream-quality) | Monitor multiple indicators of audio/video quality, such as resolution, frame rate, bit rate, sample rate, to ensure stable quality. | Scenarios with high audio/video quality demands such as bank account opening, remote authentication |
| [Network Speed Test](/real-time-voice-flutter/communication/testing-network) | Measure uplink and downlink speeds before streaming to determine the optimal bit rate for audio/video streaming in the current network. | Calling, education, and live streaming scenarios |

### Advanced Features

| Advanced Feature - 20% | Feature Description - 50% | Business Scenario - 30% |
|-----|-----|-----|
| [Co-hosted Live Streaming](/real-time-voice-flutter/quick-start/implementing-voice-call) | Multiple hosts can appear in the same room and co-stream together on the same screen. | <ul><li>Co-hosting by multiple hosts from different locations</li><li>Multi-person KTV chorus</li><li>Show live streaming</li></ul> |
| Multiple Source Capture | Provides flexible and easy-to-use audio/video source and channel management, reducing developer workload and maintenance costs. | Video conferencing, online education |
| Simultaneous Multi-streaming | A user can broadcast multiple audio/video streams simultaneously, such as sending both a camera stream and a screen share. | Presenter can display a PPT and be seen at the same time in a video conference |
| [Supplemental Enhancement Information (SEI)](/real-time-voice-flutter/communication/sei) | Bundle text and audio/video content for synchronized transmission through streaming channels, enabling precise synchronization of text and audio/video. | <ul><li>Precise video layout</li><li>Remote lyrics sync</li><li>Live quiz</li></ul> |
| [Flow Control](https://doc-zh.zego.im/faq/flowcontrol?product=ExpressVideo&platform=all) | ZEGO's industry-leading technology. The SDK dynamically adjusts video bitrate, frame rate, resolution, and audio bitrate for smooth streaming according to the network status of both ends, ensuring adaptive, high-quality real-time audio/video. | All scenarios requiring high-quality real-time AV services |
| [Cloud Proxy](/real-time-voice-flutter/communication/cloud-proxy) | By configuring the SDK’s cloud proxy interface, all related traffic is relayed through a cloud proxy server to enable RTC and L3 (ultra-low latency live streaming) communication. | Restricted network environments, e.g. hospitals, government, or companies with strict intranets |
| [Geofencing](/real-time-voice-flutter/communication/geofencing) | Restricting audio/video and signaling data to specific geographic regions to comply with privacy regulations, i.e. limiting access to certain regions. | Calling scenarios |
| [Audio/Video Stream Encryption](/real-time-voice-flutter/communication/secure-encryption-audio) | Streaming data is encrypted during publishing and requires a matching decryption key when playing back. | Scenarios where encrypted streams are needed to protect communication security |
| [Game Audio](/real-time-voice-flutter/communication/range-audio) | Simulates the real world, where sound perception differs according to distance and direction. Also supports grouping, so only people in the same group hear each other (for example, separate group chats in one room). | <ul><li>Metaverse</li><li>Group interaction or battle in one room</li></ul> |

## Audio Capabilities

### Basic Features

| Basic Feature - 20% | Feature Description - 50% | Business Scenario - 30% |
|-----|-----|-----|
| [Audio Spectrum and Volume Changes](/real-time-voice-flutter/audio/sound-level-spectrum) | Audio spectrum: the energy values at each frequency point of digital audio signals.<br /> Volume change: the level of a stream’s volume. | <ul><li>Detecting which user is speaking, whether microphones/speakers are available</li><li>Spectrum visualizations</li></ul> |
| [Headphone Monitoring and Channel Setting](/real-time-voice-flutter/audio/headphone-monitor) | <ul><li>Headphone monitoring: when plugging in headphones (wired or Bluetooth), you can hear the device’s mic via the headphones in real time.</li><li>Stereo: two sound channels so the source’s position can be identified by phase difference between left and right ears.</li></ul> |<ul><li>Show live streaming</li><li>Emotional FM</li><li>Music teaching and other pro scenarios</li></ul>|

| [Voice Changing/Reverb/Stereo](/real-time-voice-flutter/audio/audio-effects) | <p>Add fun and interactivity by using voice changers for goofy effects, reverb for atmosphere, and stereo to make sound more immersive.</p><p>ZEGO Express SDK provides various preset voice changing, reverb, echo, and stereo effects, letting developers flexibly set their own sound effects.</p> | <ul><li>Live streaming</li><li>Voice chat room</li><li>KTV room</li><li>Anonymous social</li><li>Game entertainment</li><li>Role-play</li></ul>  |
