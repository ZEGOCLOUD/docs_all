# Product Features

- - -

## Communication Capabilities

### Basic Features

| Basic Feature-20% | Description-50% | Application Scenario-30% |
|-----|-----|-----|
| [Audio Call](/real-time-voice-electron/quick-start/implementing-voice-call) | Users join the same room and conduct audio calls. | <ul><li>1v1 voice call</li> <li>Multi-party audio conference</li></ul> |
| [Audio Live Streaming](/real-time-voice-electron/quick-start/implementing-voice-call) | A room includes both hosts and audiences. Hosts can do audio live streaming and audiences can listen. | <ul><li>FM-style emotional live</li> <li>Voice chat room</li></ul> |
| [User Permission Control](/real-time-voice-electron/communication/using-token-authentication) | Use Token for user permission control, such as specifying users who can join/leave the room, speak/mute, etc. | Video conference |
| Pre-call Device Test | Before audio/video calls or live streaming, test devices such as camera, microphone, and display to ensure normal operation. | Normal call function test |
| Call Quality Monitoring | Monitor quality of audio/video such as resolution, frame rate, bitrate, and sampling rate, to ensure stable quality. | Scenarios with strict quality requirements like bank account opening, remote authentication, etc. |
| Network Speed Test | Before push/pull streaming, test uplink and downlink network speed, determine the suitable bitrate for audio/video streams under current environment. | Call, education, and live streaming scenarios |

### Advanced Features

| Advanced Feature-20% | Description-50% | Application Scenario-30% |
|-----|-----|-----|
| [Co-hosted Live Streaming](/real-time-voice-electron/quick-start/implementing-voice-call) | Multiple hosts can coexist in the same room for live streaming on a single screen. | <ul><li>Multiple hosts from different regions</li> <li>Multi-user KTV chorus</li> <li>Showroom live streaming</li></ul> |
| [Multi-source Capture](/real-time-voice-electron/communication/multi-source-capture) | Provides flexible and easy management of audio/video capture sources and channels, reducing developer workload and maintenance costs. | Video conference, online education |
| Push Multiple Streams Simultaneously | A user can push multiple audio/video streams, such as sending camera video while sharing screen. | Show presenter’s image while sharing PPT in a conference |
| Supplemental Enhancement Information (SEI) | Packages text information with audio/video content for transmission to achieve precise synchronization of data and media. | <ul><li>Accurate video layout</li> <li>Remote lyric synchronization</li> <li>Live quiz</li></ul> |
| [Traffic Control](/real-time-voice-electron/communication/traffic-control) | ZEGO’s industry-leading technology. The SDK dynamically adjusts video bitrate, frame rate, resolution, and audio bitrate based on current network status (both local and remote) for optimal adaptivity and smooth publishing. | All scenarios requiring high-quality real-time audio/video |
| [Geofencing](/real-time-voice-electron/communication/geofencing) | Restrict audio/video and signaling data transmission to specific regions to comply with data privacy laws, allowing only designated region access. | Call scenarios |
| Audio/Video Stream Encryption | Encrypt streams while pushing; pulling requires a matching decryption key for security. | Scenarios requiring encrypted communication |
| Game Voice Chat | Simulate real-world sound orientation and distance for different auditory experiences. Supports grouping, where users in different groups can't hear each other. | <ul><li>Metaverse</li> <li>Group chat or battle in one room</li></ul> |

## Audio Capabilities

### Basic Features

| Basic Feature-20% | Description-50% | Application Scenario-30% |
|-----|-----|-----|
| Audio Spectrum and Volume Change | Audio spectrum: the energy value of digital audio signals at different frequencies.<br /> Volume change: the volume level of a stream. | <ul><li>Determine who is speaking, microphone/speaker status</li><li>Audio spectrum animation display</li></ul> |
| Earback and Channel Settings | <ul><li>Monitor microphone input on the local device when headphones (wired or Bluetooth) are plugged in.</li> <li>Stereo: two audio channels to determine the direction of sound source based on ear phase difference.</li></ul> |<ul><li>Showroom live streaming</li> <li>FM</li> <li>Professional training like music education</li></ul>|
| Audio 3A Processing | 3A audio processing in real-time calls or live streaming to improve quality and experience.<ul><li>AEC (Echo Cancellation): filter captured audio to reduce echo.</li><li>AGC (Auto Gain Control): automatically adjusts mic volume for stable loudness regardless of distance.</li><li>ANS (Noise Suppression): removes background noise for clearer voice.</li></ul> | All scenarios requiring high-quality real-time A/V |
| Voice Effects/Reverb/Stereo | <p>For fun and engagement, users can use voice changing, reverb, or stereo effects. </p><p>ZEGO Express SDK offers flexible preset voice, reverb, echo, and stereo effects for customization.</p> | <ul><li>Live streaming</li><li>Voice chat room</li><li>Karaoke room</li><li>Anonymous social</li><li>Game entertainment</li><li>Role play</li></ul> |
