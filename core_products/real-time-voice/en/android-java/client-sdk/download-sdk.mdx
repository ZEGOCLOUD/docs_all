# Download SDK

ZEGO Express Audio SDK is provided by ZEGOCLOUD. You can obtain the Express-Audio SDK for Android on this page. For release notes, please refer to [Release Notes](/real-time-voice-android/client-sdk/release-notes). For compliance matters, please refer to [ZEGO Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

<Warning title="Note">


- Starting from version 3.17.0, Express SDK upgraded NDK dependency to NDK r25c. After upgrading, developers need to perform compatibility testing to prevent incompatibility issues.
- If you encounter compatibility issues during testing or need to use an older version of NDK, please contact ZEGOCLOUD Technical Support.

</Warning>
