# Download SDK

- - -

The ZEGO Real-time Audio SDK is provided by ZEGO. You can obtain the Express SDK for Linux clients on this page. For release notes, please refer to [Release Notes](/real-time-voice-linux/client-sdk/release-notes). For compliance matters, please refer to [ZEGO Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

<Note title="Note">

- The SDK package downloaded on this page contains all API interfaces related to `audio` and `video`; if you need an `audio-only` SDK package, please contact ZEGO Technical Support.
- SDK integration guide: Please refer to [Quick Start](/real-time-voice-linux/quick-start/integrating-sdk).
- SDK version changes: Please refer to [Release Notes](/real-time-voice-linux/client-sdk/release-notes).
- This SDK package includes the following architectures:
    - amd64 (x86_64-linux-gnu)
    - arm64 (aarch64-linux-gnu)
    - armhf (arm-linux-gnueabihf)
    - armel (arm-linux-gnueabi)
- If you need SDK packages for other Linux platforms or architectures (such as MIPS / RISC-V / LoongArch or specific embedded systems), please [click this link to fill out the questionnaire](https://zegocloud.feishu.cn/share/base/form/shrcngu3M4UhZiviuornyuYX6mc).

</Note>
