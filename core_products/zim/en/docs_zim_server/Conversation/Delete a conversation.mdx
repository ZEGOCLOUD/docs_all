# Delete a conversation

---

## Overview

You can use this API to delete a conversation on the server side.


## Request method and endpoint

- Request method: POST
- Request URL: `https://zim-api.zego.im/?Action=DeleteConv`
- Protocol: HTTPS
- QPS limit: 20 requests/second

## Request parameters

The following list only shows the request parameters for this interface and some common parameters. For a complete list of common parameters, please refer to [Accessing Server APIs - Common parameters](../Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter | Type | Required | Description |
|------|------|------|------|
| FromUserId | String | Yes | <ul><li>For one-to-one conversation: Enter the UserId of one conversation user.</li><li>For group conversation: Enter the UserId of one group member.</li></ul> |
| ConvId | String | Yes | <ul><li>For one-to-one conversation: Enter the UserId of the other conversation user.</li><li>For group conversation: Enter the GroupId of the group conversation.</li></ul> |
| ConvType | ConvType | Yes | Conversation type:<ul><li>0: One-to-one conversation.</li><li>2: Group conversation.</li></ul> |

## Sample request

- Request URL:
    ```json
    https://zim-api.zego.im/?Action=DeleteConv
    &<Common request parameters>
    ```

- Request body:
    ```json
    {
        "ConvId": "userB",
        "ConvType": 0,
        "FromUserId": "userA"   
    }
    ```

## Response parameters

| Parameter | Type | Description |
|------|------|------|
| Code | Number | Return code. |
| Message | String | Description of the request result. |
| RequestId | String | Request ID. |
