# Delete all messages from a group member

---

## Description

Call this API to delete all messages from a group member.

## Request method and endpoint
- Request method: POST
- Request endpoint: `https://zim-api.zego.im/?Action=ClearMemberGroupMessage`
- Protocol: HTTPS
- QPS limit: 20 requests/second

## Request parameters

The following request parameter list only shows the interface request parameters and some public parameters. For the complete list of public parameters, please refer to [Accessing Server APIs - Public parameters](./../Accessing%20Server%20APIs.mdx#public-parameters).

| Parameter  | Type   | Required | Description                                   |
| ---------- | ------ | -------- | --------------------------------------------- |
| FromUserId | String | Yes      | Delete all group messages sent by this user to `GroupId`. |
| GroupId    | String | Yes      | Group ID.                                     |

## Request example

- Request URL:

  ```json
  https://zim-api.zego.im/?Action=ClearMemberGroupMessage
  &<Public request parameters>
  ```

- Request body:
  ```json
  {
      "FromUserId": "user_a",
      "GroupId": "group_id",
  }
  ```
  
## 响应参数
| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | Return code. |
| Message | String | Description of the request result. |
| RequestId | String | Request ID. |
