# 设置会话标记

- - -

## 描述

调用此接口，可为多名用户对多个会话设置或取消标记。每个会话至多可存在 30 个标记。

客户端将通过 ZIM SDK 的回调接口，接收会话的标记变化。

## 接口原型

- 请求方法：POST
- 请求地址：`https://zim-api.zego.im/?Action=SetConvMark`
- 传输协议：HTTPS
- 调用频率限制：20次/秒。
