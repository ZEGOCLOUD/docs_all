## ZIMConnectionEvent

导致连接状态发生变更的事件。

**详情**

使得连接状态 Connection State 发生变化的原因。

<Note title="">
- **业务场景**：可用于判断登录/登出是否成功，以及处理断网等异常情况。
</Note>

<Warning title="">
- **注意事项**：请配合 Connection State 参数使用。
</Warning>

### 属性
<ParamField
  name="Success"
  prototype="Success=(0)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
成功。

<Note title="">
- **业务场景**：和连接相关操作执行成功时，连接状态机变更回调中会带上该枚举值，用户可通过该枚举判断调用的连接相关操作执行的结果。
</Note>
</ParamField>
<ParamField
  name="ActiveLogin"
  prototype="ActiveLogin=(1)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
用户主动登录。

<Note title="">
- **业务场景**：用户主动执行登陆操作时，连接状态机变更回调中会带上该枚举值，用户可通过该枚举判断登录操作是否为用户主动执行。
</Note>
</ParamField>
<ParamField
  name="LoginTimeout"
  prototype="LoginTimeout=(2)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
连接超时。
</ParamField>
<ParamField
  name="LoginInterrupted"
  prototype="LoginInterrupted=(3)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
网络连接临时中断。

<Note title="">
- **业务场景**：用户因网络连接临时中断而离线时，连接状态机变更回调中会带上该枚举值，用户可通过该枚举了解造成用户离线的事件。
</Note>
</ParamField>
<ParamField
  name="KickedOut"
  prototype="KickedOut=(4)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
被踢下线。

<Note title="">
- **业务场景**：用户因被踢出而离线时，连接状态机变更回调中会带上该枚举值，用户可通过该枚举了解造成用户离线的事件。
</Note>
</ParamField>
<ParamField
  name="TokenExpired"
  prototype="TokenExpired=(5)"
  parent_name="ZIMConnectionEvent"
  parent_type="enum">
因登录 Token 过期而断开连接。

<Note title="">
- **业务场景**：当开发者使用 Token 鉴权登录后，在 Token 有效期超过后还没调用 [renewToken] 更新 Token 时，将会断开连接。
</Note>
</ParamField>

## ZIMConnectionState

连接状态。

**详情**

标识当前连接状态的状态机。

<Note title="">
- **业务场景**：可用于判断登录/登出是否成功，以及处理断网等异常情况。
</Note>

<Warning title="">
- **注意事项**：请配合 Event 参数使用。
</Warning>

### 属性
<ParamField
  name="Disconnected"
  prototype="Disconnected=(0)"
  parent_name="ZIMConnectionState"
  parent_type="enum">
未连接状态，在登录前和登出之后进入该状态。

<Note title="">
- **业务场景**：如果登录的过程出现稳态异常，例如 AppID 或 Token 不正确，或者有相同用户名在其他地方登录导致本端被踢下线，都会进入该状态。
</Note>
</ParamField>
<ParamField
  name="Connecting"
  prototype="Connecting=(1)"
  parent_name="ZIMConnectionState"
  parent_type="enum">
正在请求连接状态，登录动作执行成功后会进入此状态。

<Note title="">
- **业务场景**：通常通过该状态进行应用界面的展示。如果因为网络质量不佳产生的中断， SDK 会进行内部重试，也会回到正在请求连接状态。
</Note>
</ParamField>
<ParamField
  name="Connected"
  prototype="Connected=(2)"
  parent_name="ZIMConnectionState"
  parent_type="enum">
连接成功状态。

<Note title="">
- **业务场景**：进入该状态表示登录已经成功，用户可以正常使用 SDK 的功能。
</Note>
</ParamField>
<ParamField
  name="Reconnecting"
  prototype="Reconnecting=(3)"
  parent_name="ZIMConnectionState"
  parent_type="enum">
正在进行重新连接状态。

<Note title="">
- **业务场景**：进入该状态表示正在重新登录。
</Note>
</ParamField>
