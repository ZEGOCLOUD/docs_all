# 下载 SDK

- - -

<Note title="说明">
- <b>ZEGO 推出全新 UTS 插件，可同时支持 uni-app、uni-app x 平台的项目</b>，强烈建议开发者 uni-app 老项目使用 UTS 插件（替换原生插件）、或者新建 uni-app x 项目使用 UTS 插件开发，来提高开发效率和 App 运行性能。[点击了解 UTS 插件](https://doc.dcloud.net.cn/uni-app-x/uts/)。
- <b>uni-app 原生插件停止维护。</b> UTS 插件持续迭代，紧跟 Native 发版节奏，且全面覆盖 ZIM、ZPNs、ZIM Audio、zego-callkit。
- ZPNs SDK 封装了主流手机厂商的离线推送功能，支持华为、小米、OPPO、vivo、苹果和 Google 的推送，支持 Android、iOS、Flutter、Web、React Native、uni-app、uni-app x 平台，请参考 [实现离线推送](./../Offline%20push%20notifications/Implement%20offline%20push%20notification.mdx)。
</Note>
