## ZPNsMessage

ZPNs 消息类。

**详情**

ZPNs 消息类。

<Note title="">
- **业务场景**：使用此数据类的对象获取推送信息的详细内容。
</Note>

### 属性
<ParamField
  name="title"
  prototype="title: string "
  parent_name="ZPNsMessage"
  parent_type="class">
消息标题, 如果是通知栏消息，则为通知栏标题。
</ParamField>
<ParamField
  name="content"
  prototype="content: string "
  parent_name="ZPNsMessage"
  parent_type="class">
消息内容，如果是通知栏则为消息通知栏内容，如果为透传消息，则为透传消息体。

<Warning title="">
- **注意事项**：oppo 所有键值大小加起来不超过500个字符(备注：目前是每个值50个字符内，后续调整)。
</Warning>
</ParamField>
<ParamField
  name="pushSourceType"
  prototype="pushSourceType: [ZPNsPushSourceType](./enum#zpnspushsourcetype) "
  parent_name="ZPNsMessage"
  parent_type="class">
push 来源。
</ParamField>
<ParamField
  name="extras"
  prototype="extras: Record<string, any> "
  parent_name="ZPNsMessage"
  parent_type="class">
各个平台的传递的源对象, 需要判断进行类型转换。
</ParamField>
