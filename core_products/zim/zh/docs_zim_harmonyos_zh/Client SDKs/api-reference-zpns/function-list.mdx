# 功能总览



- - -



ZEGO 即时通讯 SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为开发者提供全平台互动、海量并发、超低延时、消息必达的通信服务，助力打造大型直播、语聊房等场景。



## Zpns

 | 方法名 | 描述 | 
 | --- | --- | 
 | [getVersion](./class.mdx#getversion-zpnsmanager) | 获取 SDK 版本号。
[getInstance](./class.mdx#getinstance-zpnsmanager) | 获得 ZPNs 实例。
[on\<K extends keyof ZPNsEventHandler\>](./class.mdx#onk-extends-keyof-zpnseventhandler-zpnsmanager) | 设置 ZPNs 推送事件通知回调。
[off\<K extends keyof ZPNsEventHandler\>](./class.mdx#offk-extends-keyof-zpnseventhandler-zpnsmanager) | 取消设置 ZPNs 推送事件通知回调。
[registerPush](./class.mdx#registerpush-zpnsmanager) | 该方法用于注册厂商离线推送。
[unregisterPush](./class.mdx#unregisterpush-zpnsmanager) | 不需要离线推送时可调用此方法反注册。 |
