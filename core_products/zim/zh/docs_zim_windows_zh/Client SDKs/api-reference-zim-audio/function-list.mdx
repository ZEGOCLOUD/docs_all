# 功能总览



- - -



ZEGO 即时通讯 SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为开发者提供全平台互动、海量并发、超低延时、消息必达的通信服务，助力打造大型直播、语聊房等场景。



## 音频组件功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [getVersion](./class.mdx#getversion-zimaudio) | 获取 SDK 版本号。
[setAdvancedConfig](./class.mdx#setadvancedconfig-zimaudio) | 设置 SDK 进阶配置，使用前请先联系 ZEGO 技术支持。
[init](./class.mdx#init-zimaudio) | 初始化 ZIMAudio SDK
[uninit](./class.mdx#uninit-zimaudio) | 反初始化 ZIMAudio SDK
[setEventHandler](./class.mdx#seteventhandler-zimaudio) | 设置事件通知回调。
[enableANS](./class.mdx#enableans-zimaudio) | 开/关噪声抑制
[enableAGC](./class.mdx#enableagc-zimaudio) | 开/关自动增益控制
[setANSParam](./class.mdx#setansparam-zimaudio) | 设置噪声抑制参数，当前仅包含噪声抑制模式
[startRecord](./class.mdx#startrecord-zimaudio) | 开始录制音频文件
[completeRecord](./class.mdx#completerecord-zimaudio) | 完成录制音频文件
[cancelRecord](./class.mdx#cancelrecord-zimaudio) | 停止录制音频文件
[isRecording](./class.mdx#isrecording-zimaudio) | 获取是否正在录制
[setAudioRouteType](./class.mdx#setaudioroutetype-zimaudio) | 设置音频路由类型
[startPlay](./class.mdx#startplay-zimaudio) | 开始播放音频文件
[stopPlay](./class.mdx#stopplay-zimaudio) | 停止播放音频文件
[isPlaying](./class.mdx#isplaying-zimaudio) | 获取是否正在播放 |
