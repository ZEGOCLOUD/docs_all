  parent_name="ZIMConnectionState"
  parent_type="enum">

<Note title="">
- **业务场景**：如果登录的过程出现稳态异常，例如 AppID 或 Token 不正确，或者有相同用户名在其他地方登录导致本端被踢下线，都会进入该状态。
</Note>
</ParamField>

  prototype="ZIMConnectionStateConnecting=(1)"
  parent_name="ZIMConnectionState"
  parent_type="enum">

<Note title="">
- **业务场景**：通常通过该状态进行应用界面的展示。如果因为网络质量不佳产生的中断， SDK 会进行内部重试，也会回到正在请求连接状态。
</Note>
</ParamField>
<ParamField
  name="ZIMConnectionStateConnected"
  prototype="ZIMConnectionStateConnected=(2)"
  parent_name="ZIMConnectionState"
  parent_type="enum">

<Note title="">
