# 功能总览



- - -



ZEGO 即时通讯 SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为开发者提供全平台互动、海量并发、超低延时、消息必达的通信服务，助力打造大型直播、语聊房等场景。



API 返回的错误码，请参考 https://doc-zh.zego.im/article/11606。



## 离线推送功能（需单独集成）

 | 方法名 | 描述 | 
 | --- | --- | 
 | [updateUserBadge](./class.mdx#updateuserbadge-zim) | 更新 iOS 离线通知徽标数量。
[enableDebug](./class.mdx#enabledebug-zpnsmanager) | 是否开启调试模式
[getInstance](./class.mdx#getinstance-zpnsmanager) | 获得 ZPNs 实例。
[getPushConfig](./class.mdx#getpushconfig-zpnsmanager) | 获取 PushConfig 配置类。
[getZPNsMessage](./class.mdx#getzpnsmessage-zpnsmanager) | 用户点击通知栏跳转 activity 时，可通过该接口获取 payload 等信息。
[registerPush](./class.mdx#registerpush-zpnsmanager) | 该方法用于注册厂商离线推送。
[setPushConfig](./class.mdx#setpushconfig-zpnsmanager) | 设置各家厂商的推送设置项。
[unregisterPush](./class.mdx#unregisterpush-zpnsmanager) | 不需要离线推送时可调用此方法反注册。
[onNotificationArrived](./class.mdx#onnotificationarrived-zpnsmessagereceiver) | 厂商通知展示回调。
[onNotificationClicked](./class.mdx#onnotificationclicked-zpnsmessagereceiver) | 厂商通知点击回调。
[onRegistered](./class.mdx#onregistered-zpnsmessagereceiver) | 生成 PushID 后触发的回调。
[onThroughMessageReceived](./class.mdx#onthroughmessagereceived-zpnsmessagereceiver) | 厂商透传消息回调。 |
