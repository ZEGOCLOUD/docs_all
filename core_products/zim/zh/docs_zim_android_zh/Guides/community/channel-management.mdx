## еҠҹиғҪз®Җд»Ӣ

йў‘йҒ“пјҲChannelпјүжҳҜзӨҫзҫӨеҶ…иҒҠеӨ©дә’еҠЁзҡ„еҹәжң¬иҪҪдҪ“пјҢеҠҹиғҪзұ»дјјдј з»ҹзҫӨз»„пјҢжҜҸдёӘзӨҫзҫӨеҸҜеҢ…еҗ«еӨҡдёӘйў‘йҒ“гҖӮZIM SDK жҸҗдҫӣдәҶе®Ңж•ҙзҡ„йў‘йҒ“з®ЎзҗҶиғҪеҠӣпјҢж”ҜжҢҒд»ҘдёӢеҠҹиғҪпјҡ

- еҲӣе»ә/и§Јж•Јйў‘йҒ“
- жӣҙж–°йў‘йҒ“еҗҚз§°гҖҒеӨҙеғҸгҖҒе…¬е‘Ҡ
- и®ҫзҪ®/еҲ йҷӨйў‘йҒ“иҮӘе®ҡд№үеұһжҖ§
- жҹҘиҜўйў‘йҒ“дҝЎжҒҜеҸҠеҲ—иЎЁ
- и®ҫзҪ®йў‘йҒ“зҰҒиЁҖзҠ¶жҖҒ
- зӣ‘еҗ¬йў‘йҒ“еҲ—иЎЁеҸҳжӣҙдёҺдҝЎжҒҜжӣҙж–°

## еүҚжҸҗжқЎд»¶

- иҜ·еҸӮиҖғ [е®һзҺ°еҹәжң¬ж¶ҲжҒҜж”¶еҸ‘](../Messaging/Send%20and%20receive%20messages.mdx) е®ҢжҲҗ ZIM SDK иҺ·еҸ–гҖҒеҲқе§ӢеҢ–е’Ңз”ЁжҲ·зҷ»еҪ•гҖӮ
- иҜ·еҸӮиҖғ [дҪҝз”Ё Token йүҙжқғ](../Users/Authentication.mdx) е®һзҺ°з”ЁжҲ·йүҙжқғзҷ»еҪ•гҖӮ
- зӨҫзҫӨеҠҹиғҪйңҖиҰҒ ZIM SDK 3.0.0 еҸҠд»ҘдёҠзүҲжң¬гҖӮ
- зӨҫзҫӨеҠҹиғҪдёәж——иҲ°зүҲеҠҹиғҪпјҢдҪҝз”ЁеүҚиҜ·иҒ”зі» ZEGO жҠҖжңҜж”ҜжҢҒејҖйҖҡгҖӮ
- зӨҫзҫӨеҠҹиғҪзҡ„ Token з”ҹжҲҗж–№ејҸдёҺе…¶д»– ZIM еҠҹиғҪдёҖиҮҙпјҢж— йңҖйўқеӨ–жқғйҷҗеЈ°жҳҺгҖӮ

### йў‘йҒ“зұ»еһӢиҜҙжҳҺ

| зұ»еһӢ | иҜҙжҳҺ | еҠ е…Ҙж–№ејҸ | еҸҜи§ҒжҖ§ |
|------|------|----------|--------|
| GENERAL | й»ҳи®Өйў‘йҒ“пјҢеҲӣе»әзӨҫзҫӨж—¶иҮӘеҠЁеҲӣе»әпјҢзӨҫзҫӨзә§еҲ«зҡ„ Tips е’ҢйҖҡзҹҘдјҡеҸ‘еңЁжӯӨйў‘йҒ“ | иҮӘеҠЁеҠ е…Ҙ | жүҖжңүжҲҗе‘ҳеҸҜи§Ғ |
| PUBLIC | е…¬ејҖйў‘йҒ“пјҢйңҖиҰҒеҲӣе»әзӨҫзҫӨд№ӢеҗҺеҶҚеҲӣе»ә | иҮӘеҠЁеҠ е…Ҙ | жүҖжңүжҲҗе‘ҳеҸҜи§Ғ |


<Note title="иҜҙжҳҺ">
`GENERAL` е’Ң `PUBLIC` еҠҹиғҪе®Ңе…ЁдёҖж ·пјҢеҢәеҲ«д»…еңЁдәҺеҲӣе»әж—¶жңәпјҡ`GENERAL` еңЁеҲӣе»әзӨҫзҫӨж—¶иҮӘеҠЁеҲӣе»әпјҢ`PUBLIC` йңҖиҰҒеңЁеҲӣе»әзӨҫзҫӨеҗҺжүӢеҠЁеҲӣе»әгҖӮ
</Note>

## и®ҫзҪ®йў‘йҒ“зҰҒиЁҖ

и°ғз”Ё {getPlatformData(props,muteCommunityChannelsMap)} жҺҘеҸЈпјҢжү№йҮҸи®ҫзҪ®жҢҮе®ҡйў‘йҒ“зҡ„зҰҒиЁҖзҠ¶жҖҒгҖӮзҰҒиЁҖй…ҚзҪ®йҖҡиҝҮ {getPlatformData(props,ZIMCommunityChannelMuteConfigMap)} дј е…ҘгҖӮ

| й…ҚзҪ®еӯ—ж®ө | иҜҙжҳҺ |
|---------|------|
| `mode` | зҰҒиЁҖиҢғеӣҙгҖӮжһҡдёҫеҖјеҜ№з…§пјҡ`NONE`(0) = дёҚзҰҒиЁҖгҖҒ`NORMAL`(1) = зҰҒиЁҖжҷ®йҖҡжҲҗе‘ҳгҖҒ`ALL`(2) = е…Ёе‘ҳзҰҒиЁҖгҖҒ`CUSTOM`(3) = иҮӘе®ҡд№үи§’иүІиҢғеӣҙ |
| `duration` | зҰҒиЁҖж—¶й•ҝпјҲз§’пјүпјҢеҸ–еҖјиҢғеӣҙ 1-2592001пјҲзәҰ 30 еӨ©пјүпјҢжҲ– `-1` иЎЁзӨәж°ёд№…зҰҒиЁҖ |
| `roles` | еҪ“ `mode` дёә `CUSTOM` ж—¶пјҢжҢҮе®ҡиў«зҰҒиЁҖзҡ„и§’иүІеҲ—иЎЁ |
