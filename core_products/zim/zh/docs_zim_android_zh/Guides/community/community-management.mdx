# 社群管理

- - -

## 功能简介

ZIM SDK 提供了完整的社群管理能力，支持以下功能：

- 创建/解散社群
- 加入/离开社群
- 查询已加入的社群列表及社群信息
- 更新社群名称、头像、公告
- 设置/删除社群自定义属性
- 设置社群消息通知状态
- 监听社群列表变更与信息更新

<Note title="说明">

- 社群功能为旗舰版功能，使用前请联系 ZEGO 技术支持开通。
- 使用社群相关功能前，请确保已完成 ZIM SDK 登录。
</Note>

## 创建社群

登录 ZIM SDK 后，调用 {getPlatformData(props,createCommunityMap)} 接口，传入社群基础信息 {getPlatformData(props,ZIMCommunityInfoMap)} 与创建配置 {getPlatformData(props,ZIMCommunityCreateConfigMap)} 即可创建社群。创建者自动成为社群所有者。

创建结果通过 {getPlatformData(props,ZIMCommunityCreatedCallbackMap)} 返回，成功后可获取社群的完整信息。

<Warning title="注意">

- `communityID` 支持开发者自定义，仅支持数字、英文字符及以下特殊字符：`! # $ % & ( ) + - : ; < = . > ? @ [ ] ^ _ { } | ~`，且不能以 `#` 开头。若不填，ZIM 服务器将自动生成。
- 创建成功后即已加入社群，无需再调用加入接口。
</Warning>

:::if{props.platform=undefined}
<CodeGroup>
```java title="示例代码"
ZIMCommunityInfo communityInfo = new ZIMCommunityInfo();
communityInfo.communityID = "community_001";  // 可自定义，留空则由服务器自动生成
communityInfo.communityName = "我的社群";
communityInfo.communityAvatarUrl = "https://example.com/avatar.png";

ZIMCommunityCreateConfig config = new ZIMCommunityCreateConfig();
config.communityNotice = "欢迎加入！";
HashMap<String, String> attributes = new HashMap<>();
attributes.put("key1", "value1");
config.communityAttributes = attributes;

zim.createCommunity(communityInfo, config, new ZIMCommunityCreatedCallback() {
    @Override
    public void onCommunityCreated(ZIMCommunityFullInfo communityFullInfo, ZIMError errorInfo) {
        if (errorInfo.code == ZIMErrorCode.SUCCESS) {
            // 创建成功，communityFullInfo 包含社群完整信息
        }
    }
});
```
</CodeGroup>
:::
:::if{props.platform="iOS|mac"}
<CodeGroup>

<Warning title="注意">

只有社群所有者才能调用此接口，管理员和普通成员无权解散社群。
</Warning>

## 查询社群列表

调用 {getPlatformData(props,queryCommunityListMap)} 接口，分页拉取当前用户已加入的社群列表。

分页规则：
- 首次查询时，将 `config.nextFlag` 设为 `0`。
- 收到回调后，若返回的 `nextFlag` 不为 `0`，将其作为下一次请求的 `config.nextFlag` 继续拉取。
- 当返回的 `nextFlag` 为 `0` 时，表示已无更多数据。

:::if{props.platform=undefined}
<CodeGroup>
```java title="示例代码"
ZIMCommunityListQueryConfig config = new ZIMCommunityListQueryConfig();
config.nextFlag = 0;  // 首次查询设为 0

zim.queryCommunityList(100, config, new ZIMCommunityListQueriedCallback() {
    @Override
    public void onCommunityListQueried(ArrayList<ZIMCommunity> communityList, long nextFlag, ZIMError errorInfo) {
        if (errorInfo.code == ZIMErrorCode.SUCCESS) {
            // communityList 为当前页的社群列表
            // nextFlag != 0 时，可继续分页拉取
        }
    }
});
```
</CodeGroup>
:::
:::if{props.platform="iOS|mac"}
<CodeGroup>
```objc title="示例代码"

config.nextFlag = 0;

[zim queryCommunityList:100 config:config callback:^(NSArray<ZIMCommunity *> * _Nonnull communityList, unsigned int nextFlag, ZIMError * _Nonnull errorInfo) {
    if (errorInfo.code == ZIMErrorCodeSuccess) {
        // 查询成功
    }
}];
```
</CodeGroup>
:::
:::if{props.platform="window"}
<CodeGroup>

```cpp title="示例代码"
            // 查询成功
        }
```

</CodeGroup>
:::

## 监听社群变更

### 社群列表变更

当用户加入或离开社群、社群被解散等情况导致社群列表发生变化时，SDK 会触发 {getPlatformData(props,onCommunityListChangedMap)} 回调。回调中的 `changeInfoList` 包含变更类型（加入/离开）及对应的社群信息。

:::if{props.platform=undefined}
<CodeGroup>
```java title="示例代码"
zim.setEventHandler(new ZIMEventHandler() {
    @Override
    public void onCommunityListChanged(ZIM zim, ZIMCommunityListChangedEventResult result) {
        for (ZIMCommunityChangeInfo changeInfo : result.changeInfoList) {
            // changeInfo.action：变更类型（加入 / 离开）
            // changeInfo.community：变更的社群信息
        }
    }
});
```
</CodeGroup>
:::
:::if{props.platform="iOS|mac"}
<CodeGroup>
```objc title="示例代码"
- (void)zim:(ZIM *)zim communityListChangedWithResult:(ZIMCommunityListChangedEventResult *)result {

        // changeInfo.action：变更类型
        // changeInfo.community：变更的社群信息
    }
}
```
</CodeGroup>
:::
:::if{props.platform="window"}
<CodeGroup>
```cpp title="示例代码"
void onCommunityListChanged(zim::ZIM *zim,
```
