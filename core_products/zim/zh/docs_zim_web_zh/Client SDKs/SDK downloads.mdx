# 下载

- - -

ZEGO 即时通讯（ZIM）SDK 由深圳市即构科技有限公司提供，您可以在本页面获取适用于 Web 客户端的 ZIM SDK，当前可下载版本为 2.20.0，发布日志请参考 [发布日志 - ZIM](./ZIM%20release%20notes.mdx)，合规事宜请参考 [ZEGO 安全合规白皮书](https://doc-zh.zego.im/article/16142)。

| SDK | 说明 | 下载地址 | 相关文档 |
|-----|------|----------|----------|
| ZIM | ZEGO 即时通讯 SDK | [ZIM SDK v2.20.0](https://www.npmjs.com/package/zego-zim-web) | <ul><li>[快速开始](./../Send%20and%20receive%20messages.mdx)</li><li>[跑通示例源码](./../Sample%20code.mdx)</li><li>[发布日志 - ZIM](./ZIM%20release%20notes.mdx)</li></ul> |
| ZPNs | 用于接收离线推送的 SDK | [ZPNs SDK v2.5.0](https://storage.zego.im/zpns/sdk/web/ZPNs-main-web-shared-javascript.zip) | <ul><li>[实现离线推送](./../Offline%20push%20notifications/Implement%20offline%20push%20notification.mdx)</li><li>[发布日志 - ZPNs](./ZPNs%20release%20notes.mdx)</li></ul> |

<Warning title="注意">

如果将 ZIM Web SDK 从高版本回退至低版本时，请先清理本地数据库，否则会因为打开数据库异常而导致登录失败。
</Warning>
