## ZIMAudioRecordConfig

录制配置

**详情**

配置录制时的录制文件输出路径和最大的录制时长

*Declared in `ZIMAudioDefines.h`*

### 属性
<ParamField
  name="filePath"
  prototype="@property NSString *filePath"
  parent_name="ZIMAudioRecordConfig"
  parent_type="class">
录制文件的输出路径。当前仅支持后缀为 .mp3 或 .m4a 的完整绝对路径，传入其他格式的文件路径时 SDK 将会抛出非法的文件路径错误
</ParamField>
<ParamField
  name="maxDuration"
  prototype="@property unsigned int maxDuration"
  parent_name="ZIMAudioRecordConfig"
  parent_type="class">
最大录制时长，默认为 60000 ms，即 60s。此参数的单位为毫秒，最大值不超过 120000
</ParamField>

## ZIMAudioPlayConfig

播放配置

**详情**

配置指定当前需要播放的音频文件路径以及路由类型

*Declared in `ZIMAudioDefines.h`*

### 属性
<ParamField
  name="filePath"
  prototype="@property NSString *filePath"
  parent_name="ZIMAudioPlayConfig"
  parent_type="class">
需要播放文件的绝对路径。当前仅支持后缀为 .mp3 或 .m4a 的完整绝对路径，传入其他格式的文件路径时 SDK 将会抛出非法的文件路径错误
</ParamField>
<ParamField
  name="routeType"
  prototype="@property [ZIMAudioRouteType](./enum#zimaudioroutetype) routeType"
  parent_name="ZIMAudioPlayConfig"
  parent_type="class">
播放时的起始音频路由类型，默认播放设备为扬声器
</ParamField>
