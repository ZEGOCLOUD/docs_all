# 功能总览

- - -

ZEGO 即时通讯 SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为开发者提供全平台互动、海量并发、超低延时、消息必达的通信服务，助力打造大型直播、语聊房等场景。

API 返回的错误码，请参考 https://doc-zh.zego.im/article/11606。


## 离线推送功能（需单独集成）

| 方法名 | 描述 | 
| --- | --- | 
|[addLocalNotification](./class.mdx#addlocalnotification-zpns) | 添加一条本地推送。|
|[applyNotificationPermission](./class.mdx#applynotificationpermission-zpns) | 向系统申请推送通知权限|
|[createNotificationChannel](./class.mdx#createnotificationchannel-zpns) | Android 平台时，创建一个通知通道。|
|[enableDebug](./class.mdx#enabledebug-zpns) | 是否开启调试模式|
|[getInstance](./class.mdx#getinstance-zpns) | 获得 ZPNs 实例。|
|[registerPush](./class.mdx#registerpush-zpns) | 该方法用于注册厂商离线推送。|
|[setBackgroundMessageHandler](./class.mdx#setbackgroundmessagehandler-zpns) | Android 平台 App 杀掉后，接收离线的静默推送通知。|
|[setLocalBadge](./class.mdx#setlocalbadge-zpns) | 设置 app 本地角标数。|
|[setPushConfig](./class.mdx#setpushconfig-zpns) | 设置各家厂商的推送设置项。|
|[setServerBadge](./class.mdx#setserverbadge-zpns) | 通过该接口向 ZPNs 服务器上报当前 App 的角标数。|
|[unregisterPush](./class.mdx#unregisterpush-zpns) | 不需要离线推送时可调用此方法反注册。|
|[notificationArrived](./class.mdx#notificationarrived-zpnseventhandler) | 厂商通知展示回调。|
|[notificationClicked](./class.mdx#notificationclicked-zpnseventhandler) | 厂商通知点击回调。|
|[registered](./class.mdx#registered-zpnseventhandler) | 生成 PushID 后触发的回调。|
|[throughMessageReceived](./class.mdx#throughmessagereceived-zpnseventhandler) | 厂商透传消息回调。 |
