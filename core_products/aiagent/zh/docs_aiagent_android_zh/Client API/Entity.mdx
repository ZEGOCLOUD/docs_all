## AuthData

AuthData 类用于存储和管理应用程序的认证信息，包含以下属性：
- `appID`: 应用程序ID，用于标识应用程序的唯一性
- `appSign`: 应用程序签名，用于验证应用程序的合法性
- `token`: 认证令牌，用于后续的API调用认证。可以从即构控制台获取。

<Note title="说明">
关于token，请参考[Token 鉴权](../Quick%20start/Authentication.mdx)
</Note>

示例用法：
```java
AuthData authData = new AuthData(12345678, "your_app_sign", "your_token");
```

## UserProfile

UserProfile 类用于存储和管理用户的基本信息，包含以下属性：
- `userID`: 用户唯一标识符
- `userName`: 用户名称

示例用法：
```java
UserProfile profile = new UserProfile("user123", "TestUser");
```

## ChatConfig

ChatConfig 类用于配置聊天相关参数，包含以下属性：
- `llmSystemMode`: LLM系统模式
- `enableLLMServerMessage`: 是否启用LLM服务器消息

示例用法：
```java
ChatConfig chatConfig = new ChatConfig();
chatConfig.llmSystemMode = xxx;
chatConfig.enableLLMServerMessage = xxx;
```

## CustomAgentConfig

CustomAgentConfig 类用于配置 AI Agent 的会话参数，包含以下属性：
- `agentTemplateId`: 智能体模板ID
- `name`: 智能体名称
- `avatar`: 智能体头像URL
- `asrConfig`: 语音识别配置
- `llmConfig`: 大语言模型配置
- `ttsConfig`: 文本转语音配置
- `sex`: 性别
- `intro`: 介绍
- `system`: 系统提示词
- `welcomeMessage`: 欢迎消息
- `metaHumanID`: 数字人ID

示例用法：
```java
CustomAgentConfig config = new CustomAgentConfig();
config.name = "助手";
config.sex = "男";
config.intro = "这是一个智能助手";
```

## QueryConversationListData

QueryConversationListData 类用于存储会话列表查询的结果数据，包含以下属性：
- `conversationList`: 会话列表数组，包含多个 Conversation 对象
- `totalCount`: 会话总数

示例用法：
```java
QueryConversationListData data = response.getData();
List<Conversation> conversations = data.conversationList;
int total = data.totalCount;
```

## Conversation

Conversation 类用于表示一个 AI 会话的完整信息，包含以下属性：
- `conversationId`: 会话唯一标识符
- `userId`: 用户ID
- `agentId`: AI 智能体ID
- `agentTemplateId`: 智能体模板ID
- `customAgentConfig`: 自定义智能体配置
- `chatConfig`: 聊天配置
- `createTime`: 会话创建时间
- `updateTime`: 会话更新时间

示例用法：
```java
Conversation conversation = response.getConversation();
String conversationId = conversation.conversationId;
CustomAgentConfig agentConfig = conversation.customAgentConfig;
```

## LLMConfig

LLMConfig 类用于存储和管理大型语言模型（LLM）的配置信息，包含以下属性：
- `type`: 配置类型
- `model`: 模型名称
- `ignoreBracketText`: 忽略括号文本的设置（整数数组）
- `responseFormatType`: 响应格式类型（整数）
- `responseMessageName`: 响应消息名称
- `extensionLLMParams`: 扩展 LLM 参数

示例用法：
```java
LLMConfig config = new LLMConfig();
config.type = "default";
config.model = "gpt-3.5-turbo";
config.ignoreBracketText = new int[]{1, 0};
config.responseFormatType = 1;
config.responseMessageName = "response";
config.extensionLLMParams = "{}";
```

## TTSConfig

TTSConfig 类用于存储和管理文本转语音（TTS）的配置信息，包含以下属性：
- `type`: 使用的语音厂商
- `voice`: 使用的音色
- `extensionParams`: 扩展参数（ExtensionParams 类型）
- `ignoreBracketText`: 忽略括号文本的设置（整数数组）
- `ignoreCustomBracketText`: 忽略自定义括号文本的设置（字符串数组）
- `silenceDuration`: 静音时长（整数）
- `extensionTTSParams`: 扩展 TTS 参数

示例用法：
```java
TTSConfig config = new TTSConfig();
config.type = "default";
config.voice = "zh-CN-Xiaoxiao";
config.extensionParams = new ExtensionParams();
config.ignoreBracketText = new int[]{1, 0};
config.ignoreCustomBracketText = new String[]{"[example]"};
config.silenceDuration = 500;
config.extensionTTSParams = "{}";
```

## ASRConfig

```java
config.extensionASRParams = "{}";
```
