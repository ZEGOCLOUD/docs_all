# 删除智能体实例

---

## 描述

通过向本接口，您可以删除一个现有的智能体实例。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-aiagent-api.zegotech.cn?Action=DeleteAgentInstance`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数             | 类型    | 是否必选 | 描述         |
|------------------|---------|----------|--------------|
| AgentInstanceId  | String  | 是       | 智能体实例的唯一标识，通过 [创建智能体实例](./Create%20Agent%20instance.mdx) 接口的响应参数获取。 |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=DeleteAgentInstance
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "AgentInstanceId": "1909811657303920640"
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 此请求结果为 `null`。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "Succeed",
    "RequestId": "1843985617336143872",
    "Data": null
}
```
