# 获取智能体详情
---

##  描述

通过向本接口，您可以获取指定智能体的详情。

import PostPrototype from '/snippets/common/zh/server/post-prototype.mdx'

<PostPrototype link="https://aigc-aiagent-api.zegotech.cn?Action=QueryAgents" />

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../accessing-server-apis.mdx#公共请求参数)。

| 参数               | 类型   | 是否必选 | 描述             |
|------------------|------|------|-------------------------|
| AgentIds          | Array of String | 是    | 已注册智能体 ID 列表，长度上限为 10。 |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=QueryAgents
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "AgentIds": ["xiaozhi"]
    }
    ```
## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../return-codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 返回数据。 |
| └Agents | Array of Object | 智能体列表。本参数的结构，请参 [智能体参数说明 - Agents](./../common-parameter-description.mdx#agents)。
