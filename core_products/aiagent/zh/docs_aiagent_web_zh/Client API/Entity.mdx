## AIAgentResult

AIAgentResult 接口用于表示AI智能体操作的结果，包含以下属性：
- `code`: 结果码
- `message`: 结果信息
- `requestId`: 请求ID（可选）

示例用法：
```typescript
const result: AIAgentResult = {
  code: 0,
  message: "操作成功",
  requestId: "req_123456"
};
```

## AIAgentCreateResult

AIAgentCreateResult 接口继承自 AIAgentResult，用于表示创建会话的结果，额外包含以下属性：
- `conversation`: 创建的会话对象（可选）

示例用法：
```typescript
const createResult: AIAgentCreateResult = {
  code: 0,
  message: "会话创建成功",
  requestId: "req_123456",
  conversation: {
    conversationId: "conv_123",
    userId: "user_001",
    agentId: "agent_001",
    agentTemplateId: "template_001",
    customAgentConfig: {...},
    chatConfig: {...},
    isChatting: false
  }
};
```

## AIAgentConversationList

AIAgentConversationList 接口用于表示会话列表查询结果，包含以下属性：
- `code`: 结果码
- `message`: 结果信息
- `requestId`: 请求ID（可选）
- `conversationList`: 会话列表（可选）

示例用法：
```typescript
const conversationList: AIAgentConversationList = {

  conversationList: [
    {
      conversationId: "conv_123",
      userId: "user_001",
      agentId: "agent_001",
      agentTemplateId: "template_001",
      customAgentConfig: {...},
      chatConfig: {...},
      isChatting: false
    }
  ]
};
```

## Conversation

Conversation 接口用于描述会话信息，包含以下属性：
- `conversationId`: 会话ID
- `userId`: 用户ID
- `agentId`: AI智能体ID
- `agentTemplateId`: AI智能体模板ID
- [customAgentConfig](#customagentconfig): AI智能体自定义配置
- [chatConfig](#chatconfig): 聊天配置
- `isChatting`: 是否正在聊天（可选）

示例用法：
```typescript
const conversation: Conversation = {
  conversationId: "conv_123",
  userId: "user_001",
  agentId: "agent_001",
  agentTemplateId: "template_001",
  customAgentConfig: {
    agentTemplateId: "template_001",
    name: "AI助手",
    avatar: "https://example.com/avatar.png",
    intro: "我是一个AI助手",
    system: "你是一个专业的AI助手"
  },
  chatConfig: {
    llmSystemMode: 0,
    enableLLMServerMessage: 1
  },
  isChatting: false
};
```

## CustomAgentConfig

CustomAgentConfig 接口用于配置AI智能体，包含以下属性：
- `agentTemplateId`: AI智能体模板ID
- `name`: AI智能体名称
- `avatar`: AI智能体头像URL
- `intro`: AI智能体简介
- `system`: AI智能体系统提示词
- [llm](#llmconfig): 大语言模型配置（可选）
- [tts](#ttsconfig): 语音合成配置（可选）
- [asr](#asrconfig): 语音识别配置（可选）
- `welcomeMessage`: AI智能体欢迎语（可选）
- `sex`: AI智能体性别（可选）
- [metaHumanConfig](#metahumanconfig): 数字人配置（可选）

示例用法：
```typescript
const customAgentConfig: CustomAgentConfig = {
  agentTemplateId: "template_001",
  name: "AI助手",
  avatar: "https://example.com/avatar.png",
  intro: "我是一个专业的AI助手",
  system: "你是一个专业的AI助手",
  llm: {
    type: "gpt-3.5",
    model: "gpt-3.5-turbo",
    responseFormatType: 0,
    responseMessageName: "AI助手"
  },
  tts: {
    type: "standard",
    voice: "female_01"
  },
  asr: {
    hotWordId: "hotword_001"
  },
  welcomeMessage: "你好，我是AI助手",
  sex: "female",
  metaHumanConfig: {
    metaHuman: {
      metaHumanId: "mh_001",
      layout: {
        top: 0,
        left: 0,
        width: 1280,
        height: 720,
        layer: 1
      }
    }
  }
};
```

## LLMConfig

LLMConfig 接口用于配置大语言模型，包含以下属性：
- `type`: 大语言模型类型
- `model`: 模型名称
- `responseFormatType`: 响应格式类型
- `responseMessageName`: 响应消息名称

示例用法：
```typescript
const llmConfig: LLMConfig = {
  type: "gpt-3.5",
  model: "gpt-3.5-turbo",
  responseFormatType: 0,
  responseMessageName: "AI助手"
};
```

## MetaHumanConfig

MetaHumanConfig 接口用于配置数字人，包含以下属性：
- `metaHuman`: 数字人ID及布局配置（可选）
- `backgroundColor`: 背景颜色（可选）
- `driver`: 驱动配置（可选）
- `videoOption`: 视频选项配置（可选）
- `subtitle`: 字幕配置（可选）

示例用法：
```typescript
const metaHumanConfig: MetaHumanConfig = {
  metaHuman: {
    metaHumanId: "mh_001",
    layout: {
      top: 0,
      left: 0,
      width: 1280,
      height: 720,
      layer: 1
    }
  },
  backgroundColor: "#FFFFFF",
  driver: {
    driverType: 0,
    pitchRate: 1.0,
    speechRate: 1.0,
    volume: 1.0,
    timbreId: "timbre_001"
  },
  videoOption: {
    width: 1280,
    height: 720,
    format: "mp4"
  },
  subtitle: {
    enable: 1
  }
};
```

## UpdateConversationConfig

UpdateConversationConfig 接口用于更新会话配置，包含以下属性：
- `agentTemplateId`: AI智能体模板ID（可选）
- [customAgentConfig](#customagentconfig): AI智能体自定义配置（可选）
- [chatConfig](#chatconfig): 聊天配置（可选）

示例用法：
```typescript
const updateConfig: UpdateConversationConfig = {
  agentTemplateId: "new_template_001",
  customAgentConfig: {
    name: "新AI助手",
    avatar: "https://example.com/new_avatar.png"
  },
  chatConfig: {
    llmSystemMode: 1,
    enableLLMServerMessage: 1
  }
};
```

## LLMSystemMode

LLMSystemMode 枚举用于定义LLM系统模式：
- `default`: 使用会话配置CustomAgentConfig的System
- `asrParallel`: 优先使用会话配置CustomAgentConfig的System；并行业务基于ASR识别结果自定义本轮对话System
- `asrWithTimeout`: 业务基于ASR识别结果自定义本轮对话System，5s超时后使用默认System

示例用法：
```typescript
const mode: LLMSystemMode = LLMSystemMode.default; // 0
```

## EnableLLMServerMessage

EnableLLMServerMessage 枚举用于定义是否启用LLM服务器消息：
- `disable`: 不启用
- `enable`: 启用

示例用法：
```typescript
const enable: EnableLLMServerMessage = EnableLLMServerMessage.enable; // 1
```

## ChatMode

ChatMode 枚举用于定义聊天模式：
- `voice`: 语音模式
- `metaHuman`: 数字人模式

示例用法：
```typescript
const mode: ChatMode = ChatMode.voice; // 0
```
