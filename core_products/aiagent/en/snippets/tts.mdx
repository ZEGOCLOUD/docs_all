**TTS Parameters**

| Parameter | Type | Required or Optional | Description   |
| ----------------------- | --------------- | -------- | ----- |
| Type                    | String          | Yes      | Text-to-speech (TTS) type. Optional: <ul><li>`Aliyun` : Alibaba Cloud TTS.</li><li>`Huoshan` : Huoshan Cloud TTS.</li><li>`Minimax` : MiniMax.</li></ul> |
| Voice                   | String          | Yes      | Timbre, the timbres supported by different TTS systems vary. For details, please refer to the official documentation of the corresponding cloud provider: <ul><li>Alibaba Cloud TTS: [Speech Synthesis - API Description - List of Timbres](https://help.aliyun.com/zh/isi/developer-reference/overview-of-speech-synthesis?spm=a2c4g.11186623.0.0.54de622dhrDXcY#5186fe1abb7ag)</li><li>BytePlus TTS: [Large Model Speech Synthesis - List of Timbres](https://www.volcengine.com/docs/6561/1257544)</li><li>MiniMax: [T2A v2 (Speech Generation) - API Description - ](https://platform.minimaxi.com/document/T2A%20V2?key=66719005a427f0c8a5701643#YqSh1KAoyms1WH4XJrdeIrrb)</li></ul>|
| ExtensionParams         | Object          | No       | Third-party TTS extension parameters. For the structure of this parameter, see [ExtensionParams](#extensionparams). |
| IgnoreBracketText       | Array of Number | No       | <p>Filter out the text within specified punctuation marks in the LLM return content and then perform speech synthesis.</p><p>The supported values and their meanings are as follows:</p><ul><li>1: Chinese parentheses（）</li><li>2: English parentheses ()</li><li>3: Chinese square brackets【】</li><li>4: English square brackets []</li><li>5: English curly brackets {}</li><li>Default value: [1, 2], which means filtering for （）、()</li><li>Empty array: Pass [], representing an empty value, no filtering will be performed.</li></ul><Note title="Note">You can define which content needs to be placed within the specified punctuation marks in [CustomAgentConfig > System](#customagentconfig) by yourself.</Note> |
| IgnoreCustomBracketText | Array of String | No       | <p>Filter the text between two identical contents in the LLM return, and then perform speech synthesis.</p><ul><li>Configuration: ["###"]</li><li>Effect:<br/>###<br/>Ignored content<br/>###</li></ul>    |
| SilenceDuration         | Number          | No       | When performing TTS based on ZEGOCLOUD sentence segmentation, the pause duration at the sentence break will additionally increase by the corresponding time. The default is 0, and the unit is in milliseconds. The value range is [0, 5000]. |
| ExtensionTTSParams      | String          | No       | <p>Third-party TTS request parameters, usually the third-party request body.</p><p>Format: Escaped JSON string.</p>For example: `\{\"key1\":\"value1\",\"key2\":\"value2\",\"key3\":0.1,\"key4\":1.0\}`<p>Currently supported third-party parameters:</p><ul><li>MiniMax Large Model TTS.</li><li>BytePlus TTS (Large Model Voice Synthesis API).</li><li>BytePlus TTS (Bidirectional Streaming API).</li></ul>For this parameter structure, see [ExtensionTTSParams (Descriptions of Pass-Through Parameters for Each Third-Party TTS)](#extensionttsparams-descriptions-of-pass-through-parameters-for-each-third-party-tts).    |
