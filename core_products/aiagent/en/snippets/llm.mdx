**LLM Parameters**

| Parameter            | Type              | Required | Description                                                                                     |
|----------------------|-------------------|----------|-----------------------------------------------------------------------------------------------|
| Type                 | String            | Yes      | Large language model (LLM) type. Options:<ul><li>`WenXin`: Wenxin Yiyan.</li><li>`Doubao`: DouBao.</li><li>`Minimax`: MiniMax.</li><li>`StepFun`: StepFun.</li><li>`Qwen`: Tongyi Qianwen.</li></ul> |
| Model                | String            | Yes      | Model name. Different LLMs support different models, please refer to the official documentation of each cloud provider:<ul><li>Wenxin Yiyan: [Inference Service API V1 - Interface Overview - Dialogue Chat](https://cloud.baidu.com/doc/WENXINWORKSHOP/s/om5aq2brc#%E5%AF%B9%E8%AF%9Dchat)</li><li>DouBao: [Dialogue(Chat)-Text API - Request Parameters - Request Body - model](https://www.volcengine.com/docs/82379/1298454#%E8%AF%B7%E6%B1%82%E4%BD%93)</li><li>MiniMax: [ChatCompletion v2 - Supported Models and Token Count](http://platform.minimaxi.com/document/ChatCompletion%20v2?key=66701d281d57f38758d581d0#QklxsNSbaf6kM4j6wjO5eEek)</li><li>StepFun: [Model List - Text Large Model](https://platform.stepfun.com/docs/llm/text)</li><li>Tongyi Qianwen: [Model List - Tongyi Qianwen-Plus](https://help.aliyun.com/zh/model-studio/getting-started/models?spm=a2c4g.11186623.help-menu-2400256.d_0_2.4b75453a44gPA0#6c45e49509gtr)</li></ul>                                           |
| IgnoreBracketText    | Array of Number  | No       | Remove unnecessary content from the LLM return results. Supported values and meanings are as follows:<ul><li>1: emoji expressions.</li></ul>                      |
