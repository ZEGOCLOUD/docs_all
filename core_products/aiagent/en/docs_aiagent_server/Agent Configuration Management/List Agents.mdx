# List Agents

---

## Description

By calling this interface, you can obtain a list of available agents.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=ListAgents`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following request parameter list only includes the interface request parameters and some common parameters. For the complete list of common parameters, please refer to [Calling Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter          | Type   | Required | Description                                                                 |
|-------------------|--------|----------|-----------------------------------------------------------------------------|
| Limit             | Int    | No       | The number of intelligent agent information entries to retrieve in one go. The value range is [0, 10]. If not filled or the value is 0, it defaults to 5. If the value is less than 0 or greater than 10, the API call will return an error. |
| Cursor            | String | No       | Pagination retrieval marker. Leave blank for the first request, then fill in the Cursor value returned from the previous response. When the returned Cursor is empty, it indicates that the list of intelligent agents has been fully retrieved. |

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=ListAgents
    &<Common Request Parameters>
    ```

- Request Body:
    ```json
    {
        "Limit": 10,
        "Cursor": ""
    }
## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | Returned data. |
| └Total | Number | Total number of agents that meet the query criteria. |
| └Agents | Array of Object | Agent list. The structure of this parameter is described in [Common Parameter Description - Agents](./../Common%20Parameter%20Description.mdx#agents). | 
| └Cursor | String | A non-empty value only indicates that there are still agent details not returned, and you need to set this field in the request parameter `Cursor` to fetch more agent information; an empty value indicates that all agent information has been returned. |
