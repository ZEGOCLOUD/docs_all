# Register Agent

---

## Description

By passing basic configurations to this interface, you can register an intelligent agent (Agent) for creating agent instances.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=RegisterAgent`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following list of request parameters includes only the interface-specific parameters and some common parameters. For the complete list of common parameters, please refer to [Invocation Methods - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter               | Type   | Required | Description             |
|------------------|------|------|-------------------------|
| AgentId          | String | Yes    | Only supports numbers, English characters, and the following special characters: `!#$%&()+-:;\<=.>?@[]^_\{}\|~,`. The maximum length is 128 bytes.<Note title="Note">Please ensure that AgentId is unique under the same AppID.</Note>                                                              |
| Name            | String | No       | Intelligent agent name, the length does not exceed 64 bytes.                                                                 |
| LLM             | Object | Yes       | Large language model parameters. For this parameter structure, see [Common Parameter Description - LLM](./../Common%20Parameter%20Description.mdx#llm).                                                            |
| TTS             | Object | Yes       | Text-to-speech parameters. For this parameter structure, see [Common Parameter Description - TTS](./../Common%20Parameter%20Description.mdx#tts).                                                              |
| ASR             | Object | No       | Speech recognition parameters. For this parameter structure, see [Common Parameter Description - ASR](./../Common%20Parameter%20Description.mdx#asr).                                                              |

## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |
