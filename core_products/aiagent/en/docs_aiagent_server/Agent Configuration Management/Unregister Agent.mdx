# Unregister Agent

---

## Description

By passing the agent ID to this interface, you can deregister an agent.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=UnregisterAgent`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following list of request parameters includes only the interface-specific parameters and some common parameters. For the complete list of common parameters, please refer to [Invocation Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter          | Type    | Required | Description                                                                 |
|--------------------|---------|----------|-----------------------------------------------------------------------------|
| AgentId           | String  | Yes      | The unique identifier of a registered agent.                                  |

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=UnregisterAgent
    &<Common Request Parameters>
    ```

- Request Body:
    ```json
    {
        "AgentId": "zy-123"
    }
    ```

## Response Parameters

| Parameter | Type   | Description                                                                 |
|-----------|--------|-----------------------------------------------------------------------------|
| Code      | Number | Return code. 0 indicates success, other values indicate failure. For more information about error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message   | String | Explanation of the request result.                                           |
| RequestId | String | Request ID.                                                                 |
| Data      | Object | The result of this request is `null`.                                       |
