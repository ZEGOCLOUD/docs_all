# Modify An Agent

---

## Description

By passing the intelligent agent ID and basic configuration to this interface, you can update an existing intelligent agent.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgent`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following list of request parameters includes only the interface-specific parameters and some common parameters. For the complete list of common parameters, please refer to [Accessing Server APIs - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).
| Parameter         | Type   | Required | Description                                                                 |
|------------------|--------|----------|-----------------------------------------------------------------------------|
| AgentId          | String | Yes      | The unique identifier of a registered agent.                                 |
| Name             | String | No       | The name of the agent, which must not exceed 64 bytes in length.            |
| LLM              | Object | No       | Parameters for large language models. For the structure of this parameter, see [Description of Common Parameters - LLM](./../Common%20Parameter%20Description.mdx#llm). |
| TTS              | Object | No       | Text-to-speech parameters. For the structure of this parameter, see [Description of Common Parameters - TTS](./../Common%20Parameter%20Description.mdx#tts). |
| ASR              | Object | No       | Speech recognition parameters. For the structure of this parameter, see [Description of Common Parameters - ASR](./../Common%20Parameter%20Description.mdx#asr). |

<Note title="Note">
Only the parameters passed into this interface will take effect; parameters not passed in will not be updated.
</Note>

## Sample Request

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgent
    &<Common request parameters>
    ```

- Request message body:
    ```json
    {
        "AgentId": "xiaozhi",
        "LLM": {
            "Url": "https://api.minimax.chat/v2/text/chatcompletion_v2",
            "ApiKey": "your_api_key",
            "Model": "abab6.5s-chat",
            "SystemPrompt": "You are Xiaozhi, an adult woman, a companion assistant **created by ZEGO**. knowledgeable in both astronomy and geography, intelligent, wise, enthusiastic, and friendly. \nDialogue requirements: 1. Dialogue with users according to the requirements of the persona. \n2.No more than 100 words."
        },
        "TTS": {
            "Vendor": "Bytedance",
            "Params": {
                "app": {
                    "appid": "your_appid",
                    "token": "your_token",
                    "cluster": "volcano_tts"
                },
                "audio": {
                    "voice_type": "your_voice_type",
                    "emotion": "your_emotion",
                    "enable_emotion": true,
                    "speed_ratio": 1
                }
            }
        },
        "ASR": {}
    }
    ```

## Response Parameters

| Parameter | Type | Description |
|----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |
