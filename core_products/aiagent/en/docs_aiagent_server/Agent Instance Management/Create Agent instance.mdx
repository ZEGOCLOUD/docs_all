# Create Agent Instance

---

## Description

By sending a request to this interface, you can create an agent instance and add the agent instance into a voice (RTC) conversation.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=CreateAgentInstance`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following request parameter list only includes the interface request parameters and some common parameters. For the complete list of common parameters, please refer to [Invocation Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).
| Parameter            | Type    | Required | Description                                                         |
|---------------------|---------|----------|---------------------------------------------------------------------|
| AgentId             | String  | Yes      | The unique identifier of a registered agent. |
| UserId              | String  | Yes      | The real user ID used for logging into the RTC room. Only supports numbers, English characters, '-', '_', and must not exceed 32 bytes in length.                   |
| RTC                 | Object  | Yes      | RTC related information. Please refer to [RTC](#rtc) for this parameter structure.                                           |
| LLM                 | Object  | No       | Large language model parameters. Please see [Common Parameter Description - LLM](./../Common%20Parameter%20Description.mdx#llm).                                               |
| TTS                 | Object  | No       | Text-to-speech parameters. Please see [Common Parameter Description - TTS](./../Common%20Parameter%20Description.mdx#tts).                                               |
| ASR                 | Object  | No       | Speech recognition parameters. Please see [Common Parameter Description - ASR](./../Common%20Parameter%20Description.mdx#asr).                                               |
| MessageHistory      | Object  | No       | Historical messages for use by the agent instance, up to 100 entries. Please refer to [MessageHistory](#messagehistory) for this parameter structure.                           |
| CallbackConfig      | Object  | No       | Server callback configuration parameters. Please refer to [CallbackConfig](#callbackconfig) for this parameter structure.                           |
### RTC
<Note title="Note">

The explanation for RoomId, AgentStreamId, AgentUserId, and UserStreamId is as follows:
- Character restrictions: Only numbers, English letters, '-', and '_' are supported.
- Length restrictions:
    - RoomId, AgentStreamId, UserStreamId: 128 bytes.
    - AgentUserId: 32 bytes.
</Note>

| Parameter      | Type    | Required | Description                                                                 |
|----------------|---------|----------|-----------------------------------------------------------------------------|
| RoomId         | String  | Yes      | RTC room ID.                                                               |
| AgentStreamId  | String  | Yes      | Stream ID used by the intelligent entity instance for pushing streams.<Note title="Note">Ensure that multiple intelligent entity instances running currently (even if not in the same RTC room) use different stream IDs; otherwise, it will cause the push stream of the later created intelligent entity instance to fail.</Note>                                                |
| AgentUserId    | String  | Yes      | User ID of the intelligent entity instance.<Note title="Note">Ensure that multiple intelligent entity instances running simultaneously (even if not in the same RTC room) use different user IDs; otherwise, the earlier created intelligent entity instance will be kicked out of the RTC room.</Note>                                              |
| UserStreamId   | String  | Yes      | Stream ID used by the real user for pushing streams.                                   |

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=CreateAgentInstance
    &<Common Request Parameters>
    ```

- Request Body:
    ```json
    {
        "AgentId": "xiaozhi",
        "UserId": "user_1",
        "RtcInfo": {
            "RoomId": "room_1",
            "AgentStreamId": "agent_stream_1",
            "AgentUserId": "agent_user_1",
            "UserStreamId": "user_stream_1"
        }
    }
    ```
