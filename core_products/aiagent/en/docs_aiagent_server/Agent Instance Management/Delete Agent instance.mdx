# Delete An Agent Instance

---

## Description

By calling this interface, you can delete an existing agent instance.

## Request method

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=DeleteAgentInstance`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters
The following request parameter list only includes the interface request parameters and some common parameters. For the complete list of common parameters, please refer to [Calling Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters) .

| Parameter             | Type    | Required | Description         |
|----------------------|---------|----------|---------------------|
| AgentInstanceId      | String  | Yes      | The unique identifier of the agent instance, obtained via the response parameter of the [Create Agent Instance](./Create%20Agent%20instance.mdx) interface. |

## Sample Request

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=DeleteAgentInstance
    &<Common Request Parameters>
    ```

- Request Message Body:
    ```json
    {
        "AgentInstanceId": "1909811657303920640"
    }
    ```

## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |
