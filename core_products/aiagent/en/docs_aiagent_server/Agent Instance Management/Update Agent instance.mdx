# Update Agent Instance

---

## Description

By accessing this interface, you can update an existing intelligent agent instance.

## Interface Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgentInstance`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following list of request parameters includes only the interface-specific parameters and some common parameters. For the complete list of common parameters, please refer to [Invocation Methods - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#common-parameters).

| Parameter         | Type     | Required | Description                                                                 |
|-------------------|----------|----------|-----------------------------------------------------------------------------|
| AgentInstanceId  | String   | Yes      | The unique identifier of the agent instance, obtained through the response parameter of the [Create Agent Instance](./Create%20Agent%20instance.mdx) interface. |
| LLM              | Object   | No       | Large language model parameters. For this parameter structure, see [Common Parameter Description - LLM](./../Common%20Parameter%20Description.mdx#llm). |
| TTS              | Object   | No       | Text-to-speech parameters. For this parameter structure, see [Common Parameter Description - TTS](./../Common%20Parameter%20Description.mdx#tts). |
| ASR              | Object   | No       | Speech recognition parameters. For this parameter structure, see [Common Parameter Description - ASR](./../Common%20Parameter%20Description.mdx#asr). |

<Note title="Note">
Only the parameters passed into this interface will take effect, and the parameters not passed in will not be updated.
</Note>

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=UpdateAgentInstance
    &<Common Request Parameters>
    ```

- Request Body:

    ```json
    {
        "AgentInstanceId": "1912124734317838336",
        "LLM": {
            "Url": "https://api.minimax.chat/v2/text/chatcompletion_v2",
            "ApiKey": "your_api_key",
            "Model": "abab6.5s-chat",
            "SystemPrompt": "You are Xiaozhi, an adult woman, a companion assistant **created by ZEGO**. knowledgeable in both astronomy and geography, intelligent, wise, enthusiastic, and friendly. \nDialogue requirements: 1. Dialogue with users according to the requirements of the persona. \n2.No more than 100 words."
        }
    }
    ```

## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |
