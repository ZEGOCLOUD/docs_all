# API 概览

---

ZEGO 实时互动 AI Agent 服务端提供以下相关 API 接口及回调，可参考 [调用方式](./Accessing%20Server%20APIs.mdx) 调用以下接口。

## 智能体配置管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [注册智能体](./Agent%20Configuration%20Management/Register%20Agent.mdx)                |注册一个智能体（Agent），以用于创建智能体实例。       | 10 次/秒         |
| [注销智能体](./Agent%20Configuration%20Management/Unregister%20Agent.mdx)                | 注销一个智能体。           | 10 次/秒         |
| [修改智能体](./Agent%20Configuration%20Management/Update%20Agent.mdx)                | 修改一个智能体。          | 10 次/秒         |
| [获取智能体列表](./Agent%20Configuration%20Management/List%20Agents.mdx)     | 获取可用的智能体列表。       | 10 次/秒         |
| [获取智能体详情](./Agent%20Configuration%20Management/Query%20Agents.mdx)     | 获取指定智能体的详情。       | 10 次/秒         |

## 智能体实例管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [创建智能体实例](./Agent%20Instance%20Management/Create%20Agent%20instance.mdx) | 创建智能体实例，并将智能体实例加入到文本对话和语音（RTC）对话之中。 | 10 次/秒         |
| [修改智能体实例](./Agent%20Instance%20Management/Update%20Agent%20instance.mdx)       | 更新一个现有的智能体实例。 | 10 次/秒         |
| [删除智能体实例](./Agent%20Instance%20Management/Delete%20Agent%20instance.mdx)       | 删除一个现有的智能体实例。 | 10 次/秒         |

## 智能体实例控制

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [主动调用 LLM](./Agent%20Instance%20Control/Send%20Agent%20instance%20LLM.mdx)               | 以用户身份主动调用 LLM 服务，并基于 LLM 的响应内容，以智能体身份主动调用 TTS 服务，向用户发送语音消息。      | 10 次/秒         |
| [主动调用 TTS](./Agent%20Instance%20Control/Send%20Agent%20instance%20TTS.mdx)               | 以智能体身份主动调用 TTS 服务，向用户发送语音消息。     | 10 次/秒         |
| [查询智能体实例状态](./Agent%20Instance%20Control/Query%20Agent%20instance%20status.mdx)               | 在语音对话过程中查询智能体的运行状态信息。     | 10 次/秒         |

## 回调

| 回调名称 | 回调说明 |
| -------- | -------- |
| [通用回调](./Callbacks/General%20Callback.mdx)         |   凭借此回调，监听用户与智能体对话过程中所发生的事件。       |
