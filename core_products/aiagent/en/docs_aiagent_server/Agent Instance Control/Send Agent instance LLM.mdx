# Send Agent Instance LLM

---

## Description

This API can be used to actively invoke the LLM service on behalf of a user. Based on the response content from the LLM, it can also invoke the TTS service as an intelligent agent, sending voice messages to the user.
- When this interface is called, if the intelligent agent in the session is currently speaking, its speech process will be interrupted;
- The output content of this interface call will not be recorded in the history messages, will not affect the context of the next LLM invocation, and will not be issued via RTC room messages.

## API Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=SendAgentInstanceLLM`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second

## Request Parameters

The following request parameter list only includes the interface request parameters and some common parameters. For the complete list of common parameters, please refer to [Calling Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#Common%20Request%20Parameters).

| Parameter             | Type    | Required | Description                                                         |
|----------------------|---------|----------|---------------------------------------------------------------------|
| AgentInstanceId      | String  | Yes      | The unique identifier of the agent instance, obtained through the response parameter of the [Create Agent Instance](./Create%20Agent%20instance.mdx) interface. |
| Text                | String  | Yes      | The text content sent to the LLM service.                           |
| SystemPrompt        | String  | No       | Whether to temporarily modify the system prompt (prompt) for this conversation. If needed, fill in this field. If left blank, the system for this round of conversation will not be modified. |

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=SendAgentInstanceLLM
    &<Common Request Parameters>
    ```

- Request Message Body:
    ```json
    {
        "AgentInstanceId": "1907755175297171456",
        "Text": "How the weather like today?"
    }
    ```

## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Int | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |

## Response Example

```json
{
    "Code": 0,
    "Message": "Succeed",
    "RequestId": "1843985617336143872",
    "Data": null
}
```
