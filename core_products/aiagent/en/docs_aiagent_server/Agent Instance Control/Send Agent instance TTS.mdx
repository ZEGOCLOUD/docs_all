# Send Agent Instance TTS

---

## Description

This API can be used to actively call the TTS service, allowing an agent to send voice messages to users.
- When this API is called, if the agent in the session is currently speaking, its speech process will be interrupted;
- The information generated by calling this API will be recorded in the session message history, but it will not be provided as context input to the LLM.

## API Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=SendAgentInstanceTTS`
- Transfer Protocol: HTTPS
- Call Frequency Limit: 10 times/second
## Request Parameters

The following request parameter list only includes the interface request parameters and some common parameters. For the complete list of common parameters, please refer to [Calling Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#Common%20Request%20Parameters).

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| AgentInstanceId | String | Yes | The unique identifier of the agent instance, obtained through the response parameter of the [Create Agent Instance](./Create%20Agent%20instance.mdx) interface. |
| Text | String | Yes | The text content for TTS, with a maximum length not exceeding 300 characters. |

## Request Example


- Request URL:
    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=SendAgentInstanceTTS
    &<Common request parameters>
    ```

- Request body:

    ```json
    {
        "AgentInstanceId": "1907780504753553408",
        "Text": "Dear developer, hello. Welcome to use ZEGOCLOUDRTC to jointly build a real-time interactive world."
    }
    ```

## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Int | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | The result of this request is `null`. |
## Response Example

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1843985617336143872",
    "Data": null
}
```
