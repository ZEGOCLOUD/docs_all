# Query Agent Instance Status

---

## Description

This interface is used to query the runtime status information of an intelligent agent during voice conversations.

## API Prototype

- Request Method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=QueryAgentInstanceStatus`
- Transfer Protocol: HTTPS
- Invocation Frequency Limit: 10 times/second
## Request Parameters

The following request parameter list includes only the interface-specific parameters and some common parameters. For the complete list of common parameters, please refer to [Calling Method - Common Request Parameters](./../Accessing%20Server%20APIs.mdx#公共请求参数) (in Chinese, referring to "Common Request Parameters").

| Parameter         | Type     | Required | Description               |
|-------------------|----------|----------|---------------------------|
| AgentInstanceId   | String   | Yes      | Unique identifier for the session. |

## Request Example

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=QueryAgentInstanceStatus
    &<Common Request Parameters>
    ```

- Request Message Body:

    ```json
    {
        "AgentInstanceId": "1907616178474336256"
    }
    ```
## Response Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
| Data | Object | Returned data. |
| └Status | String | Intelligent agent running status:<ul><li>`IDLE`: Idle</li><li>`LISTENING`: Listening</li><li>`THINKING`: Thinking</li><li>`SPEAKING`: Speaking</li></ul> |

## Response Example

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1898302332764950528",
    "Data": {
        "Status": "IDLE"
    }
}
```
