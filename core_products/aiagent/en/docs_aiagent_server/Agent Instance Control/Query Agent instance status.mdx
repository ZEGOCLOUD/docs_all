# 查询智能体实例状态

---

## 描述

本接口用于在语音对话过程中查询智能体的运行状态信息。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-aiagent-api.zegotech.cn?Action=QueryAgentInstanceStatus`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数             | 类型    | 是否必选 | 描述                     |
|------------------|---------|----------|--------------------------|
| AgentInstanceId  | String  | 是       | 会话的唯一标识。         |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=QueryAgentInstanceStatus
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "AgentInstanceId": "1907616178474336256"
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 返回数据。 |
| └Status | String | 智能体运行状态：<ul><li>`IDLE`：空闲</li><li>`LISTENING`：聆听中</li><li>`THINKING`：思考中</li><li>`SPEAKING`：讲话中</li></ul> |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1898302332764950528",
    "Data": {
        "Status": "IDLE"
    }
}
```
