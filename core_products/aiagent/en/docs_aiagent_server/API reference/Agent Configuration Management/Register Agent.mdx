# Register An AI Agent

---

## Description

By passing basic configurations to this API, you can register an AI agent (Agent) for creating agent instances.

## Request method and endpoint

- Request method: POST
- Request URL: `https://aigc-aiagent-api.zegotech.cn?Action=RegisterAgent`
- Transmission protocol: HTTPS
- QPS limit: 10 times/second

## Request parameters

Listed below are the parameters specific to this request and part of the common request parameters. For the complete list of common request parameters, see [Accessing Server APIs - Common request parameters](./../Accessing%20Server%20APIs.mdx#common-request-parameters).

| Parameter               | Type   | Required | Description             |
|------------------|------|------|-------------------------|
| AgentId          | String | Yes    | Only supports numbers, English characters, and the following special characters: `!#$%&()+-:;\<=.>?@[]^_\{}\|~,`. The maximum length is 128 bytes.<Note title="Note">Please ensure that AgentId is unique under the same AppID.</Note>                                                              |
| Name            | String | No       | AI agent name, the length does not exceed 256 bytes.                                                                 |
| LLM             | Object | Yes       | Large language model parameters. For details, see [Frequently Used Parameters - LLM](./../Common%20Parameter%20Description.mdx#llm).                                                            |
| TTS             | Object | Yes       | Text-to-speech parameters. For details, see [Frequently Used Parameters - TTS](./../Common%20Parameter%20Description.mdx#tts).                                                              |
| ASR             | Object | No       | Automatic speech recognition parameters. For details, see [Frequently Used Parameters - ASR](./../Common%20Parameter%20Description.mdx#asr).                          |

## Sample request

- Request URL:

    ```https
    https://aigc-aiagent-api.zegotech.cn?Action=RegisterAgent
    &<Common request parameters>
    ```

- Request body:
    ```json
    {
        "AgentId": "xiaozhi",
        "Name": "Xiao Zhi",
        "LLM": {
            "Url": "https://ark.cn-beijing.volces.com/api/v3/chat/completions",
            "ApiKey": "eexxxxxxxxxxxxxxx",
            "Model": "ep-xxxxxxxxxx",
            "SystemPrompt": "You are Xiaozhi, an adult woman, a companion assistant **created by ZEGOCLOUD**. Knowledgeable in everything, intelligent, wise, enthusiastic, and friendly. \nDialogue requirements: 1. Dialogue with users according to the requirements of the persona. \n2.No more than 100 words."
        },
        "TTS": {
            "Vendor": "Bytedance",
            "Params": {
                "app": {
                    "appid": "your_appid",
                    "token": "your_token",
                    "cluster": "volcano_tts"
                },
                "audio": {
                    "voice_type": "your_voice_type"
                }
            }
        },
        "ASR": {
            "HotWord": "ZEGOCLOUD |10"
        }
    }
    ```

## Response parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Code | Number | Return code. 0 indicates success, other values indicate failure. For more information on error codes and response handling recommendations, please refer to [Return Codes](./../Return%20codes.mdx). |
| Message | String | Explanation of the request result. |
| RequestId | String | Request ID. |
