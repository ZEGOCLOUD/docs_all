## Detailed Service Pricing

### AI Agent Processing Fees

When invoking the AI Agent capability to create an intelligent agent instance for real-time voice processing, there will be basic platform service fees incurred, including costs related to computing power and outbound bandwidth. When the input and output of the agent are only voice-based, charges will apply according to the voice specifications.

| Service Item | Service Item Type | Billing Mode | Tier | Billing Price | Notes |
|-------------|-------------------|--------------|------|---------------|-------|
| AI Agent Audio Processing Fee | Basic Service | **Real-time Voice Dialogue Duration for Intelligent Agents (Yuan per thousand minutes)**<br />Billing starts from "Create Intelligent Agent Instance" and ends at "Delete Intelligent Agent Instance". | No distinction | 9 | - |

### LLM (Large Language Model)

ZEGO AI Agent supports you in connecting with any LLM service compatible with the [OpenAI Chat Completions API](https://platform.openai.com/docs/api-reference/chat). You can purchase relevant services directly from each service provider's official website.

For LLM services such as [Volcano Ark (DouBao)](https://www.volcengine.com/docs/82379/1099320), [Tongyi Qianwen](https://help.aliyun.com/zh/model-studio/models?spm=a2c4g.11186623.help-menu-2400256.d_0_2.7f9b1d1czN51sr&scm=20140722.H_2840914._.OR_help-T_cn~zh-V_1), [MiniMax](https://platform.minimaxi.com/document/Price?key=66701c7e1d57f38758d5818c), and [StepFun](https://platform.stepfun.com/docs/pricing/details), you can also purchase them through ZEGO. For more details, please consult ZEGO’s business personnel.
