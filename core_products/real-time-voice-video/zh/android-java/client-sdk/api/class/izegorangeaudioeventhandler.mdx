**参数**

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| rangeAudio | [ZegoRangeAudio](../../api-reference/class#zegorangeaudio) | 触发此次回调的范围语音实例。 |
| state | [ZegoRangeAudioMicrophoneState](../../api-reference/enum#zegorangeaudiomicrophonestate) | 麦克风使用状态。 |

**详情**

麦克风的状态变化通知，开启发送音频是异步过程，中间的状态切换都通过该函数回调。

<Note title="">
- **通知时机**：在 [enableMicrophone] 函数后。
</Note>

<Warning title="">
- **支持版本**：2.11.0 及以上。
- **注意事项**：1. 必须在 [enableMicrophone] 函数调用前监听。 2. 正常停止 [enableMicrophone] 时，是不会触发该回调的，只有异常停推的时候才会触发 Off 的状态。
</Warning>
</ParamField>
