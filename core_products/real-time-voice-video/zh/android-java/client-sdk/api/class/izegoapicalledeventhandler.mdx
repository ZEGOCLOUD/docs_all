**详情**

通过 [setApiCalledCallback] 开启监听时，会通过本回调回调所有方法的执行的结果。

<Note title="">
- **通知时机**：在开发者调用 SDK 方法时，回调该方法的执行结果。
</Note>

<Warning title="">
- **支持版本**：2.3.0 及以上。
- **使用限制**：无。
- **注意事项**：建议在开发、测试阶段监听并处理本回调，在上线后关闭对本回调的监听。
</Warning>
</ParamField>
