# 房间登录回调

当 SDK 登录房间时，会触发 `room_login` 回调。

## 回调描述

该回调在用户成功登录房间时触发。开发者可以使用此回调获取用户登录状态信息并进行后续操作。

## 回调触发时机

在用户成功登录房间后触发。

## API 结构

```javascript
callback = (response) => {
    // 回调逻辑
}
```

## 注意事项

- 回调在用户成功登录房间后触发。
- `custom_info` 字段是登录时传递的自定义信息，可用于携带用户特定信息。
- `term_type` 字段表示用户使用的终端类型，可用于实现终端特定逻辑。

## 代码示例

```javascript
// 房间登录回调
ZegoUIKit.prebuilt.createKitContext().on('room_login', (response) => {
    console.log('房间登录回调:', response);
    
    if (response.code === 200) {
        console.log('用户登录成功:', response.uid);
        console.log('房间 ID:', response.room_id);
        console.log('终端类型:', response.term_type);
    } else {
        console.error('房间登录失败:', response.message);
    }
});
```
