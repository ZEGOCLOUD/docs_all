# ZegoVideoRateControlMode

## ZegoVideoRateControlMode

视频码率控制模式，默认恒定码率

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoVideoRateControlModeConstantRate"
  prototype="ZegoVideoRateControlModeConstantRate=(0)"
  parent_name="ZegoVideoRateControlMode"
  parent_type="enum">
恒定码率。
</ParamField>
<ParamField
  name="ZegoVideoRateControlModeConstantQuality"
  prototype="ZegoVideoRateControlModeConstantQuality=(1)"
  parent_name="ZegoVideoRateControlMode"
  parent_type="enum">
恒定质量，表示保证视频的清晰度在固定水平上，因此若采用此控制模式，码率会根据网速的变化波动。比如游戏类直播时，为了让观众看到比较流畅的操作类画面会使用恒定质量模式，提升视频质量。
</ParamField>
