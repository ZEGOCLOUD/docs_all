# ZegoRoomState

## ZegoRoomState

房间状态

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoRoomStateDisconnected"
  prototype="ZegoRoomStateDisconnected=(0)"
  parent_name="ZegoRoomState"
  parent_type="enum">
未连接状态，在登录房间前和退出房间之后进入该状态。如果登录房间的过程出现稳态异常，例如 AppID 或 Token 不正确，或者有相同用户名在其他地方登录导致本端被 KickOut，都会进入该状态
</ParamField>
<ParamField
  name="ZegoRoomStateConnecting"
  prototype="ZegoRoomStateConnecting=(1)"
  parent_name="ZegoRoomState"
  parent_type="enum">
正在请求连接状态，登录房间动作执行成功后会进入此状态。通常通过该状态进行应用界面的展示。如果因为网络质量不佳产生的中断， SDK 会进行内部重试，也会回到正在请求连接状态
</ParamField>
<ParamField
  name="ZegoRoomStateConnected"
  prototype="ZegoRoomStateConnected=(2)"
  parent_name="ZegoRoomState"
  parent_type="enum">
连接成功状态，进入该状态表示登录房间已经成功，用户可以正常收到房间内的用户和流信息增删的回调通知
</ParamField>
