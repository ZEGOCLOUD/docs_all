# ZegoAudioSourceType

## ZegoAudioSourceType

音频采集源类型。

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoAudioSourceTypeDefault"
  prototype="ZegoAudioSourceTypeDefault=(0)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
默认音频采集源（主通道默认使用自定义音频采集；辅通道默认使用与主通道同样的声音）。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeCustom"
  prototype="ZegoAudioSourceTypeCustom=(1)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
使用自定义音频采集，参考 [enableCustomAudioIO] 或 [setAudioSource]。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeMediaPlayer"
  prototype="ZegoAudioSourceTypeMediaPlayer=(2)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
使用媒体播放器作为音频采集源，仅支持辅通道。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeNone"
  prototype="ZegoAudioSourceTypeNone=(3)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
不启动任何采集。仅支持在 [setAudioSource] 接口中使用，在 [enableCustomAudioIO] 接口中使用无效。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeMicrophone"
  prototype="ZegoAudioSourceTypeMicrophone=(4)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
使用麦克风作为音频采集源。仅支持在 [setAudioSource] 接口中使用，在 [enableCustomAudioIO] 接口中使用无效。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeMainPublishChannel"
  prototype="ZegoAudioSourceTypeMainPublishChannel=(5)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
使用主通道作为音频输入源。主通道使用时无效。仅支持在 [setAudioSource] 接口中使用，在 [enableCustomAudioIO] 接口中使用无效。
</ParamField>
<ParamField
  name="ZegoAudioSourceTypeScreenCapture"
  prototype="ZegoAudioSourceTypeScreenCapture=(6)"
  parent_name="ZegoAudioSourceType"
  parent_type="enum">
使用屏幕采集声音作为音频输入源。通常用于移动端屏幕共享场景下。同时仅支持在 [setAudioSource] 接口中使用，在 [enableCustomAudioIO] 接口中使用无效。
</ParamField>
