# ZegoMixerInput

## ZegoMixerInput

混流输入

**详情**

配置混流输入的流 ID、输入类型、流的布局

*Declared in `ZegoExpressDefines.h`*

  name="streamID"

  parent_name="ZegoMixerInput"
  parent_type="class">
<Warning title="">
- **注意事项**：不可以包含 URL 关键字，否则推拉流失败。仅支持数字，英文字符 和 '-', '_'。
</Warning>
</ParamField>

</ParamField>
