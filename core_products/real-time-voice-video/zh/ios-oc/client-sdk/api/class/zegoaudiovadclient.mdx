**详情**

检测音频包是否包含语音。

<Note title="">
- **调用时机**：在初始化检测音频包类 [createAudioVADClient] 之后。
</Note>

<Warning title="">
- **支持版本**：2.21.0 及以上。
- **注意事项**：语音数据异常时也会返回ZegoAudioVADTypeSpeech, 保证不会漏掉语音。
</Warning>

**返回值**

语音检测结果。
