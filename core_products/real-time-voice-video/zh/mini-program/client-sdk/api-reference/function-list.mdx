






# 功能总览



- - -



ZEGO SDK 拥有全面覆盖全球重点区域的网络节点，提供丰富的 API，为多端提供清晰稳定的实时语音视频云服务。

以下为基本 API 的简要介绍。





## 初始化相关的前置功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [destroyEngine](./class.mdx#destroyengine-zegoexpressengine-class) | 销毁引擎实例。
[checkSystemRequirements](./class.mdx#checksystemrequirements-zegoexpressengine-class) | 支持能力检测接口
[off](./class.mdx#off-zegoexpressengine-class) | 删除注册过的回调事件
[on](./class.mdx#on-zegoexpressengine-class) | 注册回调事件
[enableMultiRoom](./class.mdx#enablemultiroom-zegoexpressengine-class) | 是否开启多房间模式
[setLogConfig](./class.mdx#setlogconfig-zegoexpressengine-class) | 日志高级配置
[setDebugVerbose](./class.mdx#setdebugverbose-zegoexpressengine-class) | 是否开启Debug模式，打开时错误信息 alert 提示，默认测试环境开启
[getVersion](./class.mdx#getversion-zegoexpressengine-class) | 获取当前SDK版本 | 

## 房间功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [loginRoom](./class.mdx#loginroom-zegoexpressengine-class) | 登录房间
[logoutRoom](./class.mdx#logoutroom-zegoexpressengine-class) | 退出房间，不再接受各种房间内状态
[renewToken](./class.mdx#renewtoken-zegoexpressengine-class) | 更新房间权限token
[setRoomExtraInfo](./class.mdx#setroomextrainfo-zegoexpressengine-class) | 设置房间附加消息 | 

## 推流功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [startPublishingStream](./class.mdx#startpublishingstream-zegoexpressengine-class) | 开始推流
[stopPublishingStream](./class.mdx#stoppublishingstream-zegoexpressengine-class) | 停止将本地流推送到远端（即构服务器）
[setStreamExtraInfo](./class.mdx#setstreamextrainfo-zegoexpressengine-class) | 设置流的附加信息
[addPublishCdnUrl](./class.mdx#addpublishcdnurl-zegoexpressengine-class) | 通知即构服务器将流转推到CDN
[removePublishCdnUrl](./class.mdx#removepublishcdnurl-zegoexpressengine-class) | 通知即构服务器停止将流转推到 CDN
[updateAudioVolumeNotify](./class.mdx#updateaudiovolumenotify-zegoexpressengine-class) | 更新流音量 | 

## 拉流功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [startPlayingStream](./class.mdx#startplayingstream-zegoexpressengine-class) | 开始拉流
[stopPlayingStream](./class.mdx#stopplayingstream-zegoexpressengine-class) | 停止拉取远端流（即构服务器） | 

## 信令消息功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [sendBarrageMessage](./class.mdx#sendbarragemessage-zegoexpressengine-class) | 发送房间弹幕消息（消息不保证可靠）
[sendBroadcastMessage](./class.mdx#sendbroadcastmessage-zegoexpressengine-class) | 发送房间广播消息（消息保证可靠）
[sendCustomCommand](./class.mdx#sendcustomcommand-zegoexpressengine-class) | 发送自定义信令（消息可靠）
[sendTransparentMessage](./class.mdx#sendtransparentmessage-zegoexpressengine-class) | 发送透传消息 | 

## 混流功能

 | 方法名 | 描述 | 
 | --- | --- | 
 | [startAutoMixerTask](./class.mdx#startautomixertask-zegoexpressengine-class) | 开始自动混流任务
[startMixerTask](./class.mdx#startmixertask-zegoexpressengine-class) | 开始混流任务
[stopAutoMixerTask](./class.mdx#stopautomixertask-zegoexpressengine-class) | 停止服务端自动混流任务
[stopMixerTask](./class.mdx#stopmixertask-zegoexpressengine-class) | 停止服务端混流
[setMixerTaskConfig](./class.mdx#setmixertaskconfig-zegoexpressengine-class) | 混流高级配置 |
