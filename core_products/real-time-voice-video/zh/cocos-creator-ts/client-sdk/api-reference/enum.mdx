# Enum

## ZegoAECMode

回声消除模式。

### 属性
<ParamField
  name="ZegoAECModeAggressive"
  prototype="ZegoAECModeAggressive=(0)"
  parent_name="ZegoAECMode"
  parent_type="enum">
激进的回声抵消，可能会影响音质稍微明显，但是回声会消除得很干净。
</ParamField>
<ParamField
  name="ZegoAECModeMedium"
  prototype="ZegoAECModeMedium=(1)"
  parent_name="ZegoAECMode"
  parent_type="enum">
适度的回声抵消，就是可能会稍微影响一点点音质，但是残留的回声会更少。
</ParamField>
<ParamField
  name="ZegoAECModeSoft"
  prototype="ZegoAECModeSoft=(2)"
  parent_name="ZegoAECMode"
  parent_type="enum">
舒适的回声抵消，就是回声抵消基本不会影响声音的音质，可能有时会残留一点回声，但不会影响正常听音。
</ParamField>
<ParamField
  name="ZegoAECModeAI"
  prototype="ZegoAECModeAI=(3)"
  parent_name="ZegoAECMode"
  parent_type="enum">
AI 模式回声抵消。支持智能识别并且消除回声，相较传统AEC算法，人声保真度效果提升明显，且没有额外的延迟与功耗增量。
</ParamField>
<ParamField
  name="ZegoAECModeAIAggressive"
  prototype="ZegoAECModeAIAggressive=(4)"
  parent_name="ZegoAECMode"
  parent_type="enum">
AI 模式强回声抵消。类似于 ZegoAECModeAI 模式，在混响显著的场景下能提供更纯净的回声消除效果，因此建议在混响较大的聊天室中使用。在其他场景下（尤其是在KTV环境中，音乐大声播放时）可以关闭该功能，因为它可能会使人的声音产生更多的失真。
</ParamField>
<ParamField
  name="ZegoAECModeAIBalanced"
  prototype="ZegoAECModeAIBalanced=(5)"
  parent_name="ZegoAECMode"
  parent_type="enum">
均衡 AI 模式回声抵消。与 ZegoAECModeAIAggressive 模式相比，回声抑制效果更纯净，但对人声的损害更大。建议在语音聊天场景中使用。
</ParamField>
