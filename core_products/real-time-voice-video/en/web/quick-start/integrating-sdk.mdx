# Integrating the SDK

---

## Prerequisites

Before starting to integrate the ZEGO Express SDK, ensure that your development environment meets the following requirements:

- A Windows or macOS development computer connected to the Internet.
- A browser that meets Express Web SDK compatibility requirements (for details, refer to [Browser Compatibility and Known Issues](/real-time-video-web/introduction/browser-restrictions)). The latest version of Google Chrome browser is recommended.

## Integrating the SDK

Developers can integrate the SDK using any of the following methods.

5. Full import and on-demand import of the SDK.

<Warning title="Warning">


    - Before version 3.4.0:
      When developers integrate the ZEGO Web SDK, audio mixing, stream mixing, and other features are integrated by default. If the developer's business only requires one of the feature modules, it cannot be integrated separately.
    - Starting from version 3.4.0:
      - Developers can still import the SDK through the main path, integrating audio mixing, stream mixing, and other features simultaneously.
      - At the same time, developers can also select from the `Audio Mix`, `Stream Mixing`, `CDN`, `Range Audio`, and `AI Effects` feature modules according to business needs, and integrate them separately through package manager installation. (**Only supports integrating individual feature modules through package manager installation; other methods are not supported.**)
        - The package size for each feature module is approximately: Audio Mix (56 KB), Stream Mixing (60 KB), CDN (29 KB), Range Audio (97 KB), AI Effects (69 KB).
        - Features that are not imported will be automatically removed during the build process, reducing the SDK package size.

</Warning>

```javascript
import {ZegoExpressEngine} from 'zego-express-engine-webrtc'
```

<Tab title="Integrate Individual Feature Modules">
Please note that when creating a [ZegoExpressEngine](@-ZegoExpressEngine) instance, developers must import a specific feature module via `import { ZegoExpressEngine } from "zego-express-engine-webrtc/esm"` to achieve on-demand integration.
Please select the features you need based on your business requirements and import them via the `zego-express-engine-webrtc/esm` path.

```javascript
import { ZegoExpressEngine } from "zego-express-engine-webrtc/esm"
// Import CDN module
import { StreamCDN } from "zego-express-engine-webrtc/esm/stream-cdn"
// Import Audio Mix module
import { AudioMix } from "zego-express-engine-webrtc/esm/audio-mix"
// Import Stream Mixing module
import { MixStream }  from "zego-express-engine-webrtc/esm/mix-stream"
// Import Range Audio module
import { RangeAudio } from 'zego-express-engine-webrtc/esm/range-audio';
// Import AI Effects module
import { BeautyEffect } from 'zego-express-engine-webrtc/esm/beauty-effect';
ZegoExpressEngine.use(StreamCDN);
ZegoExpressEngine.use(AudioMix);
ZegoExpressEngine.use(MixStream);
ZegoExpressEngine.use(RangeAudio);
ZegoExpressEngine.use(BeautyEffect);
```
If you have not integrated a feature module but call API interfaces related to that module, the SDK will return error code `1101003` and related hints. Feature modules and their corresponding API interfaces are as follows:
- **CDN Feature (StreamCDN):**
  - [addPublishCdnUrl](@addPublishCdnUrl): Notify the ZEGO server to relay the stream to CDN.
  - [removePublishCdnUrl](@removePublishCdnUrl): Notify the ZEGO server to stop relaying the stream to CDN.

- **Audio Mix Feature (AudioMix):**
  - [startMixingAudio](@startMixingAudio): Start audio mixing.
  - [stopMixingAudio](@stopMixingAudio): Stop audio mixing.
  - [setMixingAudioVolume](@setMixingAudioVolume): Set audio mixing volume.
  - [enableLiveAudioEffect](@enableLiveAudioEffect): Enable or disable audio effect enhancement.
  - [setAudioChangerParam](@setAudioChangerParam): Apply pitch transformation to the input song.

- **Stream Mixing Feature (MixStream):**
  - [startMixerTask](@startMixerTask): Start stream mixing task.
  - [stopMixerTask](@stopMixerTask): Stop server-side stream mixing.
  - [setMixerTaskConfig](@setMixerTaskConfig): Advanced stream mixing configuration.

- **Range Audio Feature (RangeAudio):**
  - [createRangeAudioInstance](@createRangeAudioInstance): Create a Range Audio instance object.
  - [setVoiceChangerParam](@setVoiceChangerParam): Apply voice changing with custom parameters to media streams for Range Audio.
  - [setVoiceChangerPreset](@setVoiceChangerPreset): Apply voice changing to media streams for Range Audio.
  - [enableAudioSourceUpdateChecker](@enableAudioSourceUpdateChecker): Whether to check for user changes within range.
  - [on](@on): Register callback events.
  - [off](@off): Unregister callback events.
  - [setAudioReceiveRange](@setAudioReceiveRange): Set the maximum range for audio reception.
  - [updateSelfPosition](@updateSelfPosition): Update the listener's position and orientation.
  - [updateAudioSource](@updateAudioSource): Add or update audio source position information.
  - [enableSpatializer](@enableSpatializer): Enable or disable 3D audio effects.
  - [enableMicrophone](@enableMicrophone): Enable or disable microphone.
  - [selectMicrophone](@selectMicrophone): Specify microphone device.
  - [selectSpeaker](@selectSpeaker): Specify speaker device.
  - [enableSpeaker](@enableSpeaker): Enable or disable speaker.
  - [setRangeAudioCustomMode](@setRangeAudioCustomMode): Set advanced custom mode for Range Audio.
  - [setRangeAudioMode](@setRangeAudioMode): Set Range Audio mode.
  - [setTeamID](@setTeamID): Set team ID.
  - [setPositionUpdateFrequency](@setPositionUpdateFrequency): Set the frequency of internal real-time position updates in the SDK.
  - [setRangeAudioVolume](@setRangeAudioVolume): Set local playback volume for Range Audio.
  - [enableAiDenoise](@enableAiDenoise): Enable or disable AI noise suppression for Range Audio.
  - [isAudioContextRunning](@isAudioContextRunning): Check if the AudioContext object is enabled.
  - [resumeAudioContext](@resumeAudioContext): Re-enable the internal AudioContext object.
  - [setStreamVocalRange](@setStreamVocalRange): Set the audio transmission range for stream playing.
  - [updateStreamPosition](@updateStreamPosition): Update the position of stream playing.
  - [setCustomSourceVocalRange](@setCustomSourceVocalRange): Set the transmission range for custom audio sources.
  - [updateCustomSourcePosition](@updateCustomSourcePosition): Add or update custom audio source position information.

- **AI Effects Feature (BeautyEffect):**
  - [setEffectsBeauty](@setEffectsBeauty): Enable or disable AI effects.
  - [setLowlightEnhancement](@setLowlightEnhancement): Set low-light enhancement.
</Tab>

### Method 2: Download SDK from Official Website and Integrate Manually

<Accordion title="Download SDK from Official Website and Integrate Manually" defaultOpen="false">

2. Open the project folder in [VSCode](https://code.visualstudio.com/) and create a new "index.html" file.

    ```html
        <meta charset="UTF-8">

      <body>
        <h1>
          Zego RTC Web Demo
        </h1>

        (async () => {
          // Add subsequent code here...
        })();
      </script>
    </html>
    ```
