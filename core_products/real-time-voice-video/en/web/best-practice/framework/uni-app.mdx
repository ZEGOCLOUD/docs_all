# Use uni-app for Browser Rendering and Playback (Web)

---

## Application Scenarios

When using ZEGO Web SDK, call the [createZegoStream](https://doc-zh.zego.im/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoExpressEngine#create-zego-stream) interface to create a [ZegoLocalStream](https://doc-zh.zego.im/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoLocalStream) instance object `localStream`, then call the [playVideo](https://doc-zh.zego.im/article/api?doc=Express_Video_SDK_API~javascript_web~class~ZegoLocalStream#play-video) interface to play the preview video.



## Usage Steps

1. Create a tag.

    ```html
    <view id="localVideo" ></view>
    ```

2. Create a stream and play the preview.

    ```javascript
    const localVideo = document.querySelector("#localVideo");

    const localStream = this.zg
      .createZegoStream({
        camera: { video: true, audio: true },
      })
      .then((stream) => {
        stream.playVideo(localVideo);
      })
      .catch((err) => {
        console.error(err);
      });
    ```
