**详情**

场景 Token 鉴权将要过期的回调通知，请用户通过 [renewToken] 函数更新场景 Token 鉴权。

<Note title="">
- **通知时机**：在 Token 过期前 30 秒，SDK 会通过 [onSceneTokenWillExpire] 回调发出通知。
- **相关接口**：当开发者收到此回调后，可通过 [renewToken] 来更新 Token 鉴权信息。
</Note>

<Warning title="">
- **支持版本**：3.1.0 及以上。
- **使用限制**：请勿在回调线程中调用 SDK 接口。
</Warning>
