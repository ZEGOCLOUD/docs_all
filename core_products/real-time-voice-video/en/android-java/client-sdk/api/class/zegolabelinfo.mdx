# ZegoLabelInfo

## ZegoLabelInfo

文本信息。

**详情**

文本信息配置，可用于配置文本内容、文本位置、文本风格。

<Note title="">
- **业务场景**：手动混流场景时，设置文字水印，比如多人连麦直播。
</Note>
