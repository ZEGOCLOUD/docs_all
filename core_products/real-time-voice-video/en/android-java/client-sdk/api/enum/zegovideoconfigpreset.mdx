# ZegoVideoConfigPreset

## ZegoVideoConfigPreset

视频配置分辨率与比特率预设枚举。预设的分辨率分别针对移动端与桌面端做了适配。在移动端上 height 长于 width，而桌面端相反。例如 1080p 在移动端上实际为 1080(w) x 1920(h)，而在桌面端上实际为 1920(w) x 1080(h)

*Declared in `constants/ZegoVideoConfigPreset.java`*
