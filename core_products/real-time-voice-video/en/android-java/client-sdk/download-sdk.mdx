# Download SDK

- - -

The ZEGOCLOUD Video Call (Express Video) SDK is provided by ZEGO. You can obtain the Express-Video SDK for Android clients on this page. For release notes, please refer to [Release Notes](/real-time-video-android-java/client-sdk/release-notes). For compliance matters, please refer to [ZEGOCLOUD Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

<Warning title="Warning">


- Starting from version 3.17.0, the Express SDK has upgraded its NDK dependency to NDK r25c. After upgrading, developers need to perform compatibility testing to prevent incompatibility issues.
- If you encounter compatibility issues during testing or need to use an older version of NDK, please contact ZEGOCLOUD Technical Support.
</Warning>
