# Download SDK

ZEGO Video Call (Express Video) SDK is provided by ZEGO. You can obtain the Express-Video SDK for macOS clients on this page. For release notes, please refer to [Release Notes](/real-time-video-macos-swift/client-sdk/release-notes). For compliance matters, please refer to [ZEGO Security and Compliance White Paper](/policies-and-agreements/zego-security-and-compliance-white-paper).

- Mac Catalyst is a new framework introduced by Apple in 2019, also known as UIKit for Mac, designed to allow iPad apps to run on macOS. For details, please refer to [Mac Catalyst Official Website](https://developer.apple.com/mac-catalyst).
- Express macOS SDK supports OS X 10.10+.
- Express Mac Catalyst SDK supports macOS 11.0+.
