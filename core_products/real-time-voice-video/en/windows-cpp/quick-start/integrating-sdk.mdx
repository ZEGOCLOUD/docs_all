# Integrate SDK

- - -

## Prepare Environment

Before starting to integrate ZEGO Express SDK, ensure that the development environment meets the following requirements:

- Visual Studio 2015 or above.
- Windows 7 or above.
- External devices such as microphones and cameras that support audio and video functions.

##  Integrate SDK

### (Optional) Create New Project

<Accordion title="This step takes how to create a new project as an example. If integrating into an existing project, you can ignore this step." defaultOpen="false">
1. Open Microsoft Visual Studio and select the "File > New > Project" menu.
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/add_project_1.png" /></Frame>

2. In the new project window, select the project type as "MFC Application", enter the project name, select the project storage path, and click "OK".
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/add_project_2.png" /></Frame>

3. Enter the MFC application window, select "Application type" as "Dialog based", and click "Finish".
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/add_project_3.png" /></Frame>
</Accordion>

###  Import SDK

1. Please download ZEGO Express SDK from the [Download](/real-time-video-windows-cpp/client-sdk/download-sdk) document.

2. Decompress ZEGO Express SDK and copy the contents under the "release/Library" directory (including x86 and x64 architectures) to your project.

## Set Project Properties

1. In the Solution Explorer window, right-click the project name and click "Properties" to enter the project properties page.

2. Make the following configurations on the project properties page. After completing the configuration, click "OK".

    1. Add the "include" directory to the header file search path.

        In the left menu, select "Configuration Properties > C/C++ > General", and add the "include" directory in "Additional Include Directories".

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/project_property_1.png" /></Frame>

    2. Add the "lib" directory to the library search path.

        In the left menu, select "Configuration Properties > Linker > General", and add the "lib/x86" or "lib/x64" directory in "Additional Library Directories".
    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/project_property_2.png" /></Frame>

    3. Specify the link library "ZegoExpressEngine.lib".

        In the left menu, select "Configuration Properties > Linker > Input", and add "ZegoExpressEngine.lib" in "Additional Dependencies".

    <Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Windows/ZegoExpressEngine/Common/project_property_3.png" /></Frame>
