# ZegoMixRenderMode

## ZegoMixRenderMode

混流视频画面渲染模式。

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoMixRenderModeFill"
  prototype="ZegoMixRenderModeFill=(0)"
  parent_name="ZegoMixRenderMode"
  parent_type="enum">
填充模式，等比缩放填充整个区域，可能有部分被裁减。
</ParamField>
<ParamField
  name="ZegoMixRenderModeFit"
  prototype="ZegoMixRenderModeFit=(1)"
  parent_name="ZegoMixRenderMode"
  parent_type="enum">
适应模式，等比缩放填充区域，如果缩放后比例与设置的尺寸不匹配，多余部分显示为透明。
</ParamField>
