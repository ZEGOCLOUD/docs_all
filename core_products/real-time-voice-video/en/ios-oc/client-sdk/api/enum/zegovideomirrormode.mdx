# ZegoVideoMirrorMode

## ZegoVideoMirrorMode

预览或拉流端的镜像模式。

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoVideoMirrorModeOnlyPreviewMirror"
  prototype="ZegoVideoMirrorModeOnlyPreviewMirror=(0)"
  parent_name="ZegoVideoMirrorMode"
  parent_type="enum">
只有本地预览时才是镜像画面，默认采用此模式。当移动端使用后置摄像头时，虽然仍默认采用此模式，但是不起作用，本地预览不设置镜像。
</ParamField>
