# ZegoMixerOutputVideoConfig

## ZegoMixerOutputVideoConfig

混流输出视频配置。

**详情**

配置混流输出的视频参数，编码格式、码率。

<Note title="">
- **业务场景**：手动混流场景，比如多人连麦直播。
</Note>

*Declared in `ZegoExpressDefines.h`*

<ParamField

<ParamField

  parent_name="ZegoMixerOutputVideoConfig"

</ParamField>
