# ZegoReceiveRangeParam

## ZegoReceiveRangeParam

Receive range configuration.

*Declared in `ZegoExpressDefines.h`*

### Properties
<ParamField
  name="min"
  prototype="nonatomic, assign float min"
  parent_name="ZegoReceiveRangeParam"

The minimum distance at which the 3D sound effect starts to have attenuation effect, the value needs to be &gt;= 0 and &lt;= max, the default value is 0.
</ParamField>
<ParamField
