<Note title="Notes">


- The SDK package downloaded from this page contains all API interfaces related to `Audio` and `Video`. If you need an `Audio-only` SDK package, please contact ZEGOCLOUD Technical Support.
- For SDK integration instructions, please refer to [Quick Start](/real-time-video-linux-cpp/quick-start/integrating-sdk).
- For SDK version changes, please refer to [Release Notes](/real-time-video-linux-cpp/client-sdk/release-notes).
- This SDK package includes the following architectures:
    - amd64 (x86_64-linux-gnu)
    - arm64 (aarch64-linux-gnu)
    - armhf (arm-linux-gnueabihf)
    - armel (arm-linux-gnueabi)
- If you need SDK packages for other Linux platforms and architectures (e.g., MIPS / RISC-V / LoongArch or specific embedded systems), please [click this link to fill out the questionnaire](https://zegocloud.feishu.cn/share/base/form/shrcngu3M4UhZiviuornyuYX6mc).
