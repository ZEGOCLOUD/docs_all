# ZegoSceneState

## ZegoSceneState

范围场景状态变化原因。

*Declared in `constants/ZegoSceneState.java`*

### 属性
<ParamField
  name="ZegoSceneStateLogining"
  prototype="ZegoSceneStateLogining=(0)"
  parent_name="ZegoSceneState"
  parent_type="enum">
正在登录场景。当调用 [loginScene] 登录场景时，进入该状态，表示正在请求连接服务器。通常通过该状态进行应用界面的展示。
</ParamField>
<ParamField
  name="ZegoSceneStateLogined"
  prototype="ZegoSceneStateLogined=(1)"
  parent_name="ZegoSceneState"
  parent_type="enum">
登录场景成功。当登录场景成功后，进入该状态，表示登录场景已经成功，用户可以正常使用范围场景功能。
</ParamField>
<ParamField
  name="ZegoSceneStateLoginFailed"
  prototype="ZegoSceneStateLoginFailed=(2)"
  parent_name="ZegoSceneState"
  parent_type="enum">
登录场景失败。当登录场景失败后，进入该状态，表示登录场景已经失败，例如 AppID 或 Token 不正确等。
</ParamField>
<ParamField
  name="ZegoSceneStateReconnecting"
  prototype="ZegoSceneStateReconnecting=(3)"
  parent_name="ZegoSceneState"
  parent_type="enum">
场景连接临时中断。如果因为网络质量不佳产生的中断，SDK 会进行内部重试。
</ParamField>
<ParamField
  name="ZegoSceneStateReconnected"
  prototype="ZegoSceneStateReconnected=(4)"
  parent_name="ZegoSceneState"
  parent_type="enum">
场景重新连接成功。如果因为网络质量不佳产生的中断，SDK 会进行内部重试，重连成功后进入该状态。
</ParamField>
<ParamField
  name="ZegoSceneStateReconnectFailed"
  prototype="ZegoSceneStateReconnectFailed=(5)"
  parent_name="ZegoSceneState"
  parent_type="enum">
场景重新连接失败。如果因为网络质量不佳产生的中断，SDK 会进行内部重试，重连失败后进入该状态。
</ParamField>
<ParamField
  name="ZegoSceneStateKickOut"
  prototype="ZegoSceneStateKickOut=(6)"
  parent_name="ZegoSceneState"
  parent_type="enum">
被服务器踢出场景。例如有相同用户名在其他地方登录场景导致本端被踢出场景，会进入该状态。
</ParamField>
<ParamField
  name="ZegoSceneStateLogout"
  prototype="ZegoSceneStateLogout=(7)"
  parent_name="ZegoSceneState"
  parent_type="enum">
登出场景成功。没有登录场景前默认为该状态，当调用 [logoutScene] 登出场景成功后，进入该状态。
</ParamField>
