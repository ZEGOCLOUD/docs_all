# ZegoMixerOutputRoomInfo

## ZegoMixerOutputRoomInfo

混流中输出流的房间信息。

**详情**

为单条输出流设置房间信息；能将混流输出流增加到指定房间，让房间内的用户接收到房间内流增加的通知。

<Note title="">
- **业务场景**：手动混流场景，比如多人连麦直播。
</Note>

<Warning title="">
- **支持版本**：3.18.0 及以上。
- **使用限制**：不支持混流过程中动态更新。
</Warning>

*Declared in `entity/ZegoMixerOutputRoomInfo.java`*
