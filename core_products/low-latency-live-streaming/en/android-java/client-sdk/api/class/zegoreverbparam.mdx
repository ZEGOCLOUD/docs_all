# ZegoReverbParam

## ZegoReverbParam

音频混响参数

**详情**

开发者可以使用 SDK 的内置预置来改变混响的参数。

*Declared in `entity/ZegoReverbParam.java`*

### 属性
<ParamField
  name="roomSize"
  prototype="public float roomSize"
  parent_name="ZegoReverbParam"
  parent_type="class">
房间大小，取值范围 [0.0, 1.0]，用于控制产生混响模拟的 “房间” 的大小，房间越大，混响越强
</ParamField>
<ParamField
  name="reverberance"
  prototype="public float reverberance"
  parent_name="ZegoReverbParam"
  parent_type="class">
余响，取值范围 [0.0, 0.5]，用于控制混响的拖尾长度
</ParamField>
<ParamField
  name="damping"
  prototype="public float damping"
  parent_name="ZegoReverbParam"
  parent_type="class">
混响阻尼，取值范围 [0.0, 2.0]，控制混响的衰减程度，阻尼越大，衰减越大
</ParamField>
<ParamField
  name="dryWetRatio"
  prototype="public float dryWetRatio"
  parent_name="ZegoReverbParam"
  parent_type="class">
干湿比，取值范围大于等于 0.0，控制混响与直达声和早期反射声之间的比 例，干(dry)的部分默认定为1，当干湿比设为较小时，湿(wet)的比例较大，此时混响较强
</ParamField>
