# ZegoFontStyle

## ZegoFontStyle

字体风格。

**详情**

字体风格配置，可用于配置字体类型、字体大小、字体颜色、字体透明度。

<Note title="">
- **业务场景**：手动混流场景时，设置文字水印，比如多人连麦直播。
</Note>

*Declared in `entity/ZegoFontStyle.java`*
