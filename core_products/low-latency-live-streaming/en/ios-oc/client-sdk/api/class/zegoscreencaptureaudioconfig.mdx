# ZegoScreenCaptureAudioConfig

## ZegoScreenCaptureAudioConfig

屏幕采集音频配置

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="enableWindowCapture"
  prototype="nonatomic, assign BOOL enableWindowCapture"
  parent_name="ZegoScreenCaptureAudioConfig"
  parent_type="class">
是否采集窗口声音。当为 true 时，如果采集源为窗口，则采集窗口声音，当采集源为屏幕或者为 false 时，则采集系统声音。（仅支持 Windows 10 2004 及以上版本采集窗口声音）
</ParamField>
