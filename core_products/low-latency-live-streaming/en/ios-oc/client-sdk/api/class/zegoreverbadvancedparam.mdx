# ZegoReverbAdvancedParam

## ZegoReverbAdvancedParam

Audio reverberation advanced parameters.

**Details**

Developers can use the SDK's built-in presets to change the parameters of the reverb.

*Declared in `ZegoExpressDefines.h`*
