# ZegoNetworkProbeTracerouteResult

## ZegoNetworkProbeTracerouteResult

traceroute result

**Details**

Jump up to 30 times. The traceroute result is for reference and does not represent the final network connection result. The priority is http, tcp, udp probe result.
