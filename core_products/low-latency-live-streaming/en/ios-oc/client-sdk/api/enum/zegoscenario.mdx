# ZegoScenario

## ZegoScenario

Room scenario.

<Warning title="">
- **Available since**: 1.1.0
 To ensure that the selected room scene mode meets the developer's business scene experience requirements, it is recommended to carefully read the [https://doc-zh.zego.im/article/16316](https://doc-zh.zego.im/article/16316) document to understand the scene setting method, the differences between different scenes, and how to choose the right scene.
</Warning>
