# ZegoPublisherState

## ZegoPublisherState

推流状态

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="ZegoPublisherStateNoPublish"
  prototype="ZegoPublisherStateNoPublish=(0)"
  parent_name="ZegoPublisherState"
  parent_type="enum">
未推流状态，在推流前处于该状态。如果推流过程出现稳态的异常，例如 AppID 或 Token 不正确，或者如果其他用户已经在推送流，推送相同流 ID 的流会失败，都会进入未推流状态
</ParamField>
<ParamField
  name="ZegoPublisherStatePublishRequesting"
  prototype="ZegoPublisherStatePublishRequesting=(1)"
  parent_name="ZegoPublisherState"
  parent_type="enum">
正在请求推流状态，推流操作执行成功后会进入正在请求推流状态，通常通过该状态进行 UI 界面的展示。如果因为网络质量不佳产生的中断，SDK 会进行内部重试，也会回到正在请求推流状态
</ParamField>
<ParamField
  name="ZegoPublisherStatePublishing"
  prototype="ZegoPublisherStatePublishing=(2)"
  parent_name="ZegoPublisherState"
  parent_type="enum">
正在推流状态，进入该状态表明推流已经成功，用户可以正常通信
</ParamField>
