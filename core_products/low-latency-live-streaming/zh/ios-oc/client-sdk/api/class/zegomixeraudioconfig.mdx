# ZegoMixerAudioConfig

## ZegoMixerAudioConfig

混流音频配置

**详情**

配置混流任务的音频码率、声道数、音频编码

*Declared in `ZegoExpressDefines.h`*

### 属性
<ParamField
  name="bitrate"
  prototype="nonatomic, assign int bitrate"
  parent_name="ZegoMixerAudioConfig"
  parent_type="class">
音频码率，单位为 kbps，默认为 48 kbps，开始混流任务后不能修改
</ParamField>
<ParamField
  name="channel"
  prototype="nonatomic, assign [ZegoAudioChannel](../../api-reference/enum#zegoaudiochannel) channel"
  parent_name="ZegoMixerAudioConfig"
  parent_type="class">
音频声道，默认为 Mono 单声道
</ParamField>
<ParamField
  name="codecID"
  prototype="nonatomic, assign [ZegoAudioCodecID](../../api-reference/enum#zegoaudiocodecid) codecID"
  parent_name="ZegoMixerAudioConfig"
  parent_type="class">
编码 ID，默认为 ZegoAudioCodecIDDefault
</ParamField>
<ParamField
  name="mixMode"
  prototype="nonatomic, assign [ZegoAudioMixMode](../../api-reference/enum#zegoaudiomixmode) mixMode"
  parent_name="ZegoMixerAudioConfig"
  parent_type="class">
多路音频流混音模式。若 [ZegoAudioMixMode] 选择为 [Focused]，SDK 将会选择 4 路已设置 [isAudioFocus] 的输入流作为焦点语音突出，若未选择或选择少于 4 路，则会自动补齐 4 路。在 web 平台，此属性不生效。
</ParamField>
