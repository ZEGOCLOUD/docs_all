## 音频能力

### 基础功能

| 基础功能-20% | 功能描述-50% | 业务场景-30% |
| --- | --- | --- |
| [音频频谱与音量变化](/live-streaming-u3d/audio/sound-level-spectrum) | 音频频谱：即数字音频信号在各频点的能量值。音量变化：即某条流音量的大小。 | <ul><li>判断麦上的用户谁在说话、麦克风、扬声器等是否可用。</li><li>音频频谱动画展示。</li></ul> |
| [耳返与声道设置](/live-streaming-u3d/audio/headphone-monitor) | 耳机采集监听，在设备上插入耳机（普通耳机或蓝牙耳机）后，能从本机耳机侧听到本设备麦克风采集的声音。 | <ul><li>秀场直播。</li><li>情感 FM。</li><li>音乐教学等较为专业场景。</li></ul> |
| [音频 3A 处理](/live-streaming-u3d/audio/audio-3a-processing) | 在实时音视频通话或直播时，可以对音频进行 3A 处理，以提高通话或直播质量和用户体验。<ul><li>AEC（回声消除）：对采集到的音频数据进行过滤以减少音频中的回声。</li><li>AGC（自动增益控制）：开启该功能后，SDK 能够自动调节麦克风音量，适应远近拾音，保持音量稳定。</li><li>ANS（降噪）：识别声音中的背景噪声并进行消除，开启该功能后可以使人声更加清晰。</li></ul> | 所有希望有高质量实时音视频服务的场景。 |

### 进阶功能

| 进阶功能-20% | 功能描述-50% | 业务场景-30% |
| --- | --- | --- |
| [自定义视频采集](/live-streaming-u3d/video/custom-video-capture) | 自定义向 Zego Express SDK 提供视频输入源输入视频数据，并由 Zego Express SDK 进行编码推流的功能。 | <ul><li>使用了第三方美颜厂商的美颜 SDK。</li><li>摄像头无法被 SDK 正常使用。</li><li>直播非摄像头采集的数据。</li></ul> |
| [视频大小流和分层编码](/live-streaming-u3d/video/small-large-video-stream-and-layered-encoding) | 将码流分为基本层和扩展层，可以为不同网络状态、不同设备性能的用户提供更好的体验。 | 视频通话。 |
| [推流视频增强](/live-streaming-u3d/video/publish-video-enhancement) | ZEGO Express SDK 提供多种视频前处理增强能力，开发者可以根据业务需要，在推流端对画面的效果进行调整。<ul><li>基础美颜：美白、磨皮、锐化、红润，为用户呈现出良好的肌肤状态，打造独特自然的美颜效果。</li><li>低照度增强：在环境光较暗的情况下，摄像头采集到的画面亮度不满足看清人脸、或无法进行人脸识别等业务需求时，对画面亮度进行增强。</li><li>色彩增强：在保护人物肤色的情况下，增强欠饱和的色彩，让画面色彩更逼真，更符合人的视觉感受。</li></ul> | <ul><li>秀场直播</li><li>音视频通话</li></ul> |
