# ZegoCustomVideoRenderConfig

## ZegoCustomVideoRenderConfig

自定义视频渲染配置

**详情**

当需要使用自定义渲染功能时需要将该类的实例作为参数设置给 [enableCustomVideoRender] 函数。

*Declared in `entity/ZegoCustomVideoRenderConfig.java`*
