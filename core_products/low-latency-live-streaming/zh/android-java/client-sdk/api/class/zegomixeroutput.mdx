# ZegoMixerOutput

## ZegoMixerOutput

混流输出对象，目前一个混流任务最多只支持 4 路不同分辨率的视频流。

**详情**

配置混流输出的目标 URL 或流 ID

*Declared in `entity/ZegoMixerOutput.java`*
