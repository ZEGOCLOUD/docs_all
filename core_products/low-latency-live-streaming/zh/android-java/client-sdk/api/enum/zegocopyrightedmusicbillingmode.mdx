# ZegoCopyrightedMusicBillingMode

## ZegoCopyrightedMusicBillingMode

点歌计费模式。

*Declared in `constants/ZegoCopyrightedMusicBillingMode.java`*

### 属性
<ParamField
  name="ZegoCopyrightedMusicBillingModeCount"
  prototype="ZegoCopyrightedMusicBillingModeCount=(0)"
  parent_name="ZegoCopyrightedMusicBillingMode"
  parent_type="enum">
按次计费。用户每获取一次歌曲资源都需要计一次费用，即根据实际调用获取歌曲资源接口（如 [requestResource] 等）传参为按次的用户进行收费。
</ParamField>
<ParamField

按用户包月计费。按月维度向单个用户计费，即统计调用获取歌曲资源（如 [requestResource] 等）传参为按用户包月的用户 ID，按月维度进行收费。
</ParamField>
<ParamField

按房间包月计费。按月维度向房间用户计费，即统计调用获取歌曲资源（如 [requestResource] 等）传参为按房间包月的 Roomid，按月维度进行收费。
</ParamField>
<ParamField

</ParamField>
