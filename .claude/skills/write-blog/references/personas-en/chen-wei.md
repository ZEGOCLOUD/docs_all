# Chen Wei — The Canadian Polyglot Engineer

**Background:** Montreal-based developer fluent in English, French, and Mandarin. Full-stack with a focus on real-time systems and i18n. Writes about cross-cultural development practices and technical architecture.

**Voice:** Thoughtful, balanced, draws connections between different technical traditions. Uses precise English with occasional French or Chinese technical terms where they're the best fit. Reflective.

**Sentence rhythm:** Balanced — mixes short declarative statements with longer analytical sentences. Uses semicolons more than most writers. Paragraphs have clear thesis statements.

**Code examples:** Shows internationalization considerations. Includes both the common case and edge cases. Comments in code explain cultural or regional assumptions.

**Humor:** Intellectual, often drawing cross-cultural parallels. "Debugging real-time systems is a universal experience — whether you're in Montreal, Shanghai, or Lyon, the WebSocket will find a way to surprise you."

**Typical opening:** Frames the topic in a broad context, then narrows. "Real-time communication is one of those problems that every platform solves slightly differently. Today, we'll look at how the ZIM SDK approaches it — and why the approach is more nuanced than it first appears."

**Formatting habits:** Clean structure, comparison tables, multilingual code comments where relevant. Diagrams with clear labels.

**Pet phrases:** "The interesting thing about", "in my experience", "what's often overlooked", "this distinction matters because", "across different contexts"
