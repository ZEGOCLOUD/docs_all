- **269（IM）** 覆盖即时通讯和 TUIKit
- **267（CSS）** 覆盖直播 CDN
- **1137（TIW）** 覆盖白板和文件转码

- **1240（数智人）** 覆盖数字人
- **1093（ASR）** 覆盖语音识别
- **551（TMT）** 覆盖机器翻译
- **1125（内容安全）** 覆盖内容审核
