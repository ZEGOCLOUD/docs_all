# ReleaseNote（发布日志）结构标准

## 文档结构概述

发布日志按大版本号分组（可选），每个大版本下按日期倒序排列。每个版本更新使用 `<Accordion>` 组件包裹。

```
发布日志
├── V2（最新主版本）
│   ├── 2025-02-05
│   │   └── <Accordion title="服务端 v2.10.0">
│   ├── 2025-01-15
│   │   └── <Accordion title="服务端 v2.9.0">
│   └── ...
├── V1（历史主版本）
│   ├── 2024-12-31
│   │   └── <Accordion title="服务端 v1.0.0">
│   └── ...
└── Beta（测试版本）
    └── ...
```

## 文档模板

```mdx
---
date: "2025-02-05"
---
# 发布日志

---

## V2

### 2025-02-05

<Accordion title="服务端 v2.10.0" defaultOpen="true">

**新增功能**

| 功能项 | 功能描述 | 相关文档 |
| ------ | -------- | -------- |
| 功能名称 | 功能说明文字，可包含 `<Note>` 组件 | [文档标题](链接) |

**改进优化**

- 优化项1
- 优化项2

**问题修复**

- 修复的问题1
- 修复的问题2

**废弃删除**（可选）

- 废弃的接口/字段1
- 删除的功能2

</Accordion>

### 2025-01-15

<Accordion title="服务端 v2.9.0" defaultOpen="true">

...

</Accordion>

## V1

### 2024-12-31

<Accordion title="服务端 v1.0.0" defaultOpen="true">

**版本更新**

- 功能概述1
- 功能概述2

</Accordion>
```

## 章节规范

### 版本号分组

使用 `##` 标题按主版本号分组：

如果产品没有多个大版本，则不使用版本号分组。

- `## V2` - 最新主版本
- `## V1` - 历史主版本
- `## Beta` - 测试版本

### 日期格式

使用 `###` 标题，格式为 `YYYY-MM-DD`。

### Accordion 组件

每个版本更新使用 Accordion 包裹：

```mdx
<Accordion title="服务端 v2.10.0" defaultOpen="true">
  内容...
</Accordion>
```

- `title`：格式为 `服务端 vX.X.X`。如果不区分端则仅写 `vX.X.X`。
- `defaultOpen`：最新版本设为 `"true"`，历史版本可省略

### 内容分类

| 分类 | 格式 | 使用场景 |
|------|------|----------|
| **新增功能** | 表格 | 新增的 API、功能、字段等 |
| **改进优化** | 列表 | 性能优化、体验提升等 |
| **问题修复** | 列表 | Bug 修复 |
| **废弃删除** | 列表 | 废弃/删除的接口、字段 |
| **版本更新** | 列表 | 大版本发布时的概述 |

### 新增功能表格格式

```mdx
**新增功能**

| 功能项 | 功能描述 | 相关文档 |
| ------ | -------- | -------- |
| 功能名称 | 功能说明 | [文档标题](链接) |
```

- **功能项**：功能名称，简洁明了
- **功能描述**：详细说明，可包含 `<Note>` 组件标注注意事项
- **相关文档**：链接到相关文档，没有则填 `-`

### 改进优化/问题修复格式

```mdx
**改进优化**

- 优化项1描述
- 优化项2描述

**问题修复**

- 修复的问题描述
```

## 检查清单

### 结构规范
- [ ] 日期是否按倒序排列（最新在前）
- [ ] 是否使用 Accordion 组件
- [ ] Accordion title 格式是否为 `服务端 vX.X.X`

### 内容规范
- [ ] 新增功能是否使用表格格式
- [ ] 改进优化/问题修复是否使用列表格式
- [ ] 功能描述是否清晰
- [ ] 相关文档链接是否正确

### 格式规范
- [ ] 版本号格式是否统一（如 v2.10.0）
- [ ] 发布日期是否完整
- [ ] 敬语使用是否规范（使用"您"而非"你"）
