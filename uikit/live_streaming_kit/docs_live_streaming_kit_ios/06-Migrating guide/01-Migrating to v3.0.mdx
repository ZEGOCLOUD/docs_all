# Migrating to v3.0

## Renamed some config classes

From version v3.0 onwards, certain config classes in `ZegoUIKitPrebuiltLiveStreaming` were renamed to resolve compilation errors caused by naming conflicts with `ZegoUIKitPrebuiltCall`.

`ZegoBottomMenuBarConfig` -> `ZegoLiveStreamingBottomMenuBarConfig`
`ZegoPrebuiltAudioVideoViewConfig` -> `ZegoLiveStreamingAudioVideoViewConfig`
`ZegoTranslationText` -> `ZegoLiveStreamingTranslationText`
