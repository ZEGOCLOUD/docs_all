# Invitation Events



## onIncomingCallDeclineButtonPressed

>
> This callback will be triggered to **callee** when callee click decline button in incoming call
>
>- function prototype:
>
>```js
>onIncomingCallDeclineButtonPressed(navigation)
>```
>
>- example:
>
>```js
>return ZegoUIKitPrebuiltCallService.init(
>  Your_APPID,
>  Your_APPSign,
>  Your_UserID,
>  Your_UserName,
>  [ZIM, ZPNs],
>  {
>    onIncomingCallDeclineButtonPressed: (navigation) => {
>      //...
>    },
>  }
>);
>```

## onCallEnd

> user will receive this callback at the end of the call. 
>>Possible reasons that may trigger this callback include:
>>- 0: the call ended due to a local hang-up
>>- 1: the call ended when the remote user hung up, leaving only one local user in the call
>>- 2: the call ended due to being kicked out
>
>- function prototype:
>```js
>onCallEnd(callID, reason, duration)
>```
>- example:
>```js
>return ZegoUIKitPrebuiltCallService.init(
>  Your_APPID,
>  Your_APPSign,
>  Your_UserID,
>  Your_UserName,
>  [ZIM, ZPNs],
>  {
>    requireConfig: (data) => {
>      return {
>        onCallEnd: (callID, reason, duration) => {
>          //...
>          props.navigation.navigate('HomeScreen');
>        }
>      }
>    }
>  }
>);
>```
