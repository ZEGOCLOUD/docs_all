# API

## ZegoUIKitPrebuiltCallService

### init

init(appID, appSign, userID, userName, config): Promise\<boolean\>

接口描述：初始化ZegoUIKitPrebuiltCallService对象

参数信息：
| 属性     | 类型                                                                                        | 必需 | 描述                                                   |
| :------- | :------------------------------------------------------------------------------------------ | :--- | :----------------------------------------------------- |
| appID    | number                                                                                      | 是   | ZEGO 为开发者签发的应用 ID，请从 ZEGO 管理控制台申请。 |
| appSign  | string                                                                                      | 是   | AppID 对应的应用签名，请从 ZEGO 管理控制台申请。       |
| userID   | string                                                                                      | 是   | 用户ID。                                               |
| userName | string                                                                                      | 是   | 用户名称。                                             |
| config   | [ZegoUIKitPrebuiltCallInvitationConfig](./Config.mdx#ZegoUIKitPrebuiltCallInvitationConfig) | 是   | 呼叫邀请配置项。                                         |


### uninit

uninit(): void

接口描述：当不需要ZegoUIKitPrebuiltCallService对象时，调用此接口释放资源

### getVersion

getVersion(): string

接口描述：获取当前SDK版本号

### setCallInvitationConfig

setCallInvitationConfig(config): void

接口描述：设置呼叫邀请配置

参数信息：
| 属性     | 类型                                                                                        | 必需 | 描述                                                   |
| :------- | :------------------------------------------------------------------------------------------ | :--- | :----------------------------------------------------- |
| config   | [ZegoUIKitPrebuiltCallInvitationConfig](./Config.mdx#ZegoUIKitPrebuiltCallInvitationConfig) | 是   | 呼叫邀请配置项。                                         |

### sendInvitation

sendInvitation(config): Promise\<SendInvitationResult\>

接口描述：发送呼叫邀请

参数信息：
| 属性   | 类型                                              | 必需 | 描述               |
| :----- | :------------------------------------------------ | :--- | :----------------- |
| config | [CallInviteConfig](./Config.mdx#CallInviteConfig) | 是   | 发起呼叫邀请参数。 |

返回值：SendInvitationResult
| 属性         | 类型       | 描述                   |
| :----------- | :--------- | :--------------------- |
| invitationID | string     | 呼叫邀请 ID。          |
| callID       | string     | 通话 ID。              |
| successUsers | ZegoUser[] | 收到呼叫邀请的用户。   |
| errorUsers   | ZegoUser[] | 未收到呼叫邀请的用户。 |


### cancelInvitation

cancelInvitation(config): Promise\<CancelInvitationResult\>

接口描述：取消呼叫邀请

参数信息：
| 属性        | 类型   | 必需 | 描述          |
| :----- | :------------------------------------------------ | :--- | :----------------- |
| config | [CancelInviteConfig](./Config.mdx#CancelInviteConfig) | 是   | 取消呼叫邀请参数。 |

返回值：CancelInvitationResult
| 属性         | 类型       | 描述                   |
| :----------- | :--------- | :--------------------- |
| invitationID | string     | 呼叫邀请 ID。          |
| successUsers | ZegoUser[] | 收到取消邀请的用户。   |
| errorUsers   | ZegoUser[] | 未收到取消邀请的用户。 |
